Practice: Secure Solutions
The Secure Solutions practice provides a repeatable method of adding security to an application development project. It provides for the inclusion and implementation of security controls based on client needs and in response to known threats and vulnerabilities.

Main Description
Security within a solution is an implementation of a specialized set of requirements, the security requirements. These are mostly business related with a few specific to the IT function within the organization. As such, the success of implementing security within any solution is dependent on gathering the proper set of requirements. The majority of the tasks identified within this capability pattern are in the early stages of the project, this is where the solution requirements are gathered, sorted and applied to an initial design.
