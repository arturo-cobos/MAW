Task: Implement Security Processes
Disciplines: Security and Privacy
Purpose
Ensure the solution developed has security processes around it to maintain its security.

      Main Description
      Now that the solution is ready for production, there must be processes around it that are both secure and ensure the security of the solution. Without secure processes for maintenance and administration, opportunities exist for persons with malicious intent to access or alter information assets.

      The project team must ensure the seamless deployment of the security processes and functions into the client’s enterprise. While these new processes and functions are being deployed, their acceptance should be monitored. Any issues such as bypassing processes, processes that are too cumbersome, etc should be identified and addressed. If not, the processes will be ineffective. The team must make sure to balance the security processes with the solutions needed functionality and the client’s risk posture.
