Task: Monitor Implementation of Security Functions
Disciplines: Security and Privacy
Purpose
Monitoring the implementation of the designed security functions ensures the client’s risk model is carried through to the build cycle of the development process.
Relationships
RolesPrimary Performer:
  • Security and Privacy Consultant
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Architectural Decisions
  • Operational Model
  • Security Architecture
    Outputs
    • Security Functions Implemented
      Main Description
      At this point, the security functions and processes around the solution have been designed. Now the project team must ensure those functions and processes are built properly. This task is essentially the same as in the Micro Design phase.

      The project team must guarantee the various security functions are implemented in a manner consistent with the client’s risk model and security objectives.

      During this task, the project team must keep abreast of the detailed security architecture and the decisions supporting the design of that architecture. Additionally, the team must remain well versed in how the design integrates with the over all operational model of the solution. The team should stay engaged in decisions being made during creation of the solution, to ensure proper security delivery.
