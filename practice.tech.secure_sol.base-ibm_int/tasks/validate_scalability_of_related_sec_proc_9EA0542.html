Task: Validate Scalability of Related Security Processes
Disciplines: Security and Privacy
Purpose

Validating the security processes ensures they will scale to the client's environment.

      Main Description
      Scalability issues are identified and addressed in detailed design and product selection activities. In many cases, however, the information provided about scalability of products does not accurately reflect reality. A thorough analysis of the implemented design to include all security processes as well as technology is required. The trained security consulting team should employ adequate intellectual capital to support them in this task. Things to look for include the following:

      • Size of audit log files
      • Manning for intrusion response efforts
      • Loading requirements for initial role out and registration
      • Number of concurrent sessions allowed for any of the security related functions

      The project team should ensure the scalability is feasible for the proposed solution. Any exceptions should be noted and dealt with before the roll out of the solution.
