Task: Create Asset Profiles
Disciplines: Security and Privacy
Purpose
The purpose of this task is to create an Asset Profile which further defines the requirements and restrictions around information assets in the organization.

      Main Description
      By this point in a project, the Information Assets, their respective Owners, and the location of the assets should have been identified. Interviews with the key business representatives should yield the value and importance of the information assets recorded. This will be a key input to the Asset Profile.

      During the Asset Profile task, practitioners should determine the appropriate level of Confidentiality, Integrity, and Availability needed for each information asset. This should be based on information gleaned in the interviews. Additional items of importance that may need to be included are applicable legislative or regulatory requirements around each asset, monetary values of the asset, etc. Each of these items should be decided upon and recorded in the BUS 300 sheet for Information Asset Profiles. The applicable legislative or regulatory information around each asset should be recorded so Asset Owners and those administering the assets know the ramifications and protections necessary for each asset.
