Activity: Validate Production Security
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
To validate that all necessary security elements have been migrated or installed into the solution's production environment as required to support the anticipated workload.
Relationships
Parent Activities
  • Setup Production Environment
Description
The key to good security in solution development is to ensure the security designs created early in the project are carried all the way through to deployment. Numerous issues could arise that would degrade the security aspects of the project if not handled properly. The security team’s primary function during this activity is to validate that the security implemented matches the design criteria and the client’s risk model. This is accomplished by monitoring the deployment of the various functions.
