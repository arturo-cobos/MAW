Main Description
After much time and effort, the solution is finally ready to go into production. Before the reigns are loosened, however, a final test to ensure the security of the solution is necessary. Although many test environments attempt to simulate production, few actually directly mirror it. Therefore, a final test of the solution in the proper environment is needed to guarantee security.

The project team should ensure that the final ethical hack fully tests the security of the solution in the production environment. Any defects found should be reported and reworked. All defects should be addressed and the client satisfied with the results of the testing before the solution is fully deployed.
