Task: Obtain Current Organization Description
Understand the client's current organization's and relationships
Disciplines: Environment
Purpose
The reasons for performing this task include:
  1. Provide context of the current organization in order to identify and interact with users appropriately
  2. Provide a shared understanding of the structure of the organization within scope
Relationships
RolesPrimary Performer:
  • Strategy Consultant
Additional Performers:
Outputs
  • Current Organization Description
    Steps
    Create an inventory of organizational units within scope
    (e.g. corporate organization, strategic business unit (SBU), functions, teams, individuals)
    Determine reporting relationships between these organizational units
    Diagram these relationships using a tool such as Visio or Freelance
    Key Considerations
    Be careful to identify organization units and functions in terms meaningful to the client's business environment.
