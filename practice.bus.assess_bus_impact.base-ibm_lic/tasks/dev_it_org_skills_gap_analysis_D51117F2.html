Task: Develop IT Organization Skills Gap Analysis
Inventory current skill sets and levels owned by the staff and analyze the gap between available and required skills.
Disciplines: Due Diligence
Purpose
Developing the IT Organization Skills Gap Analysis document is the basis for anticipating the staffing and training needs that will be encountered during project development and deployment.

Outputs

    Main Description
    To perform the gap analysis:
    1. Inventory current skill sets and levels owned by the staff assigned to the project.
    2. Estimate skill sets and levels required to implement and deploy the project product based on currently available requirements and architectural documents.
    3. Analyze the gap between available and required skills.
