Task: Write Source Code
Write and test the source code to meet the goals of the programming cycle following the design laid down in the component model and other design documents.
Disciplines: Technical Implementation
Purpose
Create runnable code in an incremental manner that satisfies the goals for this programming cycle. Ensure the code functions as required by performing white box tests. Ensure that the code is built in a consistent, understood, and repeatable manner.

    Main Description
    There will be incremental development within this lowest level task. How much depends on the programming environment and tools. In the extreme case of an environment like Smalltalk the source code will be built in very small increments. Because this smallest loop of incremental development is not normally shown on a project plan, it is completely absorbed within this one task.
