Task: Implement Help Desk Support
Establish a user support organization to provide assistance during solution deployment.
Disciplines: Deployment
Purpose
The purpose of this task is to establish a help desk, which serves the specific needs of the user community and provides a high quality of service.
Relationships
RolesPrimary Performer:
  • Application Developer
Additional Performers:
InputsMandatory:
  • None
Optional:
  • IT Services Detailed Design
    Outputs
    • Help Desk Established
      Steps
      Select personnel
      Select personnel to staff the help desk.  This staff should be trained on the solution to be deployed as well as familiar with IT services procedures.
      Train personnel
      Train the selected help desk personnel on the new solutions and any new support requirements.
      Communicate to organization
      Communicate to the organization that the help desk has been established, and the services that they will provide to the user community.  Make sure that all users are aware of how to contact the help desk for assistance during solution deployment.
