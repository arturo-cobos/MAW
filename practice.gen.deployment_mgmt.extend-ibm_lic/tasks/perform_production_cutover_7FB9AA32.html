Task: Perform Production Cutover
Perform the necessary actions to cutover to the production system.
Disciplines: Deployment
Purpose
To enable the system to go-live.
Relationships
RolesPrimary Performer:
  • Systems Administrator
Additional Performers:
InputsMandatory:
  • Data Migrated
  • Deployment Packaging
  • Production Environment Established
    		Optional:
    • Deployment Procedures
      Outputs
      • System Deployed
        Main Description
        Following the deployment plan and procedures, bring the solution into a live production state for some set of users.
        Key Considerations
        This task may be relatively straight forward or extremely complex depending on the situation.
