Work Product Descriptor (Outcome): Production Environment Established
This is the required hardware infrastructure, software infrastructure and and integrated subsystem components that are needed to operate the application solution.
Based on Method Work Product: Production Environment Established

Main Description
The environment may contain some or all of the following:
  • Network hardware, both WAN and LAN
  • LAN Operating software
  • Network management tools
  • Server hardware
  • Server Operating software
  • System Management Software and monitoring workstations
  • Middleware software
  • Database software
  • Personnel
  • Application solution
Properties
Optional
Planned
Key Considerations
  • If significant LAN or WAN design is required, contact IBM's network practices.
  • Deploy networks to use a few protocols as possible, preferably only based TCP/IP (mono-protocol network).
