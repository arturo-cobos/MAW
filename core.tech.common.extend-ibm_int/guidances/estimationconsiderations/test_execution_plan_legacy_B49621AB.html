Main Description

The effort attributed to this work product includes only the effort to produce the plan. Execution is covered in the Detail Test Plan work product.

Time required for preparation of the Test Execution Plan depends mostly on the experience of the test project manager and the complexity of the test phase executed.

Based on the complexity, and similarity or dissimilarity between the test sets, amount of effort for the initial setup and preparation will vary. The more the execution details included as part of the test script, the less the effort will be for the Test Execution Plan.

Some estimating guidelines are:

  • For Unit tests, allow one to two hours per test case
  • For other levels of testing, allow four to six hours per test case
  • For each test case, allow for at least three runs and one to two hours per procedure for setting up the Test Execution Procedures
