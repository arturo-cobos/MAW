Main Description

Media Content development costs vary by media type and the stage of development.  Professional quality Media Content requires significant time for development, both in man-hours and on the calendar. This effort is frequently underestimated.

For graphical user interfaces that include text and graphics (still images) obtain a rough top-down estimate as described here to size the effort for this work product.
  • For large (50K hour) projects, the total effort can be expected to take on average 485 hours during the design phase.
  • For small (5K hours or less) projects, the total effort can be expected to take on average 83 hours during the design phase.
