Main Description
Building the plan requires a lot data collection and a good understanding of the customer operational department capabilities. Estimation of the effort to build the plan should include the following:
  • Number of locations to be visited - travel costs $1100/week/person/location
  • Labor cost - number of people collecting data (minimum 2 people)
  • Time required to collect data - 20-30 minutes per existing workstation if data is not available in a configuration database
  • Network topology - 4-5 days per site to minimally document network topology if detailed diagrams are not available.
  • Software inventory - if done for the whole enterprise and for all applications, these can demand significant effort. A year or more may be required. Usually a departmental view is taken to allow an incremental plan to be established.
