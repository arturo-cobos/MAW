Main Description
Media Content is developed from design documents and specifications, such as Interactive Concepts and Interactive Architecture. For each medium, different techniques are used based on the state of the source material and the stage of development. Techniques include the following:
  • Photography
  • Filming
  • Studio Recording
  • Foley Recording
  • Still image digitization
  • Video digitization
  • Audio digitization
  • 2-D Animation
  • 3-D Animation
  • Design Implementation
  • Video editing
  • Audio editing
  • Still image compression & processing
  • Video compression
  • Audio compression
Follow these steps in designing Media Content:
  • Review the usability requirements.
  • Determine if existing visual elements meet the design needs. If not, design media content consistent with the mental model and overall design approach.
  • Review or test the media content with end users for understandability and aesthetic appeal.
  • Ensure, via a documented process, that all media content assets created as components of the user interface for a solution are loaded into a repository.
  • Convert all assets into industry-standard usable formats.
  • Identify an owner of the repository who will have responsibility for that ensuring that the process to create and maintain the repository is followed by all members of the engagement team.
  • Review the consistency and appropriateness of the use of media content within the solution being developed.
