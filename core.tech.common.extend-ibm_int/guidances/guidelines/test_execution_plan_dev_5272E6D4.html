Guideline: Developing Test Execution Plan

Main Description
Consider the following while preparing a Test Execution Plan:
  • The Test Execution Plan should be prepared based on test environment, detailed test plans, and test specification and design work products.
  • It should cover the level-specific setup for the test environment and infrastructure, the test setup and execution, and how the testing runs will be logged.
  • Separately, it should identify all the resources need to perform the testing activities and when they will be needed.
  • It should describe how the test environment and infrastructure would be torn down after the testing is completed.
