Main Description

The following outlines the steps taken to produce a Test Results document;

  1. The actual results of each test executed are recorded, compared, manually or using automation, against the expected results as documented in earlier work products. 
  2. All variances are documented with enough detail to allow for their investigation.
  3. The variances are analyzed, cause of the variance identified and resolved. 
  4. The analysis may result in a fix to the defect or a change request being raised.  If a change request is raised the problem item is closed off and resolution noted.
