Deliverable Properties
External Description

The Solution Release Candidate is a collection of source code and other artifacts used to generate the technical programming of the solution. The purpose of this deliverable is to provide the technical artifacts that are needed to replicate the solution in the testing and production environments.

The deliverable / report will contain the following content, as appropriate:

  • Source Code
  • Media Content, as applicable
  • Recompilation and Build Instructions
  • Configuration Data, as required
  • Executables, if appropriate
  • Test Results

Note: The contents of this deliverable may be delivered in electronic form, potentially via entry into source and configuration management tools. This deliverable may simply reference the list of files with their associated date or version.

Packaging Instructions

May not want to include executables to force re-building the system for the various testing levels. Some organizations will move executables into production only after the last testing environment.
