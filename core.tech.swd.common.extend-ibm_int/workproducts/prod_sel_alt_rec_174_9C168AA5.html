Deliverable: Product Selection Alternatives and Recommendations (DCD 174)
The Product Selection Alternatives and Recommendations deliverable is a report that doucments the analysis and recommendation from a set of commercially available hardware and/or software products that could provide required application or infrastructure functionality.
Purpose
The purpose of this deliverable is to present the alternatives evaluated and make a recommendation so that the customer can make a final selection decision.
Main Description
The deliverable / report will contain the following content:
  • Vendor Qualifications
  • Product Analysis
  • Recommendations
 
Deliverable Properties
Deliverable Parts
  • Decision Model
  • Pre-Qualification Vendor Analysis
  • Request for Information
  • Request for Proposal
