Deliverable: User Experience Model (DEL 0546)
The User Experience Model is a report describing the look and feel of how users will interact with the solution, in graphical and textual terms.
Purpose

The purpose of this deliverable is to provide a sufficiently detailed description of how users will interact with the solution, in preparation for programming the user interface controls, and connecting to the technical architecture & application logic for the specified release.
