Purpose
The purpose of Custom Service Component Development Solution Template is to provide activities and tasks that are to be augmented to the SOMA delivery process in order to implement one or more service components via custom development.
Main Description

The Custom Service Component Development Solution Template includes the unique tasks/activities that help in the implementation of Service Components via custom develpoment.

The purpose of this solution template is to custom develelop service components utilizing the best practice patterns.

There are two types of scenarios:

1) Custom Development Scenario:  The business logic within the service component needs to be custom developed with possible interaction with backend system for data and/or some business function.

2) Integration Develpoment Scenario: The service component implements the functionality mainly by wrappering/exposing the back end functionality or by integrating an existing back end system.
