Task: Prepare Design to Code Transform
All objects/components generated from the analysis phase can be transformed via stereotypes/profile previously defined by a designer/specialist .In this step you will prepare for design to code transformation.
Purpose

The purpose of this task is to generate and prepare the stereotypes/profile for generating the code from the analysis elements.

      Main Description
      In this step, tooling selection plays a part. SOA development in MDD concept, all objects/components generated from the analysis phase can be transformed via stereotypes/profile previously defined by a designer/specialist.  A template will be created to support the pattern transformation. Developers need to understand how to map the component from analysis  to the design by using appropriate template to generate the design framework.
