Term Definition: metadata
Main Description
Metadata is defined as data about data - data that is used to attain a complete understanding of the meaning and characteristics of an organization’s data assets; data that is used to enable the proper, consistent and effective use of an organization’s data assets when performing business functions, processes or other activities.  Models are inherently metadata, since they describe the thing being modeled rather than being the actual thing. Other real world examples of metadata include a parts catalogue - for parts, a blueprint - for a building, or a language dictionary - for words and word usage.
