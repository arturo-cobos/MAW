Capability Pattern: Monitoring
Perform any on-going work necessary to track and report on progress, as well as make periodic updates to key measures of project health.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose

To ensure that the project is progressing as planned, and will likely continue to do so.

To ensure that the project is progressing as planned and will likely continue to do so.
Description

The execution of this work is generally scheduled periodically, and takes random events into account, covering the following topics:

  • Adjusting the project plans and assignments within the constraints of the project baseline
  • Comparing the estimating basis with the trends of the actual workload
  • Analyzing project to anticipate issues
  • Managing team communications
  • Performing reviews of subcontractors
  • Taking part in third party reviews and anticipating necessary corrective actions 
  • Correcting discrepancies and improving forecasts of related business financials with project costs and expenses
  • Reporting on project progress, financial and health status, and on major events
  • Responding to requests for new work items or service requests within the project scope

Staffing
For large projects or programs, some of the tasks included in these activities will be delegated to project office staff.
Key Considerations

It should be noted that the interval between occurrences of monitoring activities must be adjusted all along the project, with shorter intervals during more critical periods.

The activities that are included in the Monitoring activity group are generally part of a global project management level of effort and are not usually detailed in the project schedule. However it is expected that the project manager will plan them in their calendar according to the frequency chosen for each of these activities.
