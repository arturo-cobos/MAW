Artifact: Test Architecture
The Test Architecture provides a comprehensive aggregation of architectural concerns, such as test automation architecture and the test environment configuration.
Purpose

The purpose of the test architecture is to:

  • Provide conceptual integrity to the entire testing effort
  • Align the testing effort with other aspects of the development effort
  • Ensure consistency between the system-under-test and the software architecture
  • Communicate test design decisions to the test team, as well as the development team in general
