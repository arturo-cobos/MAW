Tool: IBM Rational Asset Analyzer
Rational Asset Analyzer provides in-depth insight into dependencies within and among mainframe and composite applications.

Main Description

Assists IT personnel with the maintenance, extension, and reuse of existing mainframe and web applications in modernization efforts such as SOA.

* Provides interactive textual and graphic reports that help you better understand mainframe and distributed application assets and their inter-relationships
* Enables you to identify both the mainframe and the distributed artifacts that a proposed code change to a cross platform application affects so that you can make changes more quickly, with less risk
* Helps reduce the complexity of software projects by delivering up-to-date knowledge of application components from the code itself
* Improves process and team efficiency by making the same application insight available to all team members via a web browser interface
* Installs on the Windows platform to easily inventory and analyze downloaded mainframe and distributed application assets
* Helps shorten the learning curve for new and even experienced developers by documenting your applications
