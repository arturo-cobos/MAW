Tool Mentor: Implement the Solution using Rational Application Developer

Main Description

See the following Rational Application Developer infocenter topics:

Developing Web applications: How to use the Web tools provided by the product.

Developing portal applications: Considerations and instructions for setting up your files for configuration management and sharing across a team.

Developing Java applications: The Java development tools that help you build and run optimized Java programs.

Developing EJB applications: Design, create, and test enterprise applications.

Developing Web services: Tools to help you discover, create or transform, build, deploy, test, develop samples, and publish Web services.

Developing EGL applications: A development environment specifically for EGL applications.

Building XML applications: Tools help you create, import, edit, validate, and generate XML and XSL files, DTDs, and XML schemas from DTD files.
