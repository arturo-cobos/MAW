Main Description

This guideline documents how the process template specifies the role definition.

Role Definition

The process templates currently utilizes the following roles:

  • Contributor (many)
  • Component Lead referred to as Team Lead in the template (one per team area)

The role definition specification is composed of the following fields.

  • role-id: name of the role
  • cardinality: number of members in the team that can assign to the role. There are 2 values of this field single or many.
  • description: the brief description of this role
