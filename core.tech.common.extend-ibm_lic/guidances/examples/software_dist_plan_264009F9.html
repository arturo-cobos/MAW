Main Description

Executive Summary


Analysis of the IT system environment and the proposed solution for the Customer Service and Construction and Maintenance departments has determined that an automated software distribution system is required.


Considering the costs and benefits, it was determined that a savings of $182,000 per year can be achieved, based on an investment of $287,000. These savings were computed based on a 3-year calculation, considering the current number of users (expected to stay stable), support staff, network cost, and software and hardware required to implement the new operational environment.

This conclusion was reached based on the following:

  • The operational system is mission critical and will perform core business processes across four physically distributed locations, which do not have local support staffs or computer savvy workers.
  • The work distribution to the construction department will require an investment in systems (hardware and software) that can be used for software distribution as part of the same operational systems without much additional cost. As a bonus, the same systems (with minimal additions) can also be used for software distribution to the customer service department.
  • The current operational staff is already overloaded trying to support the geographically dispersed users.
  • The high rate of change being introduced into the environment, due to industry deregulation and new competitive pressures (AM/FM system, AMR system, etc.), is increasing the error rate and causing costly downtime.

Recommendations:

  • Perform a vendor evaluation of software distribution packages, which will support the number of users and nodes, connections identified in this plan, as well as the service and support requirement policies at the ABC Company.
  • Validate the included data with the end user departments.
  • Assemble a team to start installing the necessary physical plant (testing and installation) and the hardware and software once these are selected.
  • Establish the overall change plan by coordinating the software development, the purchasing of packages, the software release plan, and the Help Desk required to introduce the changes via software configuration, installation, and distribution processes.

Cost Benefit Analysis

The following tables are the result of calculating the cost and benefit of software distribution. The following assumptions were made:

  • There are 40 hours per workweek.
  • Travel is required to reach the locations if the process is not automated.
  • Software update time, is critical for the business processes used. Also, the complexity of the distributed system requires that changes be introduced in all systems in a coordinated fashion. Back out from changes must be coordinated and occur with minimal end user downtime.
  • The manual cost must be less than the automated cost to justify the expense.
