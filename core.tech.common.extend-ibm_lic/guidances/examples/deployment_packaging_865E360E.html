Main Description

Physical Packaging for the BMSS

Each copy of the package consists of:

  • A diskette containing the install procedures and executables (contained in the BMSSINST.EXE).
  • Install procedures on a single piece of paper (currently dated Oct. 15).
  • A letter from the VP of Branch Operations, John Smith.
  • A ballpoint pen imprinted with the install campaign slogan, “I’ve got that BMSS kind of feeling!”
  • BMSS hotline sticker emblazoned with the hotline phone number.
  • A 15-inch envelope (with name of addressee) bearing the campaign slogan in the upper left corner.
Each branch manager will be sent a copy of the package. Masters of all of these items are currently in the file “BMSS Physical Packaging” in the project manager’s file cabinet. After final validation and General Availability is confirmed, the master copy is to be sent to the Central Distribution and Support (D59) to the attention of the Deputy Controller. They will then manage the production, copying, and distribution of the packages.
