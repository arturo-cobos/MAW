Main Description

Service Categories [Groups]

The following is a proposed list of service categories, or groups, which form the top level of a potential service model or service portfolio. Frequently the client will have a different perspective on how the services should be grouped and defined. The intent of this starter set is to provide a framework for an assessment of the current IT services and to accelerate the development of the IT Services Strategy.
  • User Support Services
  • Deployment Services
  • Application Services
  • Operational Services
  • Delivery Support Services
  • Management and Control Services
  • Business Support Services
  • Network Services
This starter set of IT service categories or groups has been sub-divided into individual services or service elements in the sections which follow.
User Support Services - Direct support to the user to enable day to day usage of IT applications
  • User Call Managed
  • User Trained
  • Install/Moved/Add/Change Delivered
  • Deskside Support Provided
  • Identity and Access Provisioned
Deployment Services - Planning, administering, and implementing new or updated services

  • Install/Move/Add/Change
  • Asset disposal
Application Services - Creation and support of applications to support the business
  • Applications Developed
  • Applications Integrated

Operational Services - Day to day delivery of the technology services
