Main Description

Implementation Approach:

  • Deploy a Network Infrastructure
  • Identify physical locations where the application will be installed. Ensure that Internet Access is available at sufficient bandwidth to support the load as defined in the non-functional requirements document.
  • Determine if sufficient resources exist in the LAN and WAN connections to support the servers and connections to outside service providers.
  • Deploy the LAN based on modern physical topologies.  Token-Ring  (16 Mbit) or Ethernet (10 Mbit) are sufficient if correctly designed.  Switching and routing can be used to segment traffic. Higher speed technologies such as 100BaseT ethernet and ATM can be used to further increase LAN bandwidth.
