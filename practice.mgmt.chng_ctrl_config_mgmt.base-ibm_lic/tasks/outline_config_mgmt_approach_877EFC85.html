Task: Outline Configuration Management Approach
Define how configuration managment will be handled on the project for both formally controlled work products placed under configuration management, as well as those not formally managed but are still placed under version control.
Purpose
The purpose of this task is to outline the high level approach to be taken for configuration management on a project so that the necessary costs and required resources can be factored ito the planning of the project.
