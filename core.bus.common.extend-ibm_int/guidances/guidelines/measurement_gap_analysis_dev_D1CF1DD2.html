Main Description

The Measurement Gap Analysis work product compares the client’s current metrics to the selected metrics in the current instance of the measurement model. 

These selected metrics conform to the categories of measurements in the Global AMS Delivery measurements catalog.  The Measurement Gap Analysis work product is created by doing the following:

  • Examining what metrics are collected, analyzed, and used, as embodied in the customer’s current measurements.
  • Mapping them to the selected metrics described in the measurement model.
  • Identifying discrepancies between the selected metrics in the measurement model and the client’s current metrics.
  • Determining any additional goals not stated in the measurement model but implied by the client’s current measurement collection, analysis, and use.
  • Noting metrics that do not support a goal and marking them as candidates for elimination.
  • Identifying any necessary changes to the (proposed) organizational model.
  • Identifying any necessary changes to the (proposed) process model for the organization .
  • Identifying any other requirements critical to the successful implementation of the measurement program.

This approach is described in detail in the associated technique paper.
