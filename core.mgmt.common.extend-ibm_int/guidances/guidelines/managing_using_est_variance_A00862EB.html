Guideline: Managing using Estimation Variance
This Guideline outlines some simple statistical methods to help you, as the Project Manager, monitor the progress of a project, and introduce you to the simple tools created by the IBM Estimating Community that makes this easy.

Main Description
The intended audience is any IBM practitioner who is responsible for creating initial estimates on projects, and needs to monitor the progress of a project to that baselined estimate.

This technique paper addresses the following topics:

  • Issues to Monitor during project execution
  • Do the planned risk mitigations address areas of uncertainty in the estimate?
  • Estimate Variance for Effort
  • Estimate Variance for Cost
  • Estimate Variance for Schedule
  • Re-baselining
