Task: Confirm Architecture Overview
Confirm that the Architecture Overview is up to date with the latest thinking on the architecture.
Disciplines: Technical Architecture
Purpose
To update the Architecture Overview artifact created at the beginning of the phase to bring it into line with the latest architectural thinking and confirm the overview matches the more detailed architecture models that have been developed since the initial Architecture Overview was created.

    Steps
    Analyze Input Artifacts
    Review and analyze all existing and relevant artifacts that will help in bringing the architecture as defined in the Architecture Overview up to date. In particular look at those artifacts that specify the architecture models. Also review the requirements artifacts to make sure all major requirements are still represented by the architecture diagrams in the Architecture Overview.
    Update Archtiecture Overview
    Update the Architecture Overview using an appropriate drawing or modeling tool.
