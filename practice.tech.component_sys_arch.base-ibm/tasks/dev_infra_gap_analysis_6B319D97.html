Task: Develop Infrastructure Gap Analysis
Perform a gap analysis between existing and required technologies.
Disciplines: Technical Architecture
Purpose
The purpose of this task is to identify the gaps between the existing and required technologies, networks, IT processes, and IT skill sets along with the impact of the conclusions/gaps on the organization. This information is an essential pre-requisite to planning the activities that need to be built and support the future IT infrastructure.
