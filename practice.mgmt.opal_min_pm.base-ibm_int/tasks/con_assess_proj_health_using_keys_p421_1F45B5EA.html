Task: Continually assess project health using 7 Keys (IMSP421)
This task will continuously evaluate project health and determine corrective actions (for example, issues, actions, change requests, etc.).
Purpose
To ensure that the project is monitored to identify any situational factors that could affect the health of the project, and to ensure these factors are addressed.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Event Log
  • Project Management System Summary
    		Optional:
    • Situation Analysis Documentation
      Outputs
      • Event Log
      • Project Management System Summary
        Main Description

        The key outputs of this task are the updated Risk Management Plan which may be part of the Project Management System Summary; and the following as needed: Risk Definitions, Issue Documents, Issue Log, Action Control Documents, Action Log, or the Event Log.

        This task includes the following:

        • Identify situational factors, for example:
          • Strengths and weaknesses
          • Opportunities and threats
          • Resource availability
          • Status of issues, actions, change requests
          • Health of relationship with sponsor
          • Key constraints
          • Risks
        • Select and implement health improvement actions
