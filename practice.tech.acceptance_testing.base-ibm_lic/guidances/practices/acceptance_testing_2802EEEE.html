Practice: Acceptance Testing
User acceptance tests (UAT) verify that the system meets user requirements as specified. The UAT simulates the user environment & emphasizes security, documentation & regression tests. It demonstrates that the system performs as expected to the sponsor & end-user so that they may accept the system.
