Checklist: Process Improvements for Acceptance Test
Criteria for validating the improvement of the acceptance test process through feedback.

Check Items
All members of the Acceptance Test team have been consulted for their input
Root cause analysis has been done from the defect summary
The observations from the analysis have been incorporated into the process improvement recommendations
All accepted recommendations have been reviewed by key stakeholders
All involved parties have been properly informed regarding the improved process to be implemented
The improved process has been monitored and measured against the old process
