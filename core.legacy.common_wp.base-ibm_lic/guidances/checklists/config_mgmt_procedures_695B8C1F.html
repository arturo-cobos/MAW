Checklist: Configuration Management Procedures
Checklist for validating the procedures associated with the plan.
Relationships
Related Elements
  • Configuration Management Plan
Check Items
The procedures describe the organizational responsibilities of configuration management
The procedures require the identification of the specific resources necessary to perform configuration management
The procedures describe the process by for handling problems and changes
This includes how they are submitted, reviewed, and subsequently approved or disapproved.
The procedures identify and describe formats used to document software problems and changes
The procedures describe the methods to formally control of software documentation
This included the storage, handling, and release of software documentation (including master copies) during the development process.
The procedures define the configuration status accounting system
The procedures describe a method for conducting configuration audits
The procedures identify milestones for the life-cycle phase(s) of the development effort
