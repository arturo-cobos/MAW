Checklist: Problem Ticket Form Validation and Verification
Relationships
Related Elements
  • Problem Ticket Form
Check Items
The Problem ID, Caller Information and General Information are fully completed
The helpdesk must ensure these items are fully completed before assigning the problem to another group for fixing.
The information for each entry has been verified for accuracy and completeness
The information recorded in the Problem Ticket Form should be verified by checking each entry to determine whether is it accurate, and complete. 
Each entry must clearly identifies the time / date, author of the log entry
Referenced system components have been inspected for existence and currency
System components referenced by the Problem Ticket Form may include: code, documentation, operations manuals, etc.
