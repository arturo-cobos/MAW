Checklist: User Profiles Validation and Verification
Relationships
Related Elements

Check Items
Profiles have been reviewed with actual users
Surveys can be used to quickly validate the accuracy of the descriptions.
Multiple personas have been created where needed for each user profile
For each user profile, there exists multiple personas when the user profile has bimodal representation on a particular attribute (For example, create multiple personas for a “Customer” user profile, if the Customer population is represented equally by both “power users” and “novices”)
Each user profile is sufficiently detailed
For each user profile, there is sufficient information within each applicable characteristic to guide analysis and user interface design decisions.
Rationale for omitting any characteristic in a user profile is documented
User profile documentation itself is be accessible
Narrative contents of the persona track back to data in the user profile
Captured data may be either implied or explicit.
