Root cause analysis
Root cause of incident

The root cause is the original factor or reason that caused the incident to occur.  The root cause can be defined as the factor that, when eliminated or changed, will eliminate the recurrence of a given problem forever.  For example, a software abend is a contributing factor to the appearance of an error message, incomplete coding is another contributing factor, but incorrect requirements are the root cause of the incident.

Explain how you arrived at this conclusion.  Your explanation may include details of the tools and techniques utilized during the research and analysis, and any assumptions upon which the analysis was based.
