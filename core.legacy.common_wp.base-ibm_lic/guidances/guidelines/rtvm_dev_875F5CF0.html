Main Description
The RTVM is created incrementally as the project progresses. As the project progresses through the development cycle phases (concept, plan, develop, and qualify / test), separate parts of the RTVM are completed. The data developed in previous phases may be updated in subsequent phases as additional information is defined and as project change requests are approved. The RTVM is developed as described in the steps below:
 
1. In the concept phase, the following information is developed:
RTVM Information defined Responsible Role to define information within RTVM
Customer Requirements Systems Engineer
System Requirements Systems Engineer
Acceptance Criteria for Customer and System Requirements Systems Engineer or Test Team
Test Method and Test Type for Customer and System Requirements Systems Engineer or Test Team
 
2. In the plan phase, the following information is developed:
RTVM Information defined Responsible Role to define information within RTVM
Component Requirements Systems Engineer
Acceptance Criteria for Component Requirements Systems Engineer or Development Team
Test Method and Test Type for Component Requirements Systems Engineer or Development Team
 
3. In the develop phase, the following information is developed:
RTVM Information defined Responsible Role to define information within RTVM
Design Documents Development Team
Build Components Development Team
 
4. In the qualify or test phase, the following information is developed:
RTVM Information defined Responsible Role to define information within RTVM
Test Case Test team / Development team (depending on who runs the test)
Test Result Test team / Development team (depending on who runs the test)
