Program Description:-
The customer will log into The Company web page via a username and password. The translator situated between the web server and the SAP system will accept or reject the username and password. Once accepted the user can enter via a web page their product requirements and quantities. The system will check for stock availability and confirm the order. This information is placed into a basket awaiting confirmation from the user on the web. Once the user confirms the order, the order is automatically created in SAP and an order number is send back to the user on the web page.
(See attached diagram)
=======================================================================
Use the Sales Order business object to create Sales Orders over the Internet with BAPI’s.
The business object is within SAP and consists of several ABAP programs, tables and structures. You have to use a SAP extension to the HTML language (Business HTML) to create a link between a business object and a web browser.
