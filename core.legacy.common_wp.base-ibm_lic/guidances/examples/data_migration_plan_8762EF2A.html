Assumptions
A bullet list of assumptions developed for this migration is presented here.

  • Data extract, cleansing, transformation and validation will take place in the legacy system environment(s) and will be presented to the target system.
  • No reporting of errors will be provided.
  • Development effort required for the legacy systems is not included here.
