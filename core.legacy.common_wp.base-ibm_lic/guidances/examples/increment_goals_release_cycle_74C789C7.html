Main Description

Maintenance Release 2.1 fixes Problem Reports 006, 007, 008.

Build cycle 1 will implement fixes to Problem Reports 006 and 008.

  • Iteration 1 for the customer interface team will add a new field to the subscriber GUI to allow the user to enter subscriber social security numbers (PR006, PR008).
  • Iteration 1 for the database team will modify the database interface to accept the new fields.
  • Iteration 2 for the database team will modify the database to accept the social security number data.

Build 2 will implement fixes to Problem Report 007.

  • Iteration 1 for the measurement team will modify traffic metering and measurement and give the user an average count of subscriber connection failures per day.
  • Iteration 1 for the database team will modify the database to store the average count data (PR007).
