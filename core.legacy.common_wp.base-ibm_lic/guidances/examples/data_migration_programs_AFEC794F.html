Example: Data Migration Programs
These routines are very language dependent. The following example is in pseudo code where formula is a transformation function or procedure. It is transforming a column of data in a table in a source database to a table in a target database.

Connect sqlHandle1 to database1;
Connect sqlHandle2 to database2;

Using sqlHandle1 Select column1, primarykey

From table1 into :variableA, :primarykey;

While (sqlFetch from sqlHandle1)

{
variableB = formula (variableA);
sqlExecute using sqlHandle2

(update table2

set column1 = :variableB
where primarykey = :primarykey);

}
