Artifact: Root Cause Analysis Record (APP 412)
This work product is used to document the information relating to the identification and analysis relating to the incident selected for root cause analysis.
Domains: Quality Management
Work Product Kinds: Record
Purpose
The purpose of this work product is to identify the root cause of incidents in order to determine the corrective or preventive actions to resolve the incidents, and prevent their recurrence. This also serves as a critical input for process improvements.

Description
Main DescriptionThe Root Cause Analysis Record contains the information relating to the identification and analysis of incidents selected for root cause analysis. The information contained in the Root Cause Analysis Record includes:
  • Project Information
  • Root Cause Analysis Information
  • Incident Information
  • Participants and Meeting Arrangements
  • Root Cause Analysis
  • Implementation and Tracking
Brief Outline

This artifact contains the following components:

  • Project Identification

  • Root Cause Analysis
    • Root cause incident
    • Possible corrective or preventive actions
    • Implementation and tracking

Tailoring
Impact of not havingConsequences of not having this work product are:
  • No formal record of the root cause of incidents
  • Difficulty in determining the appropriate corrective or preventive actions
  • Possibility of recurring defects or incidents leading to increase in cost of quality
Reasons for not needingThis is a mandatory work product needed to record the root cause analysis of incidents and there is no reason not to have this work product.
