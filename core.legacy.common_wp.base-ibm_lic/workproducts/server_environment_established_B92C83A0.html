Outcome: Server Environment Established
Server environments is the required set of servers that comprise of a development, test/QA, and production system.
Work Product Kinds: Installation/Configuration
Purpose
The Server environments are essential for the efficient work of the project.
Main Description

The server environments can include sandbox system(s), training, production fix and other support servers such as print, fax, interface, EDI, backup/recovery and high availability fall-over for production. A server (e.g. Lotus Notes) also may need to be installed to support a repository of processes and procedures.

Each server environment may contain some or all of the following:

  • Server hardware
  • Server Operating software
  • Server networking
  • Server management tools
  • ERP software package
  • Optional middle ware/database software
This outcome covers the actions required to install the technical server environments.
 
These environments may be described by a model of technical server architecture.
Tailoring
Impact of not havingThe package cannot be implemented without the server environments.
