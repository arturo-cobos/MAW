Outcome: Development Environment Established
This is the result of deploying the hardware and software platforms, operating systems, and connectivity tools necessary to support development.
Work Product Kinds: Installation/Configuration

  • Network management tools
  • Server hardware
  • Server Operating software
  • System Management Software and monitoring workstations
  • Middleware software
  • Database software
  • Personnel
  • Application solution
