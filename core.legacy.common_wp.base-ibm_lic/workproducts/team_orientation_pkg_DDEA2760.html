Description
Main Description

The Team Orientation Package includes the following: Meeting Agenda, List of participants, Presentation materials, and Meeting Minutes.

Presentation materials include a set of documents that will provide the new project team members and subcontractors with a common understanding of the project, including information about the organization, the project goals and objectives, the project management system, the key milestones and deliverables, and the project technological context.
