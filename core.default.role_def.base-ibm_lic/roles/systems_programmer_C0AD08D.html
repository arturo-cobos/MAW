Role: Systems Programmer (040686)
The Systems Programmer provide the underlying system platform programming support on which applications are developed and/or deployed.
Role Sets: Developer, Technology

Systems programming expertise is required in many environments which may include operating systems, application development platforms, networking, packaged software, etc.
