Role: Systems Administrator (042859)
This role is performed by technical professionals who specialize in providing perform services to clients on hardware and software components.
Role Sets: Administrator, Operations

    Main Description

    This role performs technical services for clients on hardware and software components. This includes:

    • The operation and maintenance of operating systems across multiple platforms, middleware applications, various database technologies, high availability solutions and load balancing.
    • Database administration
    • Capacity planning
    • Performance management
    • Security compliance
    • Support for systems management tools that monitor the operational status of the environment
