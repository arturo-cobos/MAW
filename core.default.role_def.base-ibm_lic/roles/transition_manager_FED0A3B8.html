Role: Transition Manager (040720)
This role is responsible to ensure that services projects are implemented within the terms of the contract.
Role Sets: Management, Manager
Relationships
Main Description
This role focuses on managing the project engagement with a focus on balancing organization profit and long-term client opportunity. This position develops and manages the tactical and strategic resource plan while balancing profit and utilization objectives.
