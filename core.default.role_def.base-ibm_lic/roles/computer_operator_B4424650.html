Role: Computer Operator (040709)
This role is responsible for monitoring, controlling and operating complex computer systems, networks, applications and/or distributed systems in a multi vendor environment.
Role Sets: Operations, Specialist
Relationships
Main Description
This role is also responsible for analysing first level problems for all systems (according to the service agreements), resolving and routing as appropriate, supporting the Systems Management Processes specific to their customer requirements, performing I/O (tape and print) duties where appropriate and batch management when required.
Staffing
Skills
  • Manage System change
  • Manage IAS
  • Operate Operating System
  • Operate Services Delivery Tools
  • Operate Automate Console Operations (AOC)
  • Perform problem mgmt
  • Use IBM internal General Tools
