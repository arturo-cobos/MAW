Role: User
This role represents the user of a service or product.
Role Sets: Adjunct, Any
Relationships

  • Plan User Acceptance Testing
  • Specify Training Requirements
  • Specify User Support Requirements

Main Description

This role represents the user of a solution.  The project team defines the skills and responsibilities of the users of the services and products.  These skills and responsibilities may be used as input when developing a service or product.

Staffing
Skills

The skills are defined by the needs of the project.
