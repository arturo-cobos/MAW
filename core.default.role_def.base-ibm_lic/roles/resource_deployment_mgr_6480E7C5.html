Role: Resource Deployment Manager (041619)
Role Sets: Management, Manager
Relationships
Main Description

This role is expected to: 

  • Manage project engagement staffing.
  • Develop and manage tactical and strategic resource plan and skill management process for organization.
  • Anticipate the needs of the principals and project managers on assigning resources with appropriate capabilities and required skills to be an effective contributor to a specific project. 
  • Understand the skill set of available resources throughout their assigned area.
Staffing

  • Apply Problem Solving Techniques
  • Apply Knowledge of WW CRM Tools
  • Apply Knowledge of WW Skills Management Process
  • Perform Interpersonal Communication
  • Perform Successful Negotiations
  • Perform WW CRM Process
  • Understand SKILLS Tool
  • Use Analysis/Reporting/Query Tools
