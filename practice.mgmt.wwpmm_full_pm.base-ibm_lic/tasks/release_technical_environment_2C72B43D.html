Task: Release technical environment
Disciplines: Environment
Purpose
To release technical environment elements and reconcile project assets.

      Main Description
      Identify technical environment elements that should be released and release them. Update the project Asset Inventory. Recover any delivery organization property from the sponsor (hardware and software documentation, any other assets). Return or destroy any of the sponsor's confidential material, as agreed with the sponsor. Return any other of the sponsor's assets held by the project.
