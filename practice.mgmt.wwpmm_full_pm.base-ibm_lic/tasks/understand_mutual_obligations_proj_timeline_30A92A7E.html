Task: Understand mutual obligations in project timeline
Disciplines: Sponsor Agreement Management
Purpose
To review key obligations and commitments as identified in the Agreement and understand the key events, dependencies and milestones and develop the Milestone List.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Agreement
  • Project Definition
  • Project Management Schedule
    Outputs
    • Milestone List
      Main Description
      Review the Agreement, together with the Project Definition, Risk Management Plan, and other documentation from the project Shaping phase. Understand the decision, delivery and acceptance processes.

      Extract all the milestones, and related dates, that are embedded in the sponsor and supplier Agreements, the key dependencies between project organizational units, and the dates at which resources are to be brought on the project. Be sure to check for contractual milestones that might have been added during contract negotiations and that are not a part of the technical milestones for the solution.

      Develop the Milestone List with these key events.
