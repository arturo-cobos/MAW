Task: Develop startup plan and orient startup staff
Disciplines: Work Plan Management
Purpose
To prepare the project team to begin project execution by ensuring that the plans and procedures needed to get project execution underway are in place.
Relationships
RolesPrimary Performer:
  • Project Manager

  • Risk Management Plan
    Outputs
    • Operational Schedule
      Main Description
      Determine the plans and procedures that need to be created -or expanded- to get project execution underway, establish a short-term startup plan to develop them and to set up the project management system.

      Note that an outline of some of the project plans, as well as some requirements for the project management system (in the outline of the Project Management System Summary), may have been produced when shaping the project.

      Take the necessary steps to prepare the project team for, at least, executing the short-term startup plan.
