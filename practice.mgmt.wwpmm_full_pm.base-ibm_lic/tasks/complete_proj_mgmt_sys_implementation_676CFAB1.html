Task: Complete project management system implementation
Disciplines: Quality Management
Purpose
To document the project management system that will be used to manage the project by summarizing it in the Project Management System Summary, then having it approved by relevant management (if required), and by communicating this information to key project members.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Project Management System Summary
    Outputs
    • Project Management System Summary
      Main Description
      Summarize in the Project Management System Summary the key elements of the project procedures and management plans (communications, financial, human resources, project schedule, etc.) that must be known by all project teams members, as well as those elements that demonstrate compliance with the delivery organization policies.

      Gain approval as necessary from the appropriate level of functional management, or review board, on the management system that is being implemented, in order to get project execution formally chartered.

      Brief project team members that are implicated in the project management system operation.
