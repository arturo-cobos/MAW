Task: Reconcile project management plans
Disciplines: Tracking and Control
Purpose
To reconcile project plans based upon any adjustments made as a result of corrective actions.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Operational Schedule
  • Project Management Schedule
    Outputs
    • Operational Schedule
    • Project Management Schedule
      Main Description
      Update the Project Quality Plan and Project Procedures Description and other work plans to reflect any corrective actions (cost-corrective or schedule-corrective) and to be kept consistent with the revised Operational and/or project Schedules.
