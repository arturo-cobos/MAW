Task: Develop orientation material and Team Charter
Disciplines: Human Resource Management
Purpose
To develop orientation materials and a Team Charter (if necessary) that help facilitate common understanding among team members and set the ground rules for how the project will operate.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Project Definition
  • Project Procedures Description
    Outputs
    • Team Charter
    • Team Orientation Package
      Main Description
      Assemble into a short presentation (orientation material) the information that will provide the new project team members with a common understanding of the project, including information about the sponsor, the project goals and objectives, the project management system, the key milestones and deliverables, and the project technological context.

      For large projects, and especially for virtual or international teams, create a sense of team identity by developing a short Team Charter which presents the project challenges, the team events and rewarding aspects, and the project ground rules.
