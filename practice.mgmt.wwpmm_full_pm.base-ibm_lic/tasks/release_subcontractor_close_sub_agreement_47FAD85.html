Task: Release subcontractor and close subcontractor Agreement
Disciplines: Supplier Management
Purpose
To ensure the recovery of assets from released suppliers and that proper administrative and security closure have occurred.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Agreement
  • Asset Inventory
  • Delivery Control Documentation
  • Subcontractor Status Report
    Outputs
    • Asset Inventory
    • Final acceptance for supplier
      Main Description
      Collect project assets and de-install technical environment provided to the subcontractor. Remove any access rights given to the subcontractor during the project. Perform administrative closure of the supplier Agreement.
