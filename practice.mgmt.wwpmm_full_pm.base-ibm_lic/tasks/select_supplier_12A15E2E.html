Task: Select supplier
Disciplines: Supplier Management
Purpose
To identify, evaluate, and select a potential supplier.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Organizational Breakdown Structure (OBS)
  • Project Definition
    Outputs
    • Supplier Selection Report
      Main Description
      Identify possible suppliers and define the criteria to be used in selecting one or a short list.

      Work with the purchasing department to proceed to the appropriate process for receiving supplier offers.

      Analyze and rate their proposals from the project point of view.

      Come to an agreement with the purchasing department on the offer to be selected. Document in the Supplier Selection Report.
