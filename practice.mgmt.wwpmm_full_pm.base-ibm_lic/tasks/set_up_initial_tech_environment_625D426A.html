Task: Set up initial technical environment
Disciplines: Environment
Purpose
To ensure that a technical environment is established and available to the startup team.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Operational Schedule
  • Outline of the Technical Environment Plan
    Outputs
    • PM and communication tools installed
    • Technical Environment Plan
      Main Description
      Establish the minimum technical environment (workstations, communications and project management tools) required to ensure efficient work by the startup team.

      Particularly, initialize the project control book and supporting project management tools with project related data, and identify the utilization rules that will need to be documented when performing the next task (Implement project management system).
