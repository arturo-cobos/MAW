The Service Request work product is used to record the details of work received from a client that IBM is required to implement.

Main Description
The Service Request work product is normally associated with an automatic system that allows the Service Request to be recorded and tracked throughout its lifecycle. This work product is a standard document used by the Project Office to document and track client requests from initiation through final delivery of the requested functionality. This work product is a standard element in the Project Control Book. Requests received for action under an established project or task are treated as Project Change Requests.
