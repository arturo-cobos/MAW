Role: Process Owner
Role Sets: Adjunct, Business
Relationships
Main Description

This role has responsibility for all aspects of their business processes.

Set strategy and goals

  • Set Strategic Direction
  • Provide Input On Investment Prioritization
  • Establish Process Performance Goals
  • Approve Process Designs

Create and update design

  • Create / Update Process Design
  • Provide IT solution
  • Direct Continuous Improvement
  • Deploy, Document, Educate

Perform process

  • Execute Process Per Design
  • Deliver Value

Manage process

  • Measure Results vs. Targets (e.g., metrics, audits)
  • Assess Gaps
  • Take Actions To Close Gaps
