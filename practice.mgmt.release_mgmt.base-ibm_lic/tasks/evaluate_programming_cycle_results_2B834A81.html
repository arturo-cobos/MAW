Task: Evaluate Programming Cycle Results
Review the source code, design documentation and test results of the programming cycle against the detailed goals that were set for it.
Disciplines: Technical Management
Purpose
To review the development and testing results of the completed programming cycle.

If appropriate to adjust the goals of future programming cycles based on the experience of this one.

      Main Description
      Review the source code, design documentation and test results of the programming cycle against the detailed goals that were set for it.

      Mainly this task is a review. However as a result of the review changes may be made to the goals of future programming cycles for reasons such as:
      • insight gained during this cycle shows that some future cycle is either easier or harder than previously thought
      • insight gained during this cycle uncovers a further prerequisite for some other cycle
      • if the work of this cycle was much harder than planned, it may be agreed to lessen the goals and either create an extra cycle or increase the goals of a later cycle.


      Adjust the goals for the cycle and refine them to the level of the iterations contained within it.
