Deliverable: Usability Test Report (DEL 0519)
This report contains the output of the usability test and is used to verify and refine the prototype and design.
Purpose

Used to document the results of these tests and establish whether the application meets its usability objectives through:

(a) Objective measures such as task completion time and number of user errors, and

(b) Users’ subjective perceptions of ease-of-use.

Main Description

A trained facilitator who does not have a stake in the design conducts usability testing. Participants are observed as they interact with or “drive” the prototype and execute realistic scenarios, taking particular note of comments made as they complete (or fail to complete) assigned tasks. The Usability Test Report contains the following as appropriate for the project:

  • Results
  • Analysis
  • Recommendations
