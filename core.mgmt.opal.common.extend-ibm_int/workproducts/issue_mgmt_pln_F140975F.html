Brief Outline

This plan broadly captures following information related to issues:

  • Strategy for issue analysis, ranking, prioritization and Evaluation
  • Issue management and documentation
  • Escalation mechanism
  • Roles and Responsibilities
  • Issue Review and monitoring plan
  • Communication plan

Key Considerations
This plan can be part of master project plan for smaller projects, where a standalone document for managing issues might be an overhead.
Tailoring
Impact of not having

Lack of an Issue Management Plan could result in:

  • The project being faced with issue consequences that could have been avoided or lessened through planned actions
  • The project being ill-prepared to react to issues when they do occur
  • Insufficient funds in the project budget to react to issues when they occur
