Artifact: Client Data Security and Privacy Plan (ART 0565)
This artifact documents client approved data access controls which appropriately restrict access to personal and sensitive personal information.
Domains: Security & Privacy
Work Product Kinds: Plan
Purpose
Its purpose is to provide an effective plan for managing and supporting access to the client's production systems and production data.

Description
Main Description
 
Brief Outline

This artifact typically contains following details:

  • The client's production systems and the types of data contained on the production systems
  • User access
  • The data security techniques used for controlling and restricting access to the client's production data in all environments (including development, test and production environments).
  • Training required to enable them to manage and support access to the client's production systems and production data
Illustrations
Reusable Assets
  • Client Data Security and Privacy Plan
Key Considerations

At a minimum the data security techniques must include:

  • Separation of duties
  • Storage and disposal of production data

Additional, optional techniques may include one or more of the following:

  • Data encryption
  • Data masking
  • Moving and copying production data
  • Client non-disclosure agreements
Tailoring
Impact of not having

Without this artifact, there is an increased likelihood that projects will not comply with the Key Controls Over Operations (KCOs) resulting in inappropriate levels of security and possible breach of contract with clients.

Reasons for not needing

This artifact may not be required if the client has no specific security or privacy requirements and the organization does not have access to any client personal information or sensitive personal information.
