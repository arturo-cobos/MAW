Example: Communications Management Plan (Organizational) for P3 CTRs using RPM
This provides an example of the Communications Management Plan (Organizational) for P3 CTRs, using Rational Portfolio Manager.

Main Description

This example mainly contains following information:

  • P3 CTR portfolio grouping and selection criteria 
  • Scheduled meetings 
  • First level project review 
  • Performance Critical Thread Review (2nd, 3rd level) 
  • Process Critical Thread Review (2nd, 3rd level) 
  • People Critical Thread Review (2nd, 3rd level) 
  • Reporting 
  • Reporting Metric - Project Health Definition 
  • RPM Reports and Views - ENG 348 Project Status Report 
  • RPM Reports and Views - ENG348 Program Status Report 
  • Schedule 
  • Assumptions, dependencies and risks 
  • Costs
