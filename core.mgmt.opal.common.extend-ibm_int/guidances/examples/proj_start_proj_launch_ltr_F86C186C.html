Example: Project Start (Project Launch) Letter
This is an example of the documentation that communicates to the project manager of a record for a contract that the project start (project launch) procedure will need to be executed for that project.
Relationships
Related Elements
  • Project Start (Project Launch) Letter
Main Description

This example is a communication from the delivery organization's portfolio manager to the project manager of record for a contract.

EMEA BCS Project Launch Letter V2.0
 

Dear [insert PM name],

Congratulations! Our records indicate that you have been nominated to be the Project Manager of the following new project within the BCS [insert sector name] Sector with effect from <add date>:

Project Information:

Customer:                     <customer name>
Project Name:               <project title>
Project Partner:             <Engagement Partner>
PM Name:                    <project manager of record>
Contract Number:          <contract num>
Contract Revenue:         <contract revenue>
Contract Gross Profit:    <contract Gross Profit>

As the PM for this engagement you will be confronted with plenty of activities that need to be pursued in the starting phase of the engagement. To help you launch the project, this letter provides summary project management information, guidance and advice to help you get the project off to a good start. Please make sure that you understand these guidelines and follow them in the course of the project. If you encounter any start-up issues or have any questions regarding this letter, please contact your Risk Management representatives for guidance.

If this message has been sent to you in error, please return it to either the sender or the Project Partner (as noted above).
