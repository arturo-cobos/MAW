Reusable Asset: Technical Environment Plan
This guidance links to an example of a technical environment plan for a small or medium size project. It is based on lessons learned from several projects.

Main Description

This is a self-contained and reusable document for a small or medium size project to be used as the basis for managing a project's technical environment. It meets the intent of the OPAL Technical Environment Procedure and Technical Environment Plan ENG352. The attachment in the link below may be tailored by replacing its content with the relevant project content. The Technical Environment Plan Example may be found here.
