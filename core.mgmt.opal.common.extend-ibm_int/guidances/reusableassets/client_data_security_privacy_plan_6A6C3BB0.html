Main Description

Detailed guidance is available on GBS Data Security & Privacy Control Framework in Security Plan Example Template.

Used to document the plan and controls to manage and support access to the client's production environment, or to manage access to the client's personal information, sensitive personal information and business sensitive information.

It typically includes:

  • Overview
  • Client requirements
  • User ID administration and access controls
  • Separation of duties
  • Reviews and approvals including management review

For more information, refer to GBS Data Security & Privacy Control Framework
