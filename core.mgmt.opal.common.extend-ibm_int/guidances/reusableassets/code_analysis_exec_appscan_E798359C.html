Main Description

The linked document provides guidance on how to use Rational AppScan which is a flexible, accurate, and efficient web application security assessment tool.

AppScan is a dynamic tool which helps us to identify vulnerabilities in web sites before the hackers do.

AppScan can test following applications:

  • Web Applications
  • Web Services
