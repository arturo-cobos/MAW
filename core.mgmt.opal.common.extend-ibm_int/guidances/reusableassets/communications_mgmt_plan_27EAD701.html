This guidance links to an example of a communications management plan for small or medium size projects and is based on lessons learned from several projects.

Main Description

This is a self-contained document and can serve as the basis for managing project communications. It is strongly recommended that the communications management domain in WWPMM be reviewed as this plan is developed. The attachment in the link below may be tailored by replacing its content with the relevant project content.

This asset describes:
  1. Organizational structure for the Client and IBM
  2. Communications with the Client and IBM key stakeholders and teams
  3. Meetings, reporting, and other project communication
  4. Key contact details for the client and IBM
