Main Description

The code analysis and reporting process runs the code through a targeted combination of static and dynamic code analyses to identify errors, vulnerabilities, and compliance issues prior to system wide integration testing. It identifies areas of potential errors related to poor coding practices and actual errors related to non conformance to standards across any combination of four defect domains:

  • Technical Quality and application development and maintenance Best Practices
  • Security Compliance Check
  • Memory Management Check
  • Accessibility and Universal Design Standards Check
