Template: Life Sciences MAW
This MS Excel template provides the list of work products that are applicable to various Delivery processes so that Life Sciences projects can use this information while tailoring their project defined process.

Main Description

This template captures the names of work products in each row. The delivery processes are listed in various columns. Work products that are applicable to each of these delivery processes are marked against each of these work products. This information serves as that basis of tailoring during Method Adoption Workshop.
