Template: Project Quality Plan - CMMI
This MS Word template is used to document the project quality plan, where the quality planning parameters are aligned with the practices of Capability Maturity Model Integrated (CMMI). The template is useful for organizations aiming for the maturity level 2 and higher of CMMI model.

Related Elements

Main Description

This template captures information such as:

  • Quality Objectives
    • Tailoring Performed on Quality Goals
    • Data Collection Package Update
    • Objectives / Control Limits Revision
  • Process Performance Model such as Defect Prediction Model
