Main Description

The agenda outlines the recommended items to be discussed on the process critical thread review meeting. This set of agenda items achieves the purpose of the P3 CTR which is to support communications of project status information up to the organization management levels, and to enable communications from the higher management levels to be delivered to the project.

Some of the important points discussed are action items, quality review reports, risks, business controls, Open Source Software (OSS) usage, process improvements, etc.
