Template: Project Management System Summary CMMI
This template is used to document the project plan, where the project planning parameters are aligned with the practices of Capability Maturity Model Integrated (CMMI). The template is useful for organizations aiming for the maturity level 2 and higher of CMMI model.
Relationships
Related Elements
  • Project Management System Summary
Main Description

This template captures information such as:

  • High-level Work breakdown structure (WBS)
  • Project dependency management
  • Technical environment plan
  • Process Improvement Plan
