Template: Project Defined Process
This excel worksheet template is used to assess the overall applicability of the project to OPAL (and any organization/project-specific) procedures, work products, methods, policies, standards and guidelines.

Related Elements

Main Description

The template typically contains following worksheets

  • Project details- for entering the project name, work classification etc
  • Process tailoring- for providing the procedure level tailoring information
  • Guidance tailoring- for providing the list of guidelines adopted/un-adopted by the project with tailoring information
  • Product tailoring- for providing the details of the work products chosen by the project and the tailoring information
  • Instructions- provides detailed instructions for filling up the remaining worksheets

GBS Project management model users must refer GBS PM work product usage guidance available on the IGSDF website, for work product specific tailoring options. Link to this guidance is available in OPAL common external references.
