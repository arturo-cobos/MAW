Template: Process Assurance Review Report - Project
This MS Excel template is used to record findings of process compliance review performed against the project plan specified policies and procedure.
Relationships
Related Elements

Main Description

This workbook contains multiple worksheets as detailed below:

  • Cover sheet to include project information
  • Questions sheet containing the compliance questions, compliance guidance and placeholder for entering comments and findings
  • Reports sheets to provide consolidated report at OPAL procedure level, Systems Development Life Cycle (SDLC) level, Key Controls of Operations (KCO) level  and CMMI process area level
  • Instructions and document control sheets
