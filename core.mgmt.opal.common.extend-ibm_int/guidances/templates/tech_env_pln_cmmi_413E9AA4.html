Template: Technical Environment Plan CMMI
This template documents the plan to establish and maintain the project technical environment required for enabling the project design and delivery. It is aligned with the practices of Capability Maturity Model Integrated and is useful for organizations aiming for the maturity level 2 or higher.

Main Description
This template captures information such as:

  • Hardware
  • System and application software
  • Tools and equipments
  • Facilities
  • Roles and Responsibilities
  • Cost
  • Maintenance of the technical environment
