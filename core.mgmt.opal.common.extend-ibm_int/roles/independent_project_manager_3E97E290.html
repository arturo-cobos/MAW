Main Description
The independent project manager is responsible for handling a project team in delivering a solution to the customer using the appropriate business measurements, terms and conditions.
This role has overall performance responsibility for holding reviews to validate that the project estimate and schedules, documents the results of the review in the evidence of governance. This independent PM has to also approve the overall project estimate to confirm that it is complete, fit for purpose, and ready for pricing.
