Capability Pattern: Integrate Technical Solution
The goal of this activity is to prepare an integrated build at some level. As this activity is performed in an on-going fashion, the objective of the build target may differ depending on whether the build is integrating development cycle or build cycle elements. Prior to creating the build, the procedures are reviewed to determine if the target integration level is already defined correctly for the build to proceed.
