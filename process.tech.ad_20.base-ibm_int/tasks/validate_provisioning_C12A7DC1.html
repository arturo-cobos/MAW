Task: Validate Service Usage by External Parties
Test to ensure proper functioning of service provisioning system
Purpose
To make sure service customers consume services as designed.

      Main Description

      Test to ensure proper functioning of service provisioning when the services are consumed by external users such as business partners, customers, etc. The test is to ensure that customers are accessing correct and authorized versions of services. This is not functionality test.
