Task: Perform Reusability Test
Execute tests to confirm reusability of the application being tested.
Purpose
To ensure the service/functionality/application under consideration can be reused.
Relationships
RolesPrimary Performer:
  • Test Specialist

InputsMandatory:

    Outputs

      Main Description

      Reusability is an important aspect of service oriented architecture. It is important to make sure that the code built for a service or a functionality can be reused by other existing/future applications. Executing this reusability testing is very important to ensure this requirement.
