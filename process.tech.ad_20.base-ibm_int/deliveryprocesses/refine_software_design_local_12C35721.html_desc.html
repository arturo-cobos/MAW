Activity: Develop Detail Component Design
Create specifications for the design of release components.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
This activity creates the initial set of design components from which the current release of the application can be constructed.
Relationships
Parent Activities
  • Micro Design
Description
To complete this activity perform the following tasks:
  • Establish the principles, standards, and patterns used in designing the system.
  • Specify the component design for each component to be build as part of the current release.
  • Incorporate architectural elements such as the templates and patterns into the component designs.
  • Document the dynamic internal behavior of the system components.
  • Describe the internal operation of complex components.
