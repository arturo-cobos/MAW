Task Descriptor: Complete Operational Model
This task completes both the Logical and Physical Operational Model for a given set of partitioned functionaility

Steps
LOM: Complete the logical node specifications
LOM: Review and refine placement of DUs on logical nodes
LOM: Complete the logical connection specifications
LOM: Complete the specification of technical components on logical nodes.
POM: Refine and complete the detailed physical node model
POM: Refine and complete the detailed physical specifications for nodes and connections
POM: Refine and complete the product selection
  • for nodes
  • for technical components on nodes
  • for connections
POM: Refine and complete the Physical Operational Model
Including completing the sizing of the nodes and connections
POM: Complete the configurations for nodes and connections
POM: Complete detailed designs where appropriate for nodes and connections
POM: Summarize the state of Physical Operational Model
