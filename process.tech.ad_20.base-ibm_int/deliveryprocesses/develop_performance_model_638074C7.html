Task Descriptor: Develop Performance Model
Build a performance model in order to understand and predict the ability of the solution to meet performance requirements
Based on Method Task: Develop Performance Model

Steps
Plan

The planning step will involve the following:

  • Review the available input artifacts to build up a good understanding of the system.
  • Decide what modeling tools, if any, will be used.
  • Decide what measurements will be taken and what units will be used.  Also consider what rating scales will be used for assessing severity, probability and risk with guidance on how to use them.
  • Decide what variants or configurations of the available architecture models will be used.
  • Decide what scenarios will be used to test out the model.
  • Develop an outline model, deciding on the focus areas, and document the expected contents of each part of the model.  If a modeling tool is being used configure the model ready to start executing it using different sets of input data.
Model

This step involves executing the model built or configured in the previous step.  The model is usually executed many times using different input data and possibly using different variants or configurations.  The results from each run of the model are recorded for analysis during the next step.

Once it has been decided what is to be measured and the model has been primed ready to begin execution and recording of results
Analyze
Once all variants and configurations of the model have been executed using the appropriate input data the results need to be analyzed and recommendations made as to the optimum configuration of the system to meet the service level requirement being placed upon it.  Where there is potentially more than one configuration list the options with associated risks and/or issues associated with each.
