Role Descriptor: Information Architect
This role focuses on the elements required to manage all aspects of data and information (both structured and unstructured) from business requirements to logical and physical design.
Based on Method Role: Information Architect
Relationships
Performs:
  • Assemble Data Model Deliverable
  • Detail Logical Data Model
  • Develop Common (Canonical) Message Model
  • Develop Data Migration Plan
  • Refine Conceptual Data Model
Responsible for:
  • Data Migration Plan
  • Data Model
  • Physical Database Design
Additionally Performs:
  • Identify Data Quality Concerns
Modifies:
  • Data Migration Plan
  • Data Model
  • Data Model Deliverable
