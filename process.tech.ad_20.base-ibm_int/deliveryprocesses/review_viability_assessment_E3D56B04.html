Task Descriptor: Review Viability Assessment
Review the findings of a previous Viability Assessment to ensure all follow-up actions have been addressed and no new issues have arisen.
Based on Method Task: Review Viability Assessment

Steps
Review Findings and Risks
Review the Viability Assessment report and check if findings and risks which had actions against them have been followed up.  If artifacts have been updated, look at each artifact and compare it against any checklists describing the validation and verification criteria for the artifact.  Decide if the solution is still viable in light of the contract and subsequent project change requests.  Will it still deliver the architectural qualities that are required?
Update Findings and Risks
Updates to risks and issues are documented and forwarded to the project manager for inclusion in the projectâ€™s risk and issue management processes.
Update Viability Assessment Report
The original Viability Assessment report should be updated with the results of this review.  Follow-up items should be stated in terms of owners and completion dates.  The need for an additional follow-up solution review by external reviewers needs to be confirmed and tentatively scheduled.
