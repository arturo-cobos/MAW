Whitepaper: IBM SOA Reference Architecture : Solution View
Provides background and guidance for the SOA Solution Stack (S3), a logical reference architecture that describes a Service-Oriented Architecture. It provides a high-level abstraction of an SOA factored into layers, each addressing specific value propositions within SOA.
