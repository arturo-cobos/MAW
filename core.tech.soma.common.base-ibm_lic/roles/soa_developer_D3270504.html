Role: SOA Developer (040684)
IT Specialists in this job role have expertise in translating IT requirements in the design, development, and assembly of components to create custom information systems.

This role is responsible for designing, developing and integrating Service Oriented Architecture (SOA) solutions.

    Main Description

    Typical examples of the deliverables are functional and technical designs, models, components, code, unit tests, and documentation.  IT Specialists who primarily apply their technical skills in an internal or external customer billable services and implementation environment.

    This role designs, develops and supports SOA solutions to meet client requirements. This includes:

    • Defining and implementing service interfaces  using  WSDL
    • Defining and implementing process flows using BPEL, SCA
    • Defining and implementing service messages using XML, XSD
    • Integrating existing services into new applications
