Activity: Integrate project organizational unit work plans (IMSP491)
This activity will consolidate the milestones, dependencies and schedule across each potential organizational unit of the project.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
To consolidate across each potential organizational unit of the project, resolve mismatches, and finalize the overall schedule and plans that will be proposed for delivering the project.
Relationships
Parent Activities
  • common_activities
Description

During a selling process, as described in the Client Value Method (CVM), this activity interacts with the phase "Develop and agree to client solution" within the bid phase.

This activity includes the following:

  • Reconcile project organization unit (POU) work plans and confirm the project schedule
    • Update the milestone list for dependencies
    • Review the project management schedule and adjust scope of work and milestones as needed
