Activity: Harvest Knowledge / Assets (IMSP240 and IMSP711)
This activity will organize and analyze data at the end of a project, such as stakeholder feedback, measurement data and lessons learned, and document it as needed. This activity will also identify, prepare and submit intellectual capital to an appropriate repository for reuse.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
To ensure that all project feedback and data is properly analyzed and documented, and that intellectual capital that may be of value, such as for future reuse, is identified and submitted to an appropriate repository.
Relationships
Parent Activities
  • common_activities
Description

This activity includes the following:

  • Harvest Knowledge/Assets:
    • Review and analyze the stakeholder feedback
    • Collect, compute and consolidate end-of project measurement data
    • Collect project estimates and actual data
    • Task the Subject Matter Expert (Estimating) with the collection of estimation factor data
    • Run a review to confirm the availability of project estimation data
    • Organize a review of estimation work products
    • Assess and document lessons learned
    • File project estimate and actual data
    • Review and identify intellectual capital candidates
    • Identify appropriate intellectual capital repository
    • Determine existence of similar work products
    • Request for approval to proceed
    • Prepare the proposed intellectual capital for submission
    • Organize a review of the proposed intellectual capital
    • Submit the intellectual capital
