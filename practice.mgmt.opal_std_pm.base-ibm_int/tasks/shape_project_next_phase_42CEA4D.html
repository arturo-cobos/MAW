Task: Shape project (Next Phase)
This task will document the project scope and the list of deliverables that will be committed to the sponsor, and identify the work streams or work units and related outcomes that are better assigned to specialized performing organizations to produce them.
Purpose

To document the project scope and the list of deliverables that will be committed to the sponsor, identify the work streams or work units and related outcomes that are better assigned to specialized performing organizations to produce them, and outline the decision structure that will likely assure consensus between the key stakeholders.

      Main Description

      This task is based on the WWPMM work pattern D4: Shape project, which is used here in the context of confirming project plans for the next phase, after having assessed both the business value (for the sponsor) provided by previously completed phases, and the business value expected from phases, projects, and services, either not started or under execution.

      This task is launched when the target solution (what must be delivered) and the overall approach (major steps towards delivering the target solution, outcome and decision points) are known.

      This may be the case directly upon completion of the task "Assess effectiveness of project plans", because the scope and approach information for the project in the sponsor's charter was clearly defined and looked feasible, or because the delivery organization will satisfy the sponsor's needs with a standard offering. However some minor changes in the sponsors environment may have occurred and need to be addressed. This task focuses on the scope which will be committed for the project's future phase(s), and the approach and organization to deliver it.

      As a result, this task defines the timeline and the framework within which each project organizational unit should deliver their contribution to the execution of the project, before launching the next activity where these organizational units will be asked to build their plans and confirm that they can deliver within the framework constraints.

      The key outputs of this activity are the new or updated Project Definition and Work Product List.

      The task includes the following:

      • Refine project approach and planning framework
      • Adjust project organization and future sub-projects
