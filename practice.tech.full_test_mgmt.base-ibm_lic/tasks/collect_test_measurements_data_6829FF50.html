      Main Description
      Test measurements provide information on the effectiveness and efficiency of the Test Process and are used as input to support management processes and decisions. Measurements are used for many different purposes:
      1. In process measurements help with management decisions as testing progresses
      2. Measurements for planning purposes help to improve the estimating process for future projects
      3. Measurements data help in assessing how well we did on the project - in improving the estimating guidelines
      4. Measurements help in tracking progress against plan
      Steps
      Collect measurements from the detailed test results data
      Consolidate them at the project level
      Document all relevant information
