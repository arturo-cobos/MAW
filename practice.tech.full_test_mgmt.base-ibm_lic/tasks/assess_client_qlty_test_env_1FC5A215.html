Task: Assess Quality and Test Environment
Analyze existing quality and test environment, identify gaps, and make recommendations for the shortfalls.
Disciplines: Test Management
Purpose
To identify those aspects of the current quality and test environment that expose the organization to significant risk and/or cost that need to be addressed for successful test and quality efforts.
Relationships
RolesPrimary Performer:
  • Test Architect
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Test Process Evaluation Report
    Outputs
    • Test Process Evaluation Report
      Steps
      Analyze existing quality and test environment
      Review the quality and test organization charts, processes and procedures, environments, standards and templates, practice guidelines, etc. and interview key people to identify the key quality and test strengths and weaknesses.  Using the questions in the evaluation report, gather data on the key processes and practices.
      Identify quality and test gaps
      Identify and evaluate the gaps in the current quality and test components, theory or practice that could cause problems in executing test strategies and plans. Confirm the findings and assessments with key stakeholders.
      Address quality and environment issues
      Make recommendations for those shortfalls of the quality and test that need to be addressed outside the project and draw up action plans for those components that can be covered as part of the test strategy and plan.  Put these recommendations and plans forward to the appropriate audience and/or sponsor, along with some assessment of the risk in not acting upon them.
