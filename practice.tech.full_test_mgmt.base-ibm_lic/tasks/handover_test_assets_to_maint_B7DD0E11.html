Task: Handover Test Assets to Maintenance
Once the tested solution has been implemented in production and all Test Assets have been documented and filed, it is time to handover the Test Assets to the group or organization that will be responsible for maintaining them.
Disciplines: Test Management
Purpose
To formally document agreement, and handover all Test Assets to Maintenance function for future use.
Relationships
RolesPrimary Performer:
  • Test Architect
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Test Findings
    Outputs
    • Test Findings
      Main Description
      Although there is normally no formal sign off, it is important for the developers who are handing over the results and the Test Manager or other developers and testers receiving the results and the report to understand and acknowledge that the test assets being handed over have been completed as stated.
      Steps
      Review the detailed list of Test Assets and the final test report
      Gain consensus for the recommendation to move to the next level
      Document the handover and any associated assumptions or conditions
