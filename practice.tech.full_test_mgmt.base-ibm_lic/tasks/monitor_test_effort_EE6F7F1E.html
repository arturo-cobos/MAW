Task: Monitor Test Effort
Focus on making timely changes in the form of metrics-based corrections to the test effort in order to increase its effectiveness.
Disciplines: Test Management
Purpose

The purpose of this task is to:

  • Make an assessment of the productivity, effectiveness, and completeness of the test effort
  • Make adjustments to the test effort (both tactical and strategic) to improve effectiveness
Relationships
RolesPrimary Performer:
  • Test Architect
Additional Performers:
InputsMandatory:
  • Test Log
    		Optional:
    • None
    Outputs
    • Test Evaluation Summary
      Steps
      Capture work status
      Gather test effort productivity and effectiveness metrics
      Gather change request distribution, trend and aging metrics
      Gather traceability, coverage and dependency metrics
      Evaluate metrics and formulate initial assessment
      Record findings
      Plan and implement improvement initiatives
      Monitor and support improvement initiatives
      Evaluate and verify your results
