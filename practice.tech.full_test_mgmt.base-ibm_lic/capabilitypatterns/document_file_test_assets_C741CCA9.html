Task Descriptor: Document and File Test Assets
For each of the test levels, the Test Assets would have been catalogued and filed as the tests were completed. After the project or release is implemented, it is time to do the final documentation of the remaining Test Assets and file them away for future re-use.

Steps
Establish a method for documenting the catalogues of Test Assets
 This includes all the Test Plans, Test Specifications, Test Environments, and Test Cases, Test Data, Test Reports  and so on.
Include any Test Reports and other assets left out in previous Cataloguing
Organize assets for easy retrieval - catalog and file
Remove all other components and documentation as per storage/disposal guidelines
