Practice: Full Lifecycle Test Management
Full lifecycle test management represents the best practices for managing a variety of stand-alone testing projects or the full lifecycle testing of development projects. It is independent of technology platform or solution development methods used.

Purpose
This practice forms the basis for managing the overall test effort when using a full lifecycle testing approach to testing on a project. It is sufficiently flexible and scalable to support different types of software development projects.
Main Description

This practice supports multiple entry points appropriate to each project, and it provides guidance to establish the building blocks of a test project. The practice is not prescriptive but provides for tailoring of the method to suit individual project needs and facilitates the asset based approach to method adoption based on the inclusion of other more specific testing practices. Moreover, the practice identifies the minimum level of structure and control required to manage and track testing activities.
