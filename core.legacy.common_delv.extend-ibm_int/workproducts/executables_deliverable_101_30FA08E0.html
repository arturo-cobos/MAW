Deliverable: Executables Deliverable (DCD 101)
This deliverable will consist of - Executable files for the release, Runtime code libraries, Customizable executable scripts, and Supporting files, as needed.
Purpose
This deliverable represents the automated functions for the solution, which are defined by the use cases. This is the primary software deliverable for the release. This deliverable also will include all required supporting files for the executable software.
Deliverable Properties
External Description

IBM will build the source code to meet the programming goals and produce the executables.  All executables will be unit tested with the results of those tests being documented elsewhere.

The customer is responsible for maintaining the development environment to ensure necessary availability for the development team.

Deliverable Parts
  • Build
Packaging Instructions

This deliverable serves as a formal packaging for the Executables work product.

IBM will deliver one copy of all executable software and all associated supporting files for the release.
