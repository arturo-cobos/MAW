Deliverable: Production Environment Readiness Report (DCD 080)
This document will consist of an assessment of the overall production environment readiness and recommendations for customer actions that are required to be completed prior to cut-over to make the environment production ready.

Deliverable Properties
External Description

This deliverable will assess the setting up of the hardware and software infrastructure, the installation of the integrated subsystems for production, and verification of the system readiness.  Additional guidance will be provided to enable the customer database administrator with creating and protecting the final solution database, migrating the data, and verifying the database readiness.  Once the production environment is configured the IBM team will conduct a series of tests to assess the readiness of the production environment and produce the Production Environment Readiness Report.
 
The customer is responsible for:

  • Procuring defined hardware and software, install the hardware and software, install integrated subsystems test and certify/document production environment readiness.
  • Creating and populating the final solution database, migrating production data, and certifying the actual database readiness.
