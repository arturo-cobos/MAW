Deliverable: Fit Gap Analysis Deliverable (DCD 019)
The Fit/Gap Analysis Deliverable is document that describes the analysis of a package in relation to the client’s future business processes.
Purpose
The purpose of this deliverable material is to describe in detail the functions, transactions, screens, reports etc. that have been selected to support the execution of the work steps of the business process. The analysis will also highlight the gaps where work steps are not fully supported. They will include proposals on what might be the change (the process or the package), and the implications of such a change.
Main Description

The Fit/Gap Analysis Deliverable should include:

  • Analyzing the package and business processes
  • Options for resolving the gaps between package and processes including estimates of effort to implement them
  • Implications of implementing the options
  • Selection of strategies based on the options
Deliverable Properties
Deliverable Parts
  • Detailed Gap Analysis
Packaging InstructionsThe Fit/Gap Analysis report will consist of a series of documents organized as a report document of approximately 10 – 20 pages.
  • Name: The business process or package module being analyzed.
  • Summary Results: A scale or narrative conclusion as to the suitability of the package to the intended business process purpose. Detailed Results: A series of points, questions, or capabilities that were tested against the package, and the results of these tests. The results should be detailed enough to be self-explanatory. The results should articulate any issues, which will result from the implementation of the package, such as the need for new skills, a change in the process flow, or a change to the package.
  • Gap Analysis and Strategy: Each gap is described fully in this section with a preliminary strategy explaining how it might be addressed. These explanations need to be of sufficient detail to be used by other project team members who are responsible for resolving the gap. The effort should be estimated if possible.
