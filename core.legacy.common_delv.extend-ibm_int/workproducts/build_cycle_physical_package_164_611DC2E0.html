Deliverable: Build Cycle Physical Package (DCD 164)
The Build Cycle Physical Package is a collection of physical artifacts intended to be deployed to targeted end users.
Purpose
The purpose of this deliverable is to provide the client with a package comprised of the release application code and installation media that will be delivered for end user installation and the operation and execution of those portions of the solution.
Main Description
The deliverable / report may contain the following content, if appropriate:
  • Installation Media
  • User Support Materials
  • Registration Instructions
  • The source code developed for the Release
  • The executable application code for the Release
  • Technical documentation for the application
Deliverable Properties
External Description

The Build Cycle Physical Package is a collection of physical artifacts that is intended to be deployed to targeted end users.

The purpose of this deliverable is to provide the client with a package comprised the release application code and installation media that will be distributed for end user installation and the operation and execution of those portions of the solution.

The deliverable / report may contain the following content, if appropriate:
  • Installation Media
  • User Support Materials
  • Registration Instructions
  • The source code developed for the Release
  • The executable application code for the Release
  • Technical documentation for the application

Note: A representative sample master copy(s) of the Physical Package will be provided for the purpose of client acceptance. Production of the actual number of packages required will be identified in the Deployment Plan.
