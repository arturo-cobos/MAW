Main Description

Estimation considerations include:

  • Number and variety of modules to be transitioned to maintenance.
  • Number, variety and complexity of assets types (checklist items) to be addressed by the AMTP DCD will also govern the resource estimate, as each asset type will require acceptance and evaluation criteria.
  • Pages of documentation supporting the enhancements and maintenance service request items to be transitioned.
  • Total number of service requests and problem incident reports to be transitioned in a release, if appropriate.
