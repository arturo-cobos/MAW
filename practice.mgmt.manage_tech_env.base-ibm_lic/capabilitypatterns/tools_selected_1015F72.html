Key Considerations

It is preferable to start with a set of established tooling standards and mark any tools that aren't applicable and note any deviations from the standard.

When selecting tools make sure that the cost of the tooling is appropriate to the size of the project and that the tooling does not have any runtime requirements that make it's use impractical (e.g. required network connectivity where not available or installation of server applications not allowable by current policy, etc.).
