Activity: Establish Initial Technical Environment
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose

To finalize any tool selections, update the overall technical environment plan and setup any tooling required for the initial design/development effort.

Relationships
Parent Activities
  • Starting
Description
The Technical Environment Plan was created, or at least outlined, when Shaping the project. At this point, that plan must be refined and implemented, and the elements of the technical environment that are needed to actually start work must be made available to the project team. However, it is likely that the bulk of the required activities to actually install and configure the environment will take place in later activities in the project.
