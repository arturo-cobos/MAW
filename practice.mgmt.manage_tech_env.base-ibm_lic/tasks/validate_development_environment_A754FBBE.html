Task: Validate Development Environment
Disciplines: Environment
Purpose
The purpose of this task is to provide confirmation that the development environment and infrastructure can support system development.
Relationships
RolesPrimary Performer:
  • Application Developer
Additional Performers:
Outputs
  • Environment Validated
    Main Description
    The development tools and environment in general should be exercised before scheduled development is attempted in order to mitigate risk to project schedule. A small team of individuals who are knowledgeable in the tools and project objectives should be assembled to run the tools and test the interfaces of the environment. Issues encountered should be resolved before the first programming cycle.
