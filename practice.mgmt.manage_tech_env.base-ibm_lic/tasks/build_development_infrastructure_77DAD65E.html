Task: Build Development Infrastructure
Deploy the hardware and software platforms, operating systems, and connectivity tools necessary to support the development tools. Ensure that the infrastructure resources are sufficient to support the development activities.
Disciplines: Environment
Purpose
The purpose of this task is to put the hardware and software platforms into place that are to support the development tools and the actual development activities.
