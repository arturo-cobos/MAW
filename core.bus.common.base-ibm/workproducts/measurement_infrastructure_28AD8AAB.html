Artifact: Measurement Infrastructure
This artifact is the actual hardware and software portions of the measurement system. It consists of the data repositories for storage of metrics information and the software to generate metrics reports.
Purpose
  • To store the raw data used to generate the metrics
  • To store rollups of data so that trend analysis can be performed
  • To generate measurement reports
Tailoring
Impact of not havingThis is a critical artifact; you must have a place to store the measurement information.
Representation OptionsThis information is normally captured in a spreadsheet or data repository.
