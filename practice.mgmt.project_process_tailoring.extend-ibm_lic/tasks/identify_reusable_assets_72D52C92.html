Task: Identify Reusable Assets (Asset Consumption)
Leverage existing assets for use on this project
Disciplines: Asset Management
Purpose
To locate and list existing assets that may leveraged on this project.
Relationships
RolesPrimary Performer:
  • Technical Solution Architect
Additional Performers:
InputsMandatory:
    • [Project Work]
      • Operational Schedule
      • Project Management Schedule
      • Work Breakdown Structure (WBS)
    		Optional:
      • [Technical Architecture]
        • Architectural Decisions
        • Architecture Overview
        • Component Model
        • Deployment Unit Model
        • Infrastructure Gap Analysis
        • Operational Model
        • Performance Model
        • Reference Architecture Fit Gap Analysis
        • Security Architecture
        • Technical Proof of Concept
        • Viability Assessment
      Outputs
      • Candidate Asset List
        Steps
        Identify where assets can be leveraged on the project
        Locate existing available candidate assets
        Search existing asset repositories based on the identified requirements and determine with the asset owner whether that asset is potentially reusable within in the context of this project.
        Evaluate the fit of the existing asset
        Determine whether the existing asset meets the needs of the current project and whether any required customization is possible
