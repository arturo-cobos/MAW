Task: Confirm MAW Results
Confirm the scope of the project by reviewing the results of the Method Adoption Workshop.
Disciplines: Technical Management
Purpose
Make sure that tailoring decisions are still valid for the next piece of work planned.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Defined Process
  • Work Product List
    		Optional:
      • [Project Definition and Scope]
        • Project Definition
        • Work Product List
      Outputs
      • Project Defined Process
      • Work Breakdown Structure (WBS)
      • Work Product List
        Main Description
        Review the outcome of the prior method adoption workshop to determine if any corrections need to be made based on new information obtained, changes in assumptions or other events that would invalidate the previous decisions.  Examples of changes may include adding work products to cover new risks or removing work products no longer required.
        Steps
        Examine project risks and revise their priorities
        Review the scope of the project
        • Are there some work products that can be skipped without undue risk?
        • Are there dependecies among the work products that need to be managed?
        • What are the high priority items that need to be developed?
        • Is the project still feasibile given the time and budget constraints?
        Obtain consensus from the sponsor on the project scope and cost
