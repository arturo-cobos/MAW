Description
Main Description

The Compliance Incident Log is the list identifying all the compliance incidents, the sponsor or supplier involved, the owner, and the status.

Brief Outline

The Compliance Incident Log contains these components:

  • List of compliance incidents
  • Compliance incidents statistics

Key Considerations

The Compliance Incident Log is updated each time a Compliance Incident Document is created and again when the status changes.

The Compliance Incident Log must be up-to-date and consistent with the corresponding Compliance Incident Documents, if any were created.

Tailoring
Impact of not having

Without a Compliance Incident Log there is no visibility regarding the number of open compliance incidents and their status.

Reasons for not needing

The Event Log or Project Status Reports can be used to record compliance incidents on simple projects where there are not many compliance incidents.
