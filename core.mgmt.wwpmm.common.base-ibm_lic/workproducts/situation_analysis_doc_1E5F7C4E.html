Artifact: Situation Analysis Documentation (ENG 354)
A document which is used when there is a need to investigate a potentially troubled situation and identify the appropriate actions to resolve it. This document records the situation analysis preparation and the resulting report.
Domains: Project Management
Work Product Kinds: Record
Purpose

The Situation Analysis Documentation supports the analysis that is launched when there are symptoms of a potentially adverse situation.

Key Considerations

The Situation Analysis Documentation initially contains the plans for the analysis. The report is filled in throughout the analysis as interviews are held and conclusions are drawn.

Tailoring
Impact of not having

Without the Situation Analysis Documentation:

  • the analysis of the situation will be performed without a methodical approach
  • there will be no record of the conclusions that were drawn from analysis of the situation.
Reasons for not needing

This work product is always required when analysis of a situation is performed.
