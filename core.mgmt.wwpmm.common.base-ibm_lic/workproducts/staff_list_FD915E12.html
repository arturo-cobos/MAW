Artifact: Staff List (ENG 405)
A list that contains basic information about each team member such as identification categories, information about the sourcing organization, and cost related historical items.
Domains: Project Management
Work Product Kinds: Record
Purpose
The purpose of the Staff List is to keep the minimum amount of information that is required about each individual who is working on the project.

Description
Main Description

The Staff List contains basic data about each member of the project team.

Brief Outline

The Staff List contains three components:

  • Individual’s identification
  • Sourcing organization
  • Costs (if under the project manager’s control)

Key Considerations

A Staff List record is created at the time the individual is selected to join the project team and is updated when necessary thereafter.

The Staff List contains an accurate, up to date record for each member of the project team.

Tailoring
Impact of not havingLack of a Staff List could result in unnecessary delays because a required piece of information is not at hand.
Reasons for not needingStaff List is not needed on a small project with short duration.
