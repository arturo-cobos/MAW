        Main Description
        Plan for performance testing which is often similar, and sometimes overlaps, other types of testing, but is focused on understanding the performance of the system being tested under certain conditions.  There are typically two desired results of a performance test.  Understanding how a system performs is the first stage; improving that performance is often the second stage.
