Purpose
Provide a viewpoint specific model for analysis of performance objectives for an operational architecture.
Main Description
This practice provides a performance specific viewpoint model to substitute for a general SLC model. This provides extended modeling capabilties for analysis of performance capabilites of a proposed operational architecture against a set of well defined performance requirements.
