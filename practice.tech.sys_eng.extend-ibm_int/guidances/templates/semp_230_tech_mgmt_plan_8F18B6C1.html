Template: Technical Management Plan for SEMP
The Technical Management Plan (TMP) Template is the complete description of plans, processes, and procedures required for the team to create and maintain the solution definition, development, test, integration, acceptance, and technology refreshment for project execution.
Relationships
Related Elements
  • Systems Engineering Management Plan (SEMP)
Main Description

The TMP integrates the SEMP with other technical planning aspects to provide a single deliverable with all technical plans. For large, complex projects, including application development, package implementation, or systems integration projects, the TMP Template should be used.
