Task: Prepare for Preliminary Design Review
This task describes how to formulate the guidance for Preliminary Design Review including setting the agenda, scheduling so that the appropriate attendees are available and developing the PDR.
Disciplines: System Engineering
Purpose
To prepare the PDR (Preliminary Design Review) template for the review.  The Preliminary Design Review is used to establish the component baseline.
