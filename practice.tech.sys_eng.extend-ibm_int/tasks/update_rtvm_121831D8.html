Task: Update Requirements Traceability and Verification Matrix
Update the Requirements Traceability and Verification Matrix for the system requirements.
Disciplines: System Engineering
Purpose

The Requirements Traceability and Verification Matrix (RTVM) serves two purposes:

1) Provides traceability of requirements through the life cycle.

2) Provides a verification matrix that shows how each requirement will be tested, when each requirement will be tested, and the test acceptance criteria for each requirement.

      Main Description
      The document is created with details for the system requirements as part of the system baseline and reviewed at the System Requirements Review. The document is then updated with details for the component requirements as part of the component baseline and reviewed at the Preliminary Design Review.

      See the Requirements Traceability and Verification Matrix template and guidance for assembly details.
