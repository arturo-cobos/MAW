Task: Prepare for Test Readiness Review
This task describes how to formulate the guidance for Test Readiness Review including setting the agenda, scheduling so that the appropriate attendees are available and developing the TRR.
Disciplines: System Engineering
Purpose
The purpose of this task is to prepare for the TRR including setting the agenda, scheduling so that the appropriate attendees are available and developing the TRR (Test Readiness Review) template.  The Test Readiness Review is used to establish the test baseline.
