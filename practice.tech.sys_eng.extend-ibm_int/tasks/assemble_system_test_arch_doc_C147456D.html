Task: Assemble System Test Architecture document
The System Test Architecture document defines the system test architecture. This architecture is a subset of the System / Component Architecture document.
Disciplines: System Engineering
Purpose
This task defines the process of developing the System Test Architecture document, which describes the architecture needed for system testing.
