Task: Start PE Plan
Create the initial draft of the PE Plan, which documents the “what” (scope and objectives), the “why” (purpose), the “when” (key milestones), and the “who” (organizational roles and responsibilities) for performance engineering across the project phases.

      Main Description

       The initial draft of the PE Plan may be created here, which can be spelled out in full details in the System Baseline phase. This plan is a document of understanding on performance engineering between the various stakeholders of the project.  The PE Plan document should contain PE Goals, PE Focus Areas, Development Phases, PE Themes and Activities, Organizational Responsibilities, Performance Decision Checkpoints, PE Methods and Tools, Metrics, PE Data Management Strategy, PE Milestones, PE Management Procedures and PE Reporting Procedures.
