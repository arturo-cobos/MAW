Task: Prepare for System Requirements Review (SRR)
This task describes how to formulate the guidance for System Requirements Review including setting the agenda, scheduling so that the appropriate attendees are available and developing the SRR.
Disciplines: System Engineering
Purpose
Prepare for the SRR (System Requirements Review) meeting.  The System Requirements Review is used to establish the system baseline.
