Task: Update Technical Solution Decisions Document
The Technical Solution Decisions document records important decisions about any aspect of the Technical Solution including the structure of the system, the provision and allocation of function, the contextual fitness of the system and adherence to standards.
Disciplines: System Engineering
Purpose
This task defines the process of updating the Technical Solutions Decisions Document with decisions made since the document was baselined as part of the system baseline.

      Main Description

      The document is created as part of the system baseline and reviewed at the System Requirements Review. The document is then updated as part of the component baseline and the updates are reviewed at the Preliminary Design Review.
