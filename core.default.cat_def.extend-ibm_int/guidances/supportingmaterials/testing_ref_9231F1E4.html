IBM Internal Lotus Notes Databases

Test Process Model V3; IT Consulting Methods Database; classification IBM Confidential; authorization required; contact Testing Methods/Southbury/IBM@IBMUS

This database contains all the activities and tasks that would be performed for a typical or standard testing engagement. The content includes a purpose and description that provides overview information, as well as steps to execute the work.


Testing Methods & Practice; Intellectual Capital Management (ICM) AssetWeb Knowledge Network Database; classification IBM Confidential; authorization required; contact ICM Info/Mount Pleasant/IBM@IBMUS

This database is designed to support the development and growth of the testing practice knowledge network. It provides a repository for the most current, best-of-breed client engagement examples, and intellectual capital submitted by IBM testing practitioners. Registration and access is via the ICM AssetWeb Navigator.

OVUM Software Testing Tools; commercial database purchased by IBM Software Test, contact Andrew Williams/Australia/IBM

This database provides test development tool evaluation information published by OVUM, a UK based consultancy (http://www.ovum.com/), which publishes their report Software Evaluations for Testing Tools on a regular basis.  The OVUM database was purchased by IBM Testing Services to aid in the selection of testing tools.  The report contents are available via Lotus Notes.  Open the database by selecting filename, t_dir\tools\ ts3_dmp.nsf, on server ATE01DB/D/ATE.  See “Using this database” for locations of other servers.

Testing Tool Knowledgebase (TTK); sponsored by IBM Software Test, contact Andrew Williams/Australia/IBM

The Testing Tool Knowledgebase (TTK) is a Lotus Notes application, which gathers information about testing tools in use by IBM and IBM Global Services during the testing phase.  The objective of this database is to become a vital reference tool for all technical testers in IBM whenever the question of testing tool occurs.  This database is sponsored by IBM Software Test IBM Testing Services.  Open the database by selecting filename, t_dir\tools\ttk.nsf, on the server ATE01DB/D/ATE.  See “Using this database” for locations of other servers.


IBM Intranet Sites

IBM Methods & Tools; http://w3-5.ibm.com/transform/project/pmtoolsuite/
This site features access to the latest project management and technical method as well as tools that can be downloaded for project control, planning, and execution.

IBM Software Test; http://w3.research.ibm.com/softeng/Testing/Testing.htm
The purpose of this site is to bring together the people, processes, and practices of the Software Test community. Links to other internal and external sites provide additional information regarding test tools, test methodologies, vendor information, publications, and education.

IBM Software Testing Technologies Group;
This site was set up as a result of requests at Software Testing Symposiums for a common source of software testing information. It contains contacts, papers, and information on specific topics, as well as supplying links to sites that have software testing information. The goal of this site is to become a central repository for information on software testing.
