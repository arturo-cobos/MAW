Main Description
The complexity of this work product will depend on the scope of the effort. Complicating factors include (but are not limited to) the amount of content, content stability, content complexity, access to SMEs, number of changes that have been requested by the client, size of the learning solution, the type of learning solution that is to be delivered, and the amount and type(s) of supporting media that are to be produced. Time required to complete, review, and approve Learning Solution Materials will depend on each of these factors.
