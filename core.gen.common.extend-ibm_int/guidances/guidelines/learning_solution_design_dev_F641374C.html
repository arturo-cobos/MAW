Guideline: Developing Learning Solution Design

Main Description

1. Identify Instructional Objectives
2. Create Detailed Course Outline
3. Determine Theme and Treatment
4. Determine Look and Feel.
5. Create a detailed Outline

Identify Instructional Objectives

Using all available documentation produced during the analysis and course content source material, identify Instructional Objectives for the course. These objectives outline the performances that learners will be expected to carry out at the end of the learning solution.

Create Detailed Course Outline

Create the structure by which content will be presented to the learner, and determine the way in which Instructional Strategies will be incorporated into the learning solution. For example, if Instructional Path Models are to be used, determine the way in which they will be structured and incorporated into the design.

Determine Theme and Treatment

Determine the learning solution's Theme, and document this along with a Treatment for keeping media in line with this theme. For example, if characters are to be used, how will they be presented in the course?

Determine Look and Feel

Once a Look and Feel has been designed for the learning solution, work to create samples to include within the Learning Solution Design. For example, if the solution will include learner handbooks with screen grabs of an application, create a sample page to include in the Learning Solution Design.

Create a detailed Outline

Create blow-by-blow design, including:

  • Practice Items -- How will learners get a chance to practice using the content they learn?
  • Test Items -- How will learners be assessed to prove they have mastered the Learning Objectives?
  • Interactivity -- What kinds of opportunities for interaction will the learners be offered?
