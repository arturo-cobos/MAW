Welcome
The Unified Method Framework is the overall brand used to describe the set of Delivery Processes that have been combined under a single architecture, built off of shared reusable components utilizing a common set of standards. The Framework is for use in Solution Design and Solution Delivery.
Main Description

This is the Application Development 2.0 UMF delivery process, release 1.11. This delivery process was developed by the Application Innovation service line of IBM Global Business Services for use as the standard method in developing IT solutions for services engagements. For an overview of the phases comprising this delivery process and their characteristics, see the AD 2.0 Lifecycle Overview guideline.

To navigate directly to this delivery process, click here: Application Development 2.0

This delivery process was developed by a core team of GBS Process, Methods, and Tools staff in conjunction with advice from a global team of AIS practitioners and guidance from AIS executive sponsors. 

The current release of AD 2.0 is 1.11, released in September 2012. The initial early adopter release of this delivery process was released in November, 2010. For additional release details on this delivery process, please refer to the Release Info entry for Application Development 2.0.
