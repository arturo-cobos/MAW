Task: Perform Perfect Hindsight Estimate
At the end of an engagement, the estimator is asked to generate a closing estimate containing the actual metrics experienced on the project.
Purpose
To perform a perfect hindsight estimate based on the project actuals in order to gather metrics to improve future estimating capabilities and accuracy.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Estimation Report
  • Project Management Schedule
    		Optional:
    • None
    Outputs
    • Estimation Report
      Main Description

      This 'estimate' is refered to as a 'Perfect Hindsight Estimate' because all of the information is known. This information is then sent to a central repository for analysis and calibration. This function serves a very critical need and yet at the same time is actually quite simple to perform.
