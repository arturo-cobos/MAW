Task: Develop Top Down Estimates
Create an initial estimate for the project.
Purpose
To produce an overall estimate for the project based on a limited set of project parameters.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
  • Application Architect
InputsMandatory:
  • Project Definition
    		Optional:
    • Work Product List
      Outputs
      • Estimation Report
        Key Considerations
        At the start of the project, an initial estimate is performed.  Of necessity, the information used to generate this estimate will be minimal. The estimate therefore will have a low confidence rating. It is important however to conduct this initial estimate in order for the project executive to understand the potential range of effort that will be required to complete the project.
