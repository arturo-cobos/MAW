Task: Prepare for Deployment
This task is used for preparation of launch package, and transition to maintenance.
Purpose
To assure all steps are planned in order to execute the deployment of the solution as flawlessly as possible.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
  • Application Architect
  • Application Developer
  • Business Analyst
  • Stakeholder
  • Test Specialist
Outputs
  • Deployment Plan
  • Launch Package
  • Project Management System Summary
    Main Description
    The Teams works together to develop a list of actions that need to be performed, in what order, by whom on the team or maintenance team and the expected duration of each effort. The team may include a decision matrix that provides team members with tips on which deployment method, contact or options, which might be available while deploying the system.
