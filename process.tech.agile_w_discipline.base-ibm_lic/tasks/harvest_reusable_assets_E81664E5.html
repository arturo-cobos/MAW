Task: Harvest Reusable Assets
Determine which components have been selected to become reusable assets, as listed on the Asset Harvesting List in the Candidate Asset List.
Purpose
Perform harvesting activity to produce a base of high quality reusable assets for future use.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Candidate Asset List
    		Optional:
    • Work Product List
      Outputs
      • Assets Harvested
        Main Description
        The work product instances for this phase should be archived or stored in a central repository so they can later be posted the required asset library . In later phases, further identify those components that have been developed as part of this release, isolate the appropriate implementation artifacts for these components, such as source code and executables, and package it for transmission to the asset library as directed by the receiver.
