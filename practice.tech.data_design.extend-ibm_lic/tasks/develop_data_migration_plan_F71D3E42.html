Task: Develop Data Migration Plan
Document the plan for migrating data whether automated or manual.
Disciplines: Technical Design
Purpose
The purpose of this task is:

  • To ensure that representative data and processes are available for the testing and acceptance of the data transfer programs.
  • To identify, plan, and document both the data and the processes that need to be in place to successfully migrate/load and accept the data required by the business to run the solution in a production environment. This is demonstrated initially in the System Integration Test.
  • To plan for / ensure the availability of the required system capacity in order too handle the production migration/load of the data, and execution of the conversion programs and interfaces.
