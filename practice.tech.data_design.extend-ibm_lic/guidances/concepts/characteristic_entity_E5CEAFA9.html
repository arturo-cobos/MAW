Concept: Characteristic Entity
A characteristic entity helps describe (i.e., is characteristic of) an entity in some way.
Relationships
Related Elements
  • Develop Conceptual Data Model
Main Description
A characteristic entity exists with the entity it is describing in a parent-child relationship. The entity being described is the parent and the characteristic entity is the child. Usually, characteristic entities have a one-to-many relationship with the parent entity (i.e., one parent has many characteristic children). However, it is possible for a one-to-one relationship to exist. Characteristic entity names should be nouns and they often include their parent names as a modifier.
