Task: Decide on Buy or Build or Enhance Data Sources
Make decision on whether to buy, custom build, or modify existing data sources to provide necessary information services.
Purpose
To make realization decisions for data sources.
Relationships
RolesPrimary Performer:
  • Information Architect
Additional Performers:
  • Software Architect
InputsMandatory:
  • Data Model
  • Data Stores
  • Service Model

    • Security Architecture

Optional:
  • None
Outputs
  • Architectural Decisions
    Main Description
    Evaluate options, pros and cons for realization options and make a decision on whether to buy, custom build, or modify existing data sources to provide necessary information services. In complex environments, multiple solutions may need to be applied according to requirements and constraints. An outline of the preferred approaches and the general solution aspects to or criteria by which they will be applied is necessary.
    Steps
    Identify alternate data stores including external data sources
    Abandon information service if no data source available and unwilling to build
