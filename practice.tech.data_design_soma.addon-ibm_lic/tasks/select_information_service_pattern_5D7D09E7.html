Task: Select Information Service Pattern
Select which information service pattern is most appropriate to realize an information service.
Purpose
To leverage industry proven information service patterns.

    Main Description

    There are various patterns to address problems in the SOA context. This task helps the architect to determine based on non-functional requirements which pattern is most appropriate for a given problem domain. This will then help the architect to start with the realization of the solution.
