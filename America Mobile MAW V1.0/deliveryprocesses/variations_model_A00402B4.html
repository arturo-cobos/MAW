Work Product Descriptor (Artifact): Variations Model
The Variations Model identifies aspects of a problem domain that are subject ot change and variation, and separating them from the non-changing aspects. Subsequent software design can then be performed in a manner that is sensitive to these variations using Variation Oriented Design (VOD).
