Task Descriptor: Refine Project Estimates
Revise the overall project plan based on the information acquired during this phase. Rework the bottom up estimate based on tasks in the plan.
Based on Method Task: Refine Project Estimates
Relationships
RolesPrimary:
  • Project Manager
Additional:
  • Application Architect
Assisting:
InputsMandatory:
  • Estimation Report
Optional:
  • Project Definition
  • Project Management Schedule
External:
  • None
Outputs
  • Estimation Report
Steps
Reestimate the workload based on the work products and major deliverables
  • Determine if additional tasks are required
  • Determine if planned tasks are no longer required
Determine required resources based on the revised estimates
  • Using the re-estimated workload, estimate the hardware resources.
  • Using the re-estimated workload, estimate the manpower and determine when the manpower is required.
  • Calculate the indirect manpower (management, development support, and other indirect manpower) by using a percentage of the total manpower or the work items.
  • Using the manpower and hardware resource re-estimates, estimate what facilities and consumable supplies are needed.
Review and adjust plans as necessary

Review the work breakdown structure in the overall project plan by comparing the re-estimated workload and hardware resources with the previous estimates in the overall project plan for manpower and schedules. If a discrepancy is found:

  1. Identify the cause.
  2. Provide an explanation to the sponsor and related organizations.
