Task Descriptor: Identify Common Interactions
Recognize the opportunity to create or utilize a set of templates or patterns that can provide for repeatable and consistent definition and structure throughout the solution architecture
Based on Method Task: Identify Common Interactions

Steps
Identify characteristic issues
These are issues which will encountered repeatedly in the design of the system (e.g. transactionality, persistence, workflow, user interface behavior)
Identify potential architectural templates

Decide which of these require an architectural template. Use architectural assets, literature on industry standards, and pattern libraries to identify the basis for the template.
