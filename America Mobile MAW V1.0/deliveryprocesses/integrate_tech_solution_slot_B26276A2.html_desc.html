Activity: Integrate Technical Solution
This is a parallel activity to the programming cycle, usually resulting in some level of completion of the integrated build of one or more solution components.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
Create an integrated set of release components for independent testing and possible identification as a release candidate.
Relationships
Parent Activities
  • Build (only for Portal & Service Frame Work)
Description

This activity addresses:

  • Creating an integrated build of the release representing the development efforts of one or more programming teams, creating a testable set of solution components.
  • Performing string testing on these integrated components to determine the progress made towards the goals of the programming cycles as well as towards the overall build objectives.
  • Evaluating the test results to determine if the integrated build is ready for independent testing to evaluate progress towards the final build results, which would be a potential candidate for deployment of the release.
