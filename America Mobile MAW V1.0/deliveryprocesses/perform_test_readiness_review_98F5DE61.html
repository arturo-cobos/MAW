Task Descriptor: Perform the Test Readiness Review
This task describes how to review environment before proceeding with user acceptance testing.
Based on Method Task: Perform the Test Readiness Review
Relationships
RolesPrimary:
  • Test Architect
Additional: Assisting:
InputsMandatory:
  • None
Optional:
  • Test Plan
  • Test Specification
External:
  • None
Outputs
  • Test Readiness Review Results
Steps
Establish a Test Baseline.
Verify that test entrance criteria has been met for each application to be tested
Verify test environment readiness
Verify test data readiness
Verify Test Team readiness
Verify that the defect estimates are valid and that actuals are being tracked
Verify that the Quality Model in the Quality Management Plan has valid defect estimates and that actual vs estimated defects are being tracked and analyzed.
Obtain customer and Project Manager approval to start test execution
