Edit Media Content
Media content may be edited to:
  • More clearly communicate the ideas in the Interactive Concept
  • Better convey tone, voice, and mood
  • More effectively delivery a brand message
  • Increase or decrease overall content size
  • Better adhere to User Interface Design Guidelines
Proof Media Content
Text content may consist of documentation, HTML, graphical text and/or text residing in a publishing application.  This content is proofed for spelling, grammar, punctuation, and adherence to style guidelines if they exist.
Cut Final Media Content
Use technical expertise to separate entire screen designs into programmable media content that deliver the highest possible throughput and quality. The programmable graphic elements are called "cuts" and the corresponding technical support documentation consists of "electronic cut guides". An electronic cut guide is an overlay for every view of the application and contains the size, position, and file name of each graphic element (cut) comprising the view.

This task includes the following steps:
  • Establish naming convention
  • Measure and draw cut guides
  • Typeset programming information on cut guides
  • Generate 24-bit master graphic cuts
  • Compress graphic cuts to appropriate programmable format
  • Unit test compressed graphic cuts
  • Generate test page code
  • Unit test page load
  • Customize and tweak cuts
  • Deliver cuts and guides to technical transfer point
  • Enforce design fidelity with the technical team
  • Verify cut integrity
Optimize Media Content
Compression and format guidelines are established in the Interactive Concept, and should have been anticipated. Apply various techniques that vary by medium, some common examples include:
  • Perform compression on still graphics, using selected compression schemes. (See task: Cut Final Media Content)
  • Cut graphics to optimize compression schemes. (See task: Cut Final Media Content)
  • Create technical palettes and covert still and motion graphics files to conform to the palettes. (See task: Cut Final Media Content)
  • Compress Video or Motion Graphics, using selected compression schemes.
  • Reduce frame rate for video or motion graphics.
  • Compress Audio using selected compression schemes
  • Reduce audio sample rates or bit-per-sample size.
  • Convert Media Content to different file formats.
This task differs from the task: Cut Final Media Content, in that it deals with a wider variety of Media Content types, and is not specifically targeted toward elements of the graphical user interface. This task covers the general case, while Cut Final Media Content covers a more specific case that has been highly streamlined because it is very frequently used.
