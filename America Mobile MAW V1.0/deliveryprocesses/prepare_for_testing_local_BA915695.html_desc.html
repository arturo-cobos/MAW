Activity: Prepare for Testing
Prepare for integration testing of specific programming cycles and for full independent testing of specific build objecives.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
The purpose of this activity is to:
  • Define the specifications for development test
  • Define the execution sequence and contents of tests
  • Establish pass-fail criteria for individual tests
  • Establish entry/exit criteria for each test
Relationships
Parent Activities
  • Build (only for Portal & Service Frame Work)
Description
Develop detailed test execution specifications to test valid and invalid input and output conditions based on the detailed test plans and specifications. This includes handling of any unexpected input and output conditions. These test execution specifications may take the form of test tool scripts used to perform the various tests, or of setup and execution instructions for performing tests manually where automated tools are not utilized.

The test execution plans involve developing a step by step description of tests to be performed including data reduction and analysis for setting up regression test data, and define evaluation criteria for tests.
