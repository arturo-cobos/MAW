Outcome: Current Application Information Gathered
Current Application Information is a set of materials which explain current application system. Collected materials should be indexed and filed.
Work Product Kinds: Completion
Purpose
Practitioners should understand the current application system for analysis and design using the Current Application Information.
Main Description

This is a collection of current application system materials such as;

  • Application process flow
  • Business processing manuals
  • Application forms, document, and screen formats
  • File or database formats
  • System operation manuals
  • Operation procedures
  • Manuals describing the background of the current system
  • Glossary of application terms
  • etc.
