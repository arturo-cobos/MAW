Artifact: Vendor Reference Guide (BUS 075)
The Vendor Reference Guide provides the client with an overview of how to conduct reference evaluations and an initial listing of questions that the client may use to ask each reference.
Domains: Selection
Work Product Kinds: Measurement
Purpose
The Vendor Reference Guide provides a structured approach for the client to check each vendor's references, either via telephone interview, through E-mail, or via mail/fax.
Description
Main DescriptionThe questions in Vendor Reference Guide investigate the references in the areas of business, technical, and general functionality as well as the reference's experience with the vendor, their customer support and the implementation process.
Brief Outline

The Vendor Reference Guide should be in a document containing the following information:

  • An overview briefly describing the purpose of the document and instructions to the evaluators.
  • A template containing information to be filled out by the evaluator regarding the customer reference contact.
  • A list of references received from each vendor.
  • A list of questions discussing the reference company's background, general information, training and support issues, technical performance, implementation issues and cost information relating to the vendor's software.
  • A template summarizing the reference check evaluation.
  • A section for additional written comments.

Key Considerations
The questions should prompt quick and concise answers in order to obtain valuable information without requiring a large amount of time.  Also, the questionnaire should be in a format that can be electronically distributed to the references.  It should be encouraged to distribute the questionnaire to multiple contacts within the Reference Company.  Normally, several contacts with different expertise will be required to answer all of the questions.
Tailoring
Impact of not havingAdequate and appropriate questions may be overlooked during the interview if the Vendor Reference Guide is not used.
Reasons for not needingA Vendor Reference Guide will not be necessary if the client has already established a strong list of questions and is knowledgeable in conducting reference evaluations.
