Outcome: Network - Workstation Infrastructure Installed
Network/Workstation Infrastructure is the required set of network equipment, network services and workstations that are needed at all remote locations and datacenter.
Work Product Kinds: Installation/Configuration
Purpose
The Network/Workstation environments are essential for the efficient work of the project.
Main Description
Each network environment may contain some or all of the following:
  • Network hardware, both WAN and LAN
  • LAN Operating software
  • Network management tools
  • ERP workstation software package (if required)
  • Optional middleware/database software
 
This work product also includes the set of documentation and actions to install the network/workstation environments.
Tailoring
Impact of not havingThe package cannot be implemented without the server environments.
Reasons for not needingNone.
