Main Description

The Information Security Assessment work products are developed using various steps depending upon the scope of the engagement. These include:

  • Review of the client’s IT security processes and related documentation in the following areas: Security Management, Security Organization, Asset Classification and Control, Personnel Security, Third Party Security, Physical and Environmental Security, System Security, Subsystem Security, Network Security, Application Security, Business Continuity, and Compliance.
  • Analysis of the information from interviews with key managers and process owners against the corporation’s standards, IBM’s own internal standards and a standards-based model of "best practices". Social engineering infiltration testing can also be included in addition to interview based data gathering.
  • Report detailing observations and recommendations made during the review, and a management presentation outlining identified strengths and weaknesses relating to the reviewed security processes and procedural compliance.
