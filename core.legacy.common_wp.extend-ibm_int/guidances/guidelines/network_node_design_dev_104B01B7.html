Main Description

The first step in creating a Network Node Design is to collect and review any existing design documentation, such as conceptual and specified network design outputs.  They will be used to create and validate the more detailed design levels of the Network Node Design.

The next step is to create the Network Node Design.  This step now adds specific vendor products to the design picture.  If the client already has a preferred vendor this step may be relatively easy â€“ work with the preferred vendor product specialist to identify the vendor products that best meet the network functionality and sizing requirements.  If the client does not have a preferred vendor, this is the point where the consulting team may propose helping the client create an RFP using the client network requirements for functionality and sizing as key selection criteria.  Once the vendor is then chosen the more detailed physical design work can be completed.
