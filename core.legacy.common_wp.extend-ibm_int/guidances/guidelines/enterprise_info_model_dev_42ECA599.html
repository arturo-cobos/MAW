Main Description

The Enterprise Information Model will be developed using a mixture of interview, workshop and analytical techniques.

A suggested approach is shown in the following work breakdown:

  • Activity: Create Information Model Baseline
    • Task: Identify Candidate Information Subject Areas
    • Task: Build Information Model Baseline
    • Task: Create quality criteria for evaluating the defined Information Model
  • Activity: Develop Information Model
    • Task: Identify & Define the major Information Subject Areas
    • Task: Identify & Define the Information Subject Area relationships
  • Activity: Complete Information Model
    • Task: Refine & Complete Information Model
    • Task: Present Information Model

This work product will be developed in parallel with the business process definition effort. Each of these will provide feedback/input to the other; e.g., the development of the information model will suggest activities that are required in the process model. The detailed execution of the development approach is documented in the Enterprise Information Modeling guideline.
