Main Description
  • Documentation of IT Service Description is executed in the initial phase of the service monitoring project.
  • In workshop mode, with the client executive sponsor and IT management team participating, the target service and its business attributes are identified and documented.
  • In a subsequent workshop, attended by technology managers and technical support personnel, the infrastructure dependencies of target service are identified, documented and decomposed in stages to the technology layers targeted for monitoring.
  • In subsequent phases of the project this information will be used to select and customize specific software and hardware monitoring tools.
