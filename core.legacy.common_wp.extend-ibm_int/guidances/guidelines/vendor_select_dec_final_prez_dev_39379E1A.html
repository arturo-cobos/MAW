Main Description

The IBM selection team is responsible for the development of this work product.  The client team lead should be involved in the development of this work product for acceptance and accuracy.

Create the following sequenced slides for presentation to the client.

  • Overview of the Process and Sources
    • This section steps the client through the final selection process and the steps taken to complete the vendor review.

  • Vendor Identification Summary
    • This section outlines what was done to identify the interim list of vendors for evaluation by the project selection team.

  • Request for Proposal/Solution Response Summary
    • This section details the vendor responses to the RFP or RFS.

  • Demonstration Performance Summary
    • This section presents the score results from the software demonstrations and how well their products met the client's future system needs.  The vendor scores are presented in a scoring model to compare each vendor's product to the client's business processes.

  • Reference Check Results
    • This section overviews the observations and findings of the vendor reference checks.

  • Initial Cost Estimates
    • This section details the initial cost estimates gathered from the software vendors.

  • High Level Gap or “Pros and Cons’
    • This section presents a “first glance” at each short list vendor's strengths and weaknesses.

  • Technical Considerations
    • This section contains any technical constraints or considerations associated to the short list vendors.

  • Elimination Criteria and Justification
    • This section outlines the reasons for elimination of short list vendors.

  • Finalist Recommendation
    • This section explains the final recommendation based on all information available on the short list vendors.
