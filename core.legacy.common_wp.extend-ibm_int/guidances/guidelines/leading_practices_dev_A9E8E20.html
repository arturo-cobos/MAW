Main Description
  1. Identify the Focal Area(s) for which Leading Practices will be researched.
  2. Determine and gain agreement on the appropriate level of detail for gathering and documenting the Leading Practices.
  3. Using secondary sources of research, scan for and review information that appears relevant to the Focal Area(s).  Key sources to consider include IBM’s leading practices databases, trade and business journals, consulting studies, external databases, on-line sources, and the project team’s knowledge.
  4. Document the Leading Practices using the agreed to documentation format and standard.
