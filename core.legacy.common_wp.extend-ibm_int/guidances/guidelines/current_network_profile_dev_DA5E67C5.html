Main Description
  1. During engagement planning, decide what attributes to include in the Current Network Profile.  A logical view of the enterprise needs to be selected and level of detail that will result in gathering sufficient data to draw meaningful conclusions needs to be chosen.  For instance, a network strategy engagement requires a broad spread of information about the enterprise but limited detail.  An engagement with the objective of producing a detailed design for a new application would have a narrow focus but possibly require considerable detail.
  2. Examples of a logical view of the enterprise are (1) by department or function for a single site, (2) by building for a campus or (3) by city for a WAN.  Examples of network attributes are the topology, architecture, vendor and service providers, performance/capacity, busy time and service levels.  The goal is to organize collected data into manageable categories with characteristics that have been clearly identified as relevant to the engagement objectives.
