Main Description
  1. It is assumed that an initial set of hypotheses is brought into an engagement as existing intellectual capital. An entire set of hypotheses should not be created as a part of the engagement.
  2. It is assumed that time spent to organize a facilitated session is not included in the development time for this work product.
  3. About a day should be spent to prepare for the content for the facilitated session, a day or two spent on conducting the session and the remainder of the week spent to document and confirm the results. Therefore, on a typical project, the Hypothesis and Data Framework work product can be completed in one week.
