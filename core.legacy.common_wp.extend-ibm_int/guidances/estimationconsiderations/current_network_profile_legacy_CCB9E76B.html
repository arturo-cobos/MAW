Estimation Considerations: Current Network Profile (Legacy)
The Current Network Profile estimation considerations is developed in conjunction with multiple data gathering profiles focused on the client’s enterprise and external factors that may influence network recommendations.

Main Description

Data collection profiles focused on the client’s enterprise might typically include a Current Network Profile, a User Profile for categorizing and characterizing groups of users in an organization and an Application Profile for organizing data about current and future applications. Data collection profiles focused on the external environment might include a Network Regulatory Profile showing the available services in countries where the client operates or an Industry Networking Profile showing the business use of networks in a client’s industry. An estimate for creating this set of profiles including row and column definitions is approximately one day. 

Completing the profiles is part of the data-gathering phase of the engagement. The initial data gathering phase typically can take from one to three weeks depending on how many interviews are done, how many locations are visited, how much revalidation of current data is required, etc.
