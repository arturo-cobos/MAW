Main Description

Effort for this work product is proportional to the size and complexity of the project.

Average hours are the followings:

  • Data Flow Diagram.

- Context Diagram:

Allow 10 hours per current physical model.

Allow 4 hours per current logical model.
Allow 7 hours per new logical model.
Allow 7 hours per new physical model.

- Level 0 Diagram:

Allow two or three times more than context diagrams for each model.

- Level 1-n Diagram:

Allow 7 hours per diagram for all the models.
  • Process Description.
Allow 5 hour per process.
  • Data Flow Description.
Allow 0.5 hour per data flow.
  • Data Store Description.
Allow 5 hour per data store.
