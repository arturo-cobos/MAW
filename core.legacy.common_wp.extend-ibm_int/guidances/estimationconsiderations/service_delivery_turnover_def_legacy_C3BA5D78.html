Main Description
Consider these factors when estimating the effort for creating the Service Delivery Turnover Definition:
  • Size and complexity of the services to be supported.
  • Size of the team.
  • Scope of the contract and included work-types.
  • Process maturity.
  • Availability and Accessibility of the information required to complete the task.
  • Previous experiences with the same account.
  • Means of documenting: On-line or hard-copy.
Three to four person-weeks based on the above factors plus the effort to elicit the information if not available from the contract plus the implementation effort if an on-line database is to be created. This estimate assumes use of established procedures and methods. For instance, if this is the first Service Delivery contract for an IBM organization, it will take time to put the necessary organization in place and deploy available procedures and the IBM Global Services Method.
