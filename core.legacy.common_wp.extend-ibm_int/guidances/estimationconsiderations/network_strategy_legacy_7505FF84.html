Main Description

Constructing a Network Strategy is highly dependent upon a number of factors, including:

  • Network geographic breadth (city, state, region, country, global).
  • Completeness, accuracy, and currency of company and IT strategy. 
  • Network technology/media scope (data only or an integrated voice/data/video network). 
  • Client culture relating to following standards, policies, and guidelines. 
  • Topology scope (LAN only or LAN/WAN/fax/voice).
  • Number of interviews, visits, surveys, etc. needed to collect necessary data.

With these qualifications taken into account, an estimate could fall within the following parameters: 

Step 1: Review Business Environment 1-4 weeks

Step 2: Review Network Environment 2-6 weeks

Step 3: Review Industry & Competitor Environment 1-3 weeks

Step 4: Construct the Network Strategy 2-5 weeks
