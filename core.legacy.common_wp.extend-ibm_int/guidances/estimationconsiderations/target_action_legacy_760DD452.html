Main Description

Consider the following:

  • the size and complexity of the application
  • number of potential candidates
  • size of maintenance log and backlog of service requests
  • time required to estimate the target actions
  • time required to evaluate and set priority for implementing target actions

The time required to produce this work product may vary from 2 to 10 days.
