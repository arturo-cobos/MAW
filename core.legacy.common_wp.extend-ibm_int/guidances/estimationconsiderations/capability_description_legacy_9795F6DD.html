Main Description
The time to describe all the attributes for a single capability, assuming that all the necessary information is available, is about one hour. Given that the number of capabilities in an average capability model is typically anywhere from 50 to 100, the initial set of capability descriptions could take one to two and a half weeks.
 
If additional information has to be collected for a capability, then an additional half-hour for phone interviews should be factored into the estimate.
 
Finally, the production and distribution of a report for review by the client, plus making changes to reflect client comments should be factored at 50% of the time to create the initial draft.
 
On an average engagement, the total time to produce a final set of capability descriptions would take about four person-weeks.
