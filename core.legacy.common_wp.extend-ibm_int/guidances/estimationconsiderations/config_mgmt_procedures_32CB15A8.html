Estimation Considerations: Configuration Management Procedures
Determining the effort to develop the related configuration management procedures.
Relationships
Related Elements
  • Configuration Management Plan
Main Description

It is assumed that the project manager has access to one or more of the referenced materials. The effort to build the procedures is then mainly dependent on the number of components for which a procedure must be specified.

Early in the engagement:

  • Allow 1 to 2 days to select an appropriate procedure template and basic documentation

For any phase delivering configuration items:

  • Allow 2 hours to 4 hours per item requiring configuration management, depending on the number of persons to be involved in sign-off and administration of each procedure
