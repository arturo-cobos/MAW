Main Description

This is a detailed analytical task that requires consensus between the process expert and the package expert. The scope of these estimation considerations is confined to analyzing gaps, reviewing options, selecting the resolution strategies, and making an estimate of the effort required to implement the resolution strategies. The scope does not include the effort to implement the resolution strategies selected.

The primary units of work are a function of:

  • The number of processes and packages in the scope of the project
  • The time to review each process and package
  • The number of gaps for which resolution strategy options must be defined
  • The time to analyze the options and select the resolution strategies
  • The time required to estimate the effort to implement the selected resolution strategies
  • The number of people who need to be involved in the sessions

While each engagement will have different profiles with respect to the units of work mentioned above, the range of effort for completing Detailed Gap Analysis could be in the following range:

  • High: 24 hours
  • Low: 8 hours
