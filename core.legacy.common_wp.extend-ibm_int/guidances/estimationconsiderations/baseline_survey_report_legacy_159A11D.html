Main Description

Where available, similar historical data may be relied upon for estimating the Baseline Survey Report task otherwise plan for 3-15 days depending on the:

  • Availability and accessibility of the information required to complete the task
  • Size of the engagement scope or application system’s relevant documentation to be covered
  • Complexity of the project
  • Writer’s level of contextual knowledge (e.g. Organization)
  • Writer’s level of understanding in the pertinent application areas to be covered
