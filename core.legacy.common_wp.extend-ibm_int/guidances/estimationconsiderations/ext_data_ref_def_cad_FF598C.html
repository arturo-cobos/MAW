Estimation Considerations: External Data Reference Definition (Custom Development)
Estimating considerations are for a multi-release application development project. You should make appropriate adjustments for the type of project for which this is being applied.

Main Description

Consider an estimating guideline of 4 hours effort per reference plus an additional 40 hours to prepare the work product. If the current source or target system environments are particularly antiquated and knowledgeable personnel are scarce, it may be necessary to add a one-time overhead hours effort estimate to account for difficulties in obtaining current system information.
