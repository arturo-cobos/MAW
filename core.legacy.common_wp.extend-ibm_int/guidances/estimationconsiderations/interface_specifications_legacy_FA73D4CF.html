Main Description
The effort involved in producing this work product is influenced by:
  • The number of interfaces both internal and external.
  • The number of unique transactions conducted across each interface. This may be derived from the use case descriptions, requirement statements, interface diagrams, APIs, package documentation, or standards definitions.
  • The number of entity references per transaction.
  • The number of error conditions for each interface.
The effort is also affected to a significant extent by the technical complexity of the application.
