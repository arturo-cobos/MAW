Main Description

Significant factors:

  • The technical complexity of the solution (for technical complexity rating, refer to definition in Architectural Decisions, estimating considerations)

- A complex solution is more likely to encompass a greater number of architectural issues for which this work product might be useful.

- A complex solution is more likely to have more architects and developers working independently, with the requirement this work product to ensure that their work is mutually consistent.

  • The technical complexity of the chosen templates (the number of components within it, the number of key concepts to be defined and documented) more complex templates will require greater effort to define and document
  • Templates (design patterns) are reusable. A factor would be availability and quality of existing templates.
