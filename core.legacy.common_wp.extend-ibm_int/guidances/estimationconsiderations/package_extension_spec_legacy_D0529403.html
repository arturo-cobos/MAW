Main Description

The effort involved in producing this work product is influenced by:

  • The number of extensions to be specified
  • The number of unique transactions conducted with each extension, this may be derived from the use case descriptions, requirement statements, interface diagrams, APIs, package documentation, or standards definitions
  • The number of error conditions for each interface

The effort is also affected to a significant extent by the technical complexity of the application.
