Estimation Considerations: Application Programming Interface (Legacy)

Related Elements
  • Application Programming Interface
Main Description
The effort required to produce an API is influenced by:
  • The number of operations documented in the API.
  • The extent to which the intended users are familiar with the application (the less familiar, the more effort is required on descriptions, background information, and ensuring understandability).
  • Whether the API is deigned to be used "as-is" (by writing client code that calls it) or whether the reader needs information on how to customize the API. Having to provide customization information increases the effort by a factor of at least 2.
  • The extent to which the work product can be generated from definitions already captured in the tool set.
