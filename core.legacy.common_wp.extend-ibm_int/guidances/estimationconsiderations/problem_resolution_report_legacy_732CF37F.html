Main Description

There is no purpose to estimating the time needed to complete the Problem Resolution Report, as it is a byproduct of task(s), which must be performed under any circumstances.   

For resource estimation purposes this document can be measured in the same context as a "log book." Each entry will take 2 - 5 minutes to enter (but entry is done in parallel with other activities.) There is no basis for estimating the number of entries that will be logged.
