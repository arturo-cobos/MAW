Main Description
The quality of the existing metrics will vary widely by company and industry. If it is generally available, this is a consensus building process that should take no more than 2-3 weeks. If it is not available, the data collection could take months all by itself, and may not justify the investment.
 
This would be very short time frame to set up a survey, approach companies, conduct survey and analyze results.
