Template: Release Plan
Spreadsheet for tracking the status and reviews for a planned releases.
Relationships
Related Elements
  • Release Plan
Main Description

This template helps to establish a control over project release by capturing following information: 

  • Release ID or Name
  • Release Description
  • Planned release date
  • Related Change Request
  • Priority
  • Current Status
  • "Release Components: (Requirement IDs/CR #/Defects #)"
  • Release Type (Full/Patch)
  • Release Test Results
  • Remediation Plan (in the event the release has to be withdrawn or backed out)
  • Reviewers
  • Review Date
  • Review Actions
  • Actual Release Date
