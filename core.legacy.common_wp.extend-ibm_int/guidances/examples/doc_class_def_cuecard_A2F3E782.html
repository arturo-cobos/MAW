Cue Card
Identifying Information



Title Cue card

Summary 		This class contains a cue card intervention.



Source Details




Source reference
Cue cards are an invention of the web-based intervention delivery system that presents the information managed by this information management system. See the current release of this delivery system for examples of cue cards.
Cue cards were characterized in the overhead projector slides prepared during the document analysis meeting of 18 September 1996. These notes are not available electronically.

Source example
No electronic examples are available.



Class Details




Description
This class is a document that contains useful information of a general nature to the associate. It is help information for the SHARE information delivery system.

Applicability
This class is only used by the SHARE system.

Context Notes
This class is a complete document and has no context in the sense of being included by containment or reference. It is used in the context of the SHARE system and contains the answer to a question as part of the help information for the SHARE system.

Content Notes
A cue card intervention is a document composed of the following major sections:
· Title
In this context, the title may be generated when the document is created based on the attributes that are set. Its primary purpose is to give the human author feedback regarding which cue card intervention is being edited.
· Cue card body
This element contains the body of the cue card information.
· Sources
This is a list of sources used during the creation of the cue card. It is not currently intended for either impact analysis or for presentation to end users.

Processing and Presentation Notes
The cue cards are presented as web pages in the HTML format. They must be translated into HTML prior to presentation. Each contained element will be mapped to an HTML element, if it is not already defined as such. See the other individual object classes for more information about the specifics of this translation.
