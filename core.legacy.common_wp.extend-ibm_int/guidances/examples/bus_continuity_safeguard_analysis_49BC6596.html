Main Description

The table provides the client with a guide as to which recommendations should be addressed and in what order. The primary logic used was lowest cost to implement with highest return on investment. In addition, those recommendations that are easily implemented with minimum cost are given higher priority. The basic priority scheme is:

  • High return on investment, greatest exposure, or easily implemented with little or no cost.
  • Moderate return on investment, moderate exposure.
  • Low return on investment, low exposure.
  • Long term capital expense, moderate exposure, difficult to implement.
