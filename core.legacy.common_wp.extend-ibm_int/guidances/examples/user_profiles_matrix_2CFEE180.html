Main Description

This matrix is from a motor vehicles web application. As can be seen, candidate User Profiles are listed in the first row (System administrator, Call Center rep, Law enforcement agent, insurance agency staff) and User Profile attributes are listed in the first column (Prior Knowledge and Experience, Physical Characteristics, Social and Physical Environment, etc.).
