o Split userID and password in two different emails.

o Send only the password, the userID is already known.

o Send a new password and force the user to change it at the first login.

o Send a link with a one time token to a page where the user can change his/her password.

o Communicate the password by phone.

o Send and encrypted message (PGP or S/Mime)
