Example: Competitor Analysis - Competitive Positioning
This example describes how enterprise market positioning can be analyzed using various sources.
Relationships
Related Elements
  • Competitor Analysis
Main Description

Analysis in this area can include evaluating:

  • Key market segments versus competitors.
  • Market share in each segment versus competitors.
  • Revenue in each segment versus competitors.
  • Profitability in each segment versus competitors.
  • Expected change in the future (growth, stability, decline, etc.).
  • Strengths, weaknesses, opportunities & threats (SWOT) in each segment.
