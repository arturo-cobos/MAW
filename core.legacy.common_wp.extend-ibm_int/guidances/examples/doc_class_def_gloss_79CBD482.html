Identifying Information

Title Gloss Definition
Summary This class contains a definition for a term within a gloss.

Source Details

Source reference		 Glosses and gloss definitions were characterized in the overhead projector slides prepared during the document analysis meeting of 18 September 1996. These notes are not available electronically.
Source example No electronic examples are available.

Class Details
Description This class contains a definition for the term this is found within the same gloss containing the definition.
Applicability N/A

Context Notes		 This element defines the meaning of the term that is contained by the gloss that contains it. There may be more than one definition for a single term. When referencing a gloss, the target for the reference is this element, the definition, rather than the term, since referencing the term or the entire gloss would be ambiguous.
Content Notes This element contains the text of the definition and may contain a list of other related information and a list of sources for the information found in the definition.

Sample

<gloss content=gloss> 

                 <glossterm>Registered Owner</glossterm> 

                 <glossdefn id=regowner001            

                            group=afs            

                             legal=nonlegal>  

<defn>   

<p>The legal owner(s) of the shares as listed in the

<term defnid=registra001>registration</term>. 

The owner could be one of the following:    

           <ul>     

                 <li>One or more people</li>     

                 <li>A business</li>     

                 <li>A retirement plan</li>    

           </ul>   

</p>  

</defn>   

<relatedinfo>   

<li><term

defnid=custodia001>Custodian</term>   

<li><term

defnid=sharehol001>Shareholder</term>  

<sources>   

<li><cit>Dictionary of Finance and Investment Terms</cit> 

</glossdefn>

</gloss>

Processing and Presentation Notes		 Generally, when presented, this objectâ€™s content is presented indented a fixed amount that is normally large enough that it is to the right of the term it defines. If this definition is the first definition, it is presented on the same line as the term if this is supported by the rendering mechanism.
The Sources content may or may not be presented depending on the specific presentation context (associates may not need to see the source for the definition information, while authors may need this information).
This definition object should be the target for links to definitions for a specific term since this will unambiguously indicate which definition applies to a specific use of a term.
