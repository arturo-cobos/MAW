Example: Customized Software Image
This example provides information regarding the different types of Software images.
Relationships
Related Elements
  • Customized Software Image
Main Description

Software images will normally contain:

  • A copy of the chosen operating system.
  • Device drivers specific to the equipment being deployed.
  • User specified software.

The images may also contain:

  • Netware.
  • Communication protocols.
  • User rights/privileges.
  • Specific encryption levels/requirements.
  • Desktop icons.
  • Firewall software and settings.
  • Anti Virus software and settings.
  • Client proprietary software.
  • Internet/Intranet configurations.
