Check Items
Each process maps to a set of modules
Every process in the functional model is implemented by one or more modules in the Program Module Invocation Model
Each module exhibits a fan-out of no more than seven, approximately
Each module exhibits intuitive cohesion, whether functional, sequential, or logical, etc.
Each module exhibits low levels of coupling
Global data and execution jumps should seldom be allowed.  Formal interfaces should be minimal.
Each data aggregate in an interface has been evaluated as to whether it is appropriate
Aggregates should simplify understanding of the interface without unnecessarily increasing inter-module dependence.
Each module has been documented in a module specification
The module interface should be documented in the interface section of the specification.
