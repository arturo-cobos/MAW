Task: Assign Service Integration Mechanism
Assign/designate service integration mechanism to mediate, route, and transform service requests between service requestors and service providers. Enterprise Service Bus (ESB) is one example that fulfills the responsibility of this layer. This is a key enabler for SOA.
Disciplines: Technical Architecture
Purpose
To enable mediation, routing and transformation capabilities.
