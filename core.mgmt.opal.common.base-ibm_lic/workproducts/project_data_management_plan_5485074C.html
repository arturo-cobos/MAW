Artifact: Project Data Management Plan (ENG 429)
This artifact documents each data element with its purpose, distribution plans, and security requirements to ensure that the data elements identified for the project will fulfill the needs of the project.
Domains: Data
Work Product Kinds: Plan
Purpose

Its purpose is to:

  • plan and manage all the data elements that can be an input to or an output from the project
  • identify the purpose for this data and the distribution plans of these data elements to fulfill the needs of the project
