Outcome: Work Effort
A work effort performed during the project lifecycle.
Work Product Kinds: Completion
Purpose

This outcome is a result of reviewing the work to determine the actual work effort needed to complete the project. The work effort is used for assistance in the Work Classification. (See Work Classification Guideline)
