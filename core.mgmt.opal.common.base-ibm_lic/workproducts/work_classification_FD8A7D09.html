Outcome: Work Classification
This outcome is to verify that the work classification (understand the project's classification for tailoring, type of work, and level of project management to apply) has been determined for the service request or project.
Work Product Kinds: Confirmation
Purpose
The purpose of this outcome is to verify that work classification has been determined for the service request or project.
Relationships
RolesResponsible: Modified By:
  • Project Manager
TasksInput To: Output From:
  • Determine the work classification (IMSP491 and IMSP454)
Main Description

The classification is used to:

  • Understand the project's classification for later use when tailoring the project 
  • Determine the type of request and the type of work 
  • Classify the service request as a work effort or project, based on estimated effort and risk 
  • Determine the level of project management to apply

For assistance with the work classification, see the guidance attached to this outcome.
