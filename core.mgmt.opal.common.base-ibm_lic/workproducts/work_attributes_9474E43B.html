Outcome: Work Attributes
This outcome is to verify that work attributes have been assigned when completing a work classification.
Work Product Kinds: Confirmation
Purpose
The purpose of this outcome is to verify that work attributes have been assigned as an evaluation of the work classification. This is generally done by risk assessment, effort estimate, and other criteria in accordance with the organizations work classification criteria.
Relationships

Main Description

For assistance with the work attribute assignments, see the guidance attached to this outcome.
