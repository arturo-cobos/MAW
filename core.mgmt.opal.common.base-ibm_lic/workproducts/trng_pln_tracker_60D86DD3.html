Artifact: Training Plan and Tracker (ORG 418)
This artifact captures current skills, required skills for executing the project, associated training needs and a plan to provide required training.
Domains: Learning
Work Product Kinds: Assessment, Plan
Purpose

The purpose of this artifact is to ensure that project resources have required skills to perform their role. This includes following:

  • Identifying skill gaps (current skill vs. required skills) within a project
  • Planning and tracking the training needs to closure
  • Assessing the effectiveness of training by evaluating preparedness and skill level of resources for executing the project
  • Documenting the waiver details incase waiver is taken
Description
Brief Outline

This artifact captures one or more of the following information:

  • Identified roles or resources within a project
  • Current skill levels
  • Required skill levels
  • Identified training needs
  • Type of training such as Class Room / Mentoring / On the Job / Self Study
  • Waiver associated with a training, if applicable
  • Technique used to assess the skill level after training
Illustrations
Templates
  • Skill Tracker for Annuity Project
  • Training Plan and Tracker
Key Considerations

This artifact can be part of the master project plan document or can be cross referenced from existing planning document. 

For annuity projects which have wide ranging application specific skills and need to have a comprehensive compilation of current technical skills of project resources can do so by additionally using a separate skill tracking mechanism.

Tailoring
Impact of not having

Absence of this artifact can lead to following consequences:

  • Skill gaps within a project may not be identified thereby impacting productivity of project resources and the deliverable quality
  • Identified training might not be provided or could be ineffective thereby impacting both productivity and deliverable quality
Reasons for not needingThis artifact is not needed when skill gaps are addressed using an existing project planning document.
