Outcome: Project Procurement Needs
This outcome reflects the need to understand what is required for a project to facilitate the supplier procurement process.
Work Product Kinds: Completion
Purpose
Its purpose is to ensure that what is required for a project is understood and considered for the impact it has on the procurement process.
