Template: Application Recovery Plan
This template is a formal documentation of the plan recovery in the case where disaster recovery is needed. It documents the recovery actions to be taken for a critical application if a business disaster or interruption to that application occurs.

Related Elements

Main Description

This template provides an effective plan to ensure continuity or rapid restoration of a specific critical application following a business disaster or interruption to that application. This template will define for each application:

  • Application recovery points
  • Application backup details (type, frequency, time, storage)
  • Disaster Recovery Classification
  • Security Classifications
  • Support Libraries / Utilities and Tools
  • Special forms (if appropriate)
  • Special Hardware
  • Dependencies
