Template: Skill Tracker for Annuity Project
This MS Excel template captures the current skill level of project resources against the required skill or knowledge areas. It is intended to be used by projects which handle small work requests and have a wide range of skill set or application areas in which project resources need to be skilled.

Main Description

This template captures the current skill set of the project team; it is intended to be used for the service request allocation based on skill level of resource(s). For annuity projects which have wide ranging application specific skills and need to have a comprehensive compilation of current technical skills of project resources can do so by additionally using this separate skill tracking mechanism.
