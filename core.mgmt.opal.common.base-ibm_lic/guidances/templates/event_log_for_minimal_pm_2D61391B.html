Template: Event Log for Minimal PM
This MS Excel template provides a format for recording project information in a combined version of the template for events. This template is typically used by projects that require a minimum level of project management documentation, and want a single document to plan and track this detail.
Relationships
Related Elements

Main Description

Use this template to record information on the events within a project.  The template contains Instructions for information on using the Variables tab to populate fields on all of the tabs, and to add or remove tabs as needed.
