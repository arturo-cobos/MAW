Template: Resource Workload Tracker for Annuity Project
This MS Excel template documents the current workload of project resources for in scope applications and modules. For annuity projects which have small and very small work requests coming frequently can use this template for a detailed tracking of resource workload.
Relationships
Related Elements
  • Operational Schedule
Main Description

This template captures information such as:

  • Application and Module Name Resource responsible for a given application and/or module
  • Number of active service requests or tickets allocated to each resource
  • Number of active service requests or tickets allocated to each resource (for complexity levels - High, Medium and Low)
  • Total number of service requests allocated to each resource
