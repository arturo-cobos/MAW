Template: Program Model System Summary
This MS Word template is used to capture the planning details of projects or applications that follow a program model.
Relationships
Related Elements
  • Project Management System Summary
Main Description

This template contains sections for program management planning and individual project/application planning. For program model planning, following indicative sections are recommended. Other auxiliary plans are required to be added as appropriate.

  • Planning charter
  • Project definition
  • Agreements
  • High level work breakdown structure
  • Size and effort estimation
  • Milestone list
  • Financial plan
  • Risk management plan
  • Deliverable definition
  • Human resource plan
  • Organizational breakdown structure
  • Project management schedule
  • Operational schedule
  • Project defined process
  • Project quality plan
  • Communication management plan
  • Project dependency management
  • Project decision structure
  • Work product list
  • Technical environment plan
  • Product breakdown structure
  • Master test plan
  • Project procedure description
  • Reusability

The project/ application specific plan captures only the specific details as follows since majority of the details are available in program plan section. Auxiliary plans can include information such as:

  • Work breakdown structure
  • Risk management
  • Human resource plan
  • Operational schedule
  • Communication management plan
  • Work product list
  • Deliverable definition
