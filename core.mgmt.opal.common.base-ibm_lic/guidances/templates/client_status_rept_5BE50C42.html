Template: Client Status Report
This MS Word template is used to report project status to the client on a periodic basis.
Relationships
Related Elements
  • Client Status Report
Main Description

This template is typically used to capture following information

  • Project Schedule
  • Project status including milestones and accomplishments
  • Quality Status
  • Project Change Requests
  • Issues
  • Risks
  • Dependencies
  • Action items
  • Compliance incidents
  • Entry and Exit criteria for milestones
  • Planned activities for next reporting period
  • Attachments containing other relevant reports
