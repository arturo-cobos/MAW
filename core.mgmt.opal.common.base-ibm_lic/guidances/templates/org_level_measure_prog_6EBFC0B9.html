Template: Organization Level Measurement Program
This MS Word template is used to document the organization's measurement program to enable the organization to plan, perform, and manage the appropriate measurement activities.

Main Description
The scope of measurement program is explicitly mentioned in this template. Additionally, following sections are included in the template.

  • Program Objectives
  • Measurement program related to business goals
  • Criteria for selecting processes or sub-processes
  • Measurement program training
  • Rules for Measurement in Project Management (PM) structure projects
  • Measurement normalization practices
  • Metric attributes
  • Data gathering rules
  • Relating goals, objectives, processes, subprocesses, measures/metrics, associated statistical techniques and performance models
  • Performance models
  • Early warning notifications
  • Project Quality Analysis (PQA)
  • Statistical analysis by project categories and project attributes
  • Organizational Metrics Repository
  • Gathering project data
  • Validating and maintaining project data
  • Submitting project data to IBM
  • Requesting benchmarking data from IBM
  • Metrics roles and responsibilities
  • Reporting
