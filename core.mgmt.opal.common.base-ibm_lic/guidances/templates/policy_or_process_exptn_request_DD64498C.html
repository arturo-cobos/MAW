Template: Policy or Process Exception Request
This MS Word template is used to document the exceptions requested to the organization policies, standards and procedures.
Relationships
Related Elements
  • Policy or Process Exception Request
Main Description
This template is used to obtain approval for exceptions from relevant authorities. The requester fills up the project name and project phase in this template. The relevant OPAL references are made to support the exception. The details of the exception, justification for the request, alternate process, endorsement and reason for the decision are explained in the template. The names and signature of all approval authorities is also included in this template.
