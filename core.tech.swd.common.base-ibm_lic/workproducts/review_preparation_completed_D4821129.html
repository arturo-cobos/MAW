Main Description

The preparation for the review includes the following items:

  • Setup the actual meeting,
  • Updating the review template with the appropriate information,
  • Ensuring that the attendees are available when needed,
  • Ensuring that the material to be reviewed is completed and distributed to the reviewers prior to the actual review meeting
