Deliverable Properties
External Description

The User Support Material deliverable is a collection of user support media and materials that users may refer to during normal operations of the solution in production mode.  The purpose of this deliverable is to provide the customer with a master set of end user documentation and support media that will be distributed to the targeted user groups.  The deliverable / report may contain the following content for the targeted user group(s), as identified in the User Training and Support Specifications deliverable:

  • User Manuals
  • Reference Cards
  • On-line help
  • Support Web Site reference and content
  • Audio and Video media, if indicated

Note: A representative sample master copy(s) of the User Support Materials will be provided for the purpose of customer acceptance. Production of the actual number of User Support Materials required will be identified in the Deployment Plan, and is otherwise outside the scope of this agreement.
