External Description

The User Training Material is a collection of education material to be used for training users for operation of the solution.  The purpose of this deliverable is to provide the customer with a master set of education and training material, that will be delivered to the targeted user group(s) prior to using the solution in a production mode.  The deliverable / report may contain the following content for the targeted user group(s), as identified in the User Training and Support Specifications deliverable:

  • Education Synopsis
  • Classroom Materials
  • Instructor Qualifications, if appropriate
  • On-line Tutorial, if identified
  • Training Evaluation

Note: A representative sample master copy(s) of the User Training Materials will be provided for the purpose of customer acceptance. Production of the actual number of User Training Materials required will be identified in the Deployment Plan, and is otherwise outside the scope of this agreement.
