Work Product Descriptor (Artifact): Estimation Report
This artifacts documents the results of following the estimation process by recording the values of the estimate, along with the approach used in creating it, at each governance point. The estimate itself is created using the standard templates, tools, or models chosen by the project team.

Key Considerations
Best practices require the use of a minimum of two approaches when estimating. Each approach worksheet documents the results of a single approach (e.g. use of Function Points or use of a tool). The approach worksheets should be copied if more are needed.
