Activity: Implement project management system (IMSP391)
This activity expands upon the summary project management system produced while planning the project and defines a detailed management system as the basis for managing the project. This includes defining or expanding upon various plans and documenting project specific procedures and standards.
Extends: Implement project management system (IMSP391)
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
To ensure an appropriate project management system, including plans, procedures and standards, is in place to implement and support the project successfully.
Relationships
Parent Activities
  • Starting
Description

This activity includes the following:

  • Identify deliverable quality requirements, expanding the project quality plan
  • Expand the technical environment plan for PM tools
  • Define the project management system by tailoring it to build the project specific procedures and standards to be used to implement the project successfully
  • Define project reporting in the communications management plan
  • Set up billing and time recording mechanisms
  • Develop the project data management plan
