Main Description

The key output of this task is the Technical Readiness Assessment.

This task includes the following:

  • Complete the technical readiness assessment which is used as a self assessment to review and validate the technical solution, communicating readiness for completing the technical work products and deliverables for the project
  • Assess technical feasibility and requirements, such as obtaining approvals for using open source software
