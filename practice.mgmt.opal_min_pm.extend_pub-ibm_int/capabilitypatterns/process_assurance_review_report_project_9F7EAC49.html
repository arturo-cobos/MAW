Key Considerations
While reviewing projects that are grouped under Technology Assembly Centre for access control mechanisms (question 20), verify if there is any evidence of usage of shared desktops/ laptops. If yes, the login id used by each member is expected to be different. In cases where there is shared user id, a noncompliance is raised and GDIT (Global Delivery IT) team is informed immediately.
