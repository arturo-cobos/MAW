Work Product Descriptor (Artifact): Subcontractor Status Report
A report prepared by the subcontractor which addresses the status of the sub-project for which the subcontractor is responsible. The report is provided to the project manager. The format is identical to that of the project status report.

Main Description
The Subcontractor Status Report is a scheduled report that is created by the subcontractor and addresses the sub-project for which the subcontractor is responsible. The report provides information about the progress that the subcontractor has made during the reporting period.

Key Considerations

The Subcontractor Status Report is written by the subcontractor's project manager, according to format and frequency specified in the agreement and the subcontractor's project procedures description, and then reviewed by the responsible project manager from the delivery organization.

The accuracy and completeness each Subcontractor Status Report is validated at progress review meeting held between the project managers of the two parties shortly after the report is distributed.
