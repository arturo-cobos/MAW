Task Descriptor: Reconcile POUs work plans and confirm the project schedule (IMSP491)
This task will consolidate the project plans, milestones, dependencies and the schedule, across each potential organizational unit of the project.

Main Description

This task is based on the WWPMM work pattern P3: Integrate project organizational unit work plans.

Each potential project organizational unit (POU) - or performing organization - has received a Project Definition where an overall high level schedule is outlined. They have assessed the feasibility of their involvement, and designed their own plans for that. Also, conditional agreements have been signed with critical planned suppliers.

This task will add necessary effort to integrate work performed by the various POUs. It will complete the work plans by incorporating tasks for project management, acceptance of supplier deliverables, and integration of work products from various project organizational units.

Ensure the set of draft project plans developed during Task: Build project organizational unit work plans (IMSP491 and IMSP380) is adjusted and finalized from the bottom of the Organizational breakdown structure (OBS) up to the top POU.

This task confirms or adjusts the high-level overall schedule that was outlined in the Project Definition, ensures that appropriate slack exists for dependencies between POUs, and may adjust the POU's scope of work and milestones to ensure a proper synchronization between POUs, and eliminate possible bottlenecks that may affect shared resources.

The key outputs of this task are the Milestone List and the Project Management Schedule.

This task includes the following:

  • Reconcile project organization unit (POU) work plans and confirm the project schedule
    • Update the milestone list for dependencies
    • Review the project management schedule and adjust scope of work and milestones as needed
