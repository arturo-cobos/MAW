Main Description
The Milestone List is a definition of the key milestones contained in one Project Management Schedule together with a log of all the significant events related to those milestones, that occur during the course of the project.

Key Considerations

The Milestone List is extracted from the Agreement and the corresponding Project Management Schedule as part of the Starting activities of project execution. At a minimum, it is updated whenever a milestone is revised or is achieved.

It should contain:

  • All the key commitments made by the project manager to their manager (or, in the case of the top-level project manager, to the sponsor as specified in the sponsor agreement).
  • All the key dependencies on reporting project organizational units, including subcontractors.
  • Any other important events to which the project manager must pay particular attention.

The Milestone List is validated by ensuring that:

  • It is up to date and is consistent with the corresponding Project Management Schedule.
  • All the important commitments and dependencies of the project organizational unit are listed.
  • The history of each milestone is recorded accurately.
