Main Description
The Human Resource Plan shows the number of people required to staff the project, analyzed by human resource category and time. It also specifies the facilities and training required for each human resource category and the planned team-building activities.

Key Considerations

The Human Resource Plan is created during Planning and kept up to date as the project proceeds.

Early in the planning phase, this work product is no more than an estimate of the skills required and the number of people needed within each skill category. The numbers are derived from the initial Project Management Schedule and the project managerâ€™s experience. As planning proceeds, the Human resource plan becomes more solid and acts as a constraint on the Project Management Schedule.  During Starting, when the people are joining the project, it is expanded to include summary information about the actual project term.

Throughout the project, any plans for team-building activities can be added to the plan.

The Human Resource Plan may be revised if staffing requirements change because of any Change Orders that arise during the course of the project

On large projects, each project organizational unit may build its own Human Resource Plan. In that case, a summary Human resource plan is created for the entire project.

The project manager verifies that the Human Resource Plan is consistent with all the other project management plans, in particular the Project Management Schedule.
