Work Product Descriptor (Artifact): Project Decision Structure
One or more interrelated boards responsible for making specific decisions about the direction, management, and results of a project . The decision structure includes a specification of participants, authorities, responsibilities, and procedures.

Main Description

The Project Decision Structure documents the decision making processes which involve the project manager and other stakeholders such as the sponsor.

Those principles must be compliant with the project organization chart (part of the Organization Breakdown Structure (OBS)).

Internal control decision which are made on a daily basis by the project manager are not covered by this work product.

Key Considerations

The Project Decision Structure is built in the final stage of the Defining activities, once the project organization chart (part of the project OBS is known).

It can be updated during the project when changes affect the key roles. Another example of changes which occur to the decision structure is when more reactivity is required from one or the other decision team so that the frequency needs to be adapted.

The Project Decision Structure is approved by the delivery organization and by the sponsor.
