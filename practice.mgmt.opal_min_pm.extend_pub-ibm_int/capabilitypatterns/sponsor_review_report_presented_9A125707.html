Main Description
The Sponsor review report is usually presented periodically to an external sponsor. Its content and how it will be presented need to be specified in the Agreement. The Sponsor review report must not be confused with the Project Status Report, although it can be partially derived from the Project Status Report which is presented to the delivery organization’s management. Communication can take the form of a written communiqué or memo, group meeting, or individual meetings.
