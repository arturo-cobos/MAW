Task Descriptor: Compare estimates, adjust assignments, reconcile plans (IMSP421)
This task compares the estimating basis used to initially estimate the workload of the project or project phase, with the trends of the actual workload spent on work units that are completed or in progress. Adjustments are then made to assignments, estimates to complete the work, and project plans.

Main Description

The key outputs of this task are the Estimation Factors Report; Operational Schedule; Project Management Schedule; and the Event Log adjusted for staffing, changes or actions, as needed.  Outputs also include the Project Quality Plan, which may be part of the Project Management System Summary.

This task includes the following:

  • Compare productivity measurement with estimating basis
    • Analyze the effort data (actuals and total estimates to complete) for each activity type and compare it with the current estimating basis
    • Identify the variance for each type of activity, and update data for the chosen estimation factors for the reporting period
    • Assess if any other factors have affected the productivity before changing the estimating basis
  • Adjust assignments and estimate to complete
    • For situations in which there is a significant variance between actual productivity and the estimating basis, analyze the root cause
    • After root cause analysis, determine corrective actions to "get back on track" or to better prepare for the next period
    • Document the corrective actions
  • Reconcile project management plans
