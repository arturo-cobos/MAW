Task Descriptor: Track expenses and adjust financials (IMSP421)
This task compares project and business financial data, corrects discrepancies, and updates forecasts for future tracking periods.

Main Description

The key output of this task is the Financial Plan which may be part of the Project Management System Summary.  Additional outputs may include the Issue Log, Issue Documents, Action Log, Action Control Documents or the Event Log, as needed.

This task includes the following:

  • Compare project and business financial data
    • Compare the project and business system financial data
    • Reconcile the business system financial data with the financial plan and the expenditure log
    • Document the discrepancies so they can be tracked for resolution
  • Correct discrepancies and update forecast data
    • Determine corrections to be made in response to errors or variances, and document them
    • Determine the impact of resulting variances on the forecasts in the financial plan
    • Update the forecast data within the appropriate project documentation with the re-forecasted actuals for the remaining periods and the project at completion
