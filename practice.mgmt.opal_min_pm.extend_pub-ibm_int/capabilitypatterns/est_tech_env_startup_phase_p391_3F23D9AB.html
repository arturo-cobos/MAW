Main Description

This task is based on the WWPMM work pattern S1: Initiate phase.

This task is part of activity that is the first to be launched when starting the execution of a project (or of a new major technical phase or sub-project), and needs a first set of project team members (at least the project manager) who will plan the activities necessary to get the project up and running.

The key outputs for this task are a working technical environment for starting the project and the Technical Environment Plan, which is a section within the Project Management System Summary. 

While it is preferable to begin working in the technical environment which is intended to be used to manage the project, a temporary solution may have to be accepted for the initial startup.
