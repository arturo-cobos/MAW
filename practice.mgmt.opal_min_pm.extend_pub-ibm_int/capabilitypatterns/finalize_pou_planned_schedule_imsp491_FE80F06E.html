Task Descriptor: Finalize POU planned schedule (IMSP491)
This task will review and finalize the planned project organizational unit (POU) schedule.

Main Description

This task includes the following:

  • Review the effort, duration, and number of resources planned to perform the activities that constitute the critical path within the project organizational unit's Work Breakdown Structure (WBS), and adjust as necessary to optimize leads and lags.
  • Update the Project Management Schedule for the project organizational unit (POU), as needed, noting that it will be expanded when starting the project.

The key output of this task is the Project Management Schedule.  Input may include the Human Resource Plan, which may be part of the Project Management System Summary
