Task Descriptor: Start global delivery resource (IMSP391)
This task will welcome and start new global delivery team members on the project.

Main Description

This task is based on the WWPMM work pattern S6: Start staff.

This task is part of the activity that should anticipate the date when team members are going to join the project for producing its deliverables, and should finish when it is clear that these team members are up to speed. Thus this task may be planned several times in the course of project execution.

The focus in this task is on providing the new team members with the big picture of the project, the overview of the specific part of project execution they are going to be involved in, and on ensuring they are brought up to speed with the work they have to perform.

The key output of this task is the Operational Schedule.

This task includes the following:

  • Start the global delivery resources, consulting available guidelines
  • Update the Operational Schedule, as needed
