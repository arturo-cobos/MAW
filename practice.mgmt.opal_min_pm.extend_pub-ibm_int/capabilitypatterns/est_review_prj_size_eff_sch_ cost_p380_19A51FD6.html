Task Descriptor: Estimate and review project size, effort, schedule and cost (P380)
This task will estimate the total project size, effort, schedule, and cost, and conduct an independent review of the estimate.

Main Description

This task includes the following:

  • Estimate total project size, effort, schedule & cost
    • Select at least two estimating approaches to use to convert the baselined technical effort estimate into a total project effort estimate
    • Estimate any extra technical effort and all non-technical effort (such as information security control requirements, etc.), documenting the risks and any effort associated with mitigating these risks
    • Calculate the schedule associated with this effort
    • Update the total project effort after scheduling
    • Calculate the resource costs based on overall schedule and effort
    • Estimate the hardware, network and software costs
    • Provide an indication of the accuracy of the estimate, including at least the best, worst and nominal figures
    • Document the estimates
  • Conduct project estimate review
  • Conduct independent review (Technical and Delivery Assessment as per WW Quality Assurance Process, if appropriate)
    • Hold a review
    • Ensure the Independent Subject Matter Expert (Estimating) chooses an approval option (approved, approved with conditions, or not approved)
    • Document the results of the review
  • Baseline the estimate by placing the estimation report under version control

The key outputs of this task are the Estimation Report, Independent Review Report and Project Management Schedule.  Input may include the Financial Plan, which may be part of the Project Management System Summary.
