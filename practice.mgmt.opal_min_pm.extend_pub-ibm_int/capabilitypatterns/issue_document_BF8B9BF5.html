Work Product Descriptor (Artifact): Issue Document
A document describing, in detail, the issue and a complete history of its progression through analysis and resolution.
Based on Method Work Product: Issue Document
Main Description

An issue is a matter of concern on a project.

The Issue Document describes, in detail, the issue and provides a complete history of it as it moves through analysis and resolution.

Key Considerations

An issue is opened and analyzed.

If the analysis reveals that several actions or specific resolution processes need to be performed to resolve the issue, appropriate documents are opened, and a target resolution date is determined.

When all corresponding resolution actions, including those from specific resolution processes, have been performed, the issue is closed as resolved.

When needed, the issue is transferred to another level of project management. After it has been transferred, it might or might not require follow-up by the original issue owner.
