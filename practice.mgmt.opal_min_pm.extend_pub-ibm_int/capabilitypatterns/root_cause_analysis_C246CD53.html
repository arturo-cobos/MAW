Work Product Descriptor (Artifact): Root Cause Analysis Record
This work product is used to document the information relating to the identification and analysis relating to the incident selected for root cause analysis.

Main Description
The Root Cause Analysis Record contains the information relating to the identification and analysis of incidents selected for root cause analysis. The information contained in the Root Cause Analysis Record includes:
  • Project Information
  • Root Cause Analysis Information
  • Incident Information
  • Participants and Meeting Arrangements
  • Root Cause Analysis
  • Implementation and Tracking
