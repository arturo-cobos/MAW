Task Descriptor: Select supplier (IMSP711)
This task will select an appropriate supplier for a project.

Main Description

The key output of this task is the Supplier Selection Report.

This task includes the following:

  • Select supplier, working with the purchasing department to improve the match between the project and the supplier
    • Identify possible suppliers and define the criteria to be used in selecting one or a short list
    • Work with the purchasing department to proceed to the appropriate process for receiving supplier offers
    • Analyze and rate their proposals from the project point of view
    • Come to an agreement with the purchasing department on the offer to be selected and document in the Supplier Selection Report.
