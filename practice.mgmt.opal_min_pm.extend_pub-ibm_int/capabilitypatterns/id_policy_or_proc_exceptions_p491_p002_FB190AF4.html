Task Descriptor: Identify policy or process exceptions (IMSP491 and IMSP002)
This task will document the need for and obtain approval for, exceptions to organizational policies, standards, and procedures, for projects or organizational units with a valid business need to do so.

Main Description

This task is based on the WWPMM work pattern P4: Finalize sponsor agreement.

The key outputs of this task are the Policy or Process Exception Request, Project Defined Process and Project Procedures Description.

This task includes the following:

  • Identify the need for an exception to an organizational policy, standard, or procedure
  • Create the process exception request
  • Submit the process exception request and obtain all necessary endorsements and approvals
    • Review request
    • Determine and document impact and risk
    • Determine approval status
    • Update request, as needed
    • Return updated request and notify stakeholders
  • Update the project defined process in accordance with the exceptions
