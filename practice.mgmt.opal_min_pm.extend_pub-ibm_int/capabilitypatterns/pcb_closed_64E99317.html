Work Product Descriptor (Outcome): Project Control Book Closed
At the completion of this task, the Project Control Book will be closed and filed with the project deliverables in the appropriate contract storage area.
Based on Method Work Product: Project Control Book Closed
Relationships

Output From

Main Description
When the Project Control Book will be closed, the following items will be addressed:
  • The project financials will be reviewed for accuracy and discussed with the Geography and Offering Principals.
  • The project plan will be updated and closed out.
  • Any subcontractors will be released.
  • A review of all project issues and action items will be reviewed for closure.
  • The contract will be closed.
