Task Descriptor: Report project status (IMSP421)
This task reports to the sponsor and delivery organization management on project progress, financial and health status, and on the major events of which they should be made aware.

Main Description

The key outputs of this task are the Project Status Report and the Client Status Report.  Additional outputs may include the Issue Log, Issue Documents, Action Log, Action Control Documents or the Event Log, as needed.  Inputs include, but are not limited to, the Communication Management Plan, which may be part of the PMSS.
This task includes the following:

  • Prepare report components
    • Document the process, methods and tools utilized on the project in the Process, Methods and Tools (PM&T) Profile
  • Report to delivery organization management and escalate issues
  • Report to sponsor and agree on actions
