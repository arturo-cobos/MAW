Work Product Descriptor (Artifact): Change Log
The list of all changes, the owner, due date, and status.
Based on Method Work Product: Change Log
Main Description

A Change Log is a list containing some of the information about all or some of the Change Request.

Key Considerations

A Change Log is built by extracting some of the information from some or all of the Change Requests.  This can be done manually but is more easily achieved when the Change Requests are stored in a database. It is usual to produce several different Change Logs for different purposes.

See the Change Request work product description for information on how to validate this work product.
