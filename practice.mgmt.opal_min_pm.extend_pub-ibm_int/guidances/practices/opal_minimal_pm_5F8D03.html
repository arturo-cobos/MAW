Practice: Minimal Project Management (OPAL/WWPMM)
To be used on small to medium projects. This practice uses some of the WWPMM Full Project Management Model, and Standard Project Management Model elements (tasks, WP, Roles, etc). It also uses Core OPAL Work Products, Roles, and Guidance.
