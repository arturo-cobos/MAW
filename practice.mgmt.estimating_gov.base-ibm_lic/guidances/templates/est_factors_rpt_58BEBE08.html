Main Description

The Project Details worksheet documents basic project information and contains the following:

  • An overview of the Estimation Factors Report work product
  • Instructions on how to complete the report
  • Basic project information, such as the project name, client name, and project dates, category, business sector, and industry

 The Committed and Committed Charts worksheets record and chart the actual progress and productivity for each estimation factor. It contains the following:

  • The current baseline and actual counts per estimation factor for the project as of the checkpoint date
  • The assumed and actual effort per estimation factor as of the checkpoint date
  • Graphs of the productivity of each estimation factor as measured over time at each checkpoint review

The Closed worksheet records the actual values achieved during the project and calculates the final effort per estimation factor for improving future estimates. It contains the following:

  • The final actual count and effort expended for each estimation factor as at project close

Once you have decided on how the various team leads will contribute their content, complete the Project Details worksheet.

At the start of the project (Checkpoint 0), enter the baseline count for every estimation factor into the Committed worksheet. Complete the Checkpoint 0 data to record the initial productivity assumed in the project.

Just prior to each estimation checkpoint review, update the Committed worksheet with the actual counts completed, in progress, and not yet started for every estimation factor, and the effort expended thus far on the complete and in progress estimation factors. Estimate the degree of completion of any work in progress. The report will calculate and chart the actual progress and productivity experienced on the project.

If the project must be re-baselined due to change requests, changed assumptions or any other reason, the Estimation Factors Report must be updated to reflect the change in the baseline at that point in time. The original baseline must not be modified but the current baseline counts and effort for each estimation factor must be updated on the Committed worksheet to reflect the date at which the baseline changed, and the impact that change has on the project. An example of a change in baseline is included in the Committed Example worksheet.

At project close, enter the actual count and effort expended for every estimation factor into the Closed worksheet.
