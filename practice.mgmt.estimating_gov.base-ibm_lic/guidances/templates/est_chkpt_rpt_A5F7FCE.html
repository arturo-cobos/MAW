Main Description

The Estimation Checkpoint Report contains three worksheets – an Instructions worksheet to capture the Project Details, a Committed worksheet for recording progress at each checkpoint, and a Charts worksheet graphing the results of measuring the variance of the project.

The Instructions worksheet contains the following project details:

  • An overview of the Estimation Checkpoint Report work product
  • Instructions on how to complete the report
  • Basic project details such as the project name, customer, and categorization of the project

The Committed worksheet contains the following:

  • The baseline estimate of effort, cost and schedule as at each checkpoint review
  • The actual used effort, cost and schedule as at each checkpoint review
  • The estimate to complete of effort, cost and schedule as at each checkpoint review 
  • A measure of the variance of the estimate against the baseline as at each checkpoint review

The Charts worksheet contains the following:

  • Graphs of the variance in effort, cost, and schedule as measured over time at each checkpoint review
  • Graphs comparing the current estimate at completion to the baseline estimate as measured over time

Complete the Instructions worksheet.

At the start of the project (Month 0), enter the baseline estimate of effort, cost and schedule into the Committed worksheet. The baseline estimate can be found in the Estimation Report from the original project estimate. Complete the Month 0 data to record the initial variance inherent in the project.

At each checkpoint, re-estimate the project and update the Committed worksheet with the actual used effort, cost, and schedule and the new estimate to complete data for the appropriate checkpoint.

If the project must be re-baselined due to change requests, changed assumptions or any other reason, the Estimation Checkpoint Report must be updated to reflect the change in the baseline at that point in time. The original baseline must not be modified but the current baseline estimate of effort, cost and schedule must be updated on the Committed worksheet to reflect the date at which the baseline changed, and the impact that change has on the project. An example of a change in baseline is included in the Committed Example worksheet.
