Guideline: Monitoring Project Progress through Estimation Factors
This Guideline outlines some simple methods to help you, as a Team Lead, verify that the scope and assumptions on which the baseline estimate was based on are holding true as the project progresses. Simple tools created by the IBM Estimating Community make this easy.

Main Description

The intended audience is any IBM practitioner who is responsible for creating initial estimates for projects, and who needs to monitor the progress of a project to a baselined estimate.

This technique paper addresses the following topics:

  • Issues to Monitor during project execution
  • Was the scope properly understood? Was the sizing done correctly?
  • Is the team delivering as expected?
  • Re-baselining
  • Project close
