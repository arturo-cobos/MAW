Task: Expand the technical environment plan for PM tools (IMSP391)
This task will expand the technical environment plan to identify the project management tools supporting the project management system.
Purpose

To expand the technical environment plan that will identify the project management (PM) tools on the project.


Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Operational Schedule
  • Technical Environment Plan
    		Optional:
    • None
    Outputs
    • Technical Environment Plan
      Main Description

      Expand the technical environment plan to identify the PM tools supporting the project management system. Examples are tools which support PM:

      • Scheduling  planning  and work progress tracking against baseline plans and estimates.
      • WBS planning and tracking.
      • Project Control Book management, exception management.
      • Risk assessment and management.
      • Financial tracking including time recording, cost recording  and comparisons of actual against budget.
      • Project status reporting.
      • Project metrics reporting environment to support the quantitative goals established in the project quality plan.
