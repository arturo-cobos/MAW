Task: Prepare for review (IMSP421)
Disciplines: Quality Management
Purpose
To prepare for a project review by ensuring the type of review to be conducted is known, making sure that key project documents are easily available, and collaborating on the agenda for the review.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Process Assurance Review Report - Project
    Outputs
    • Quality Review Documentation
      Main Description
      Preparation for reviews takes place in response to a planned activity and/or the request of the review team. The role of the project management team is to help prepare for the review:
      • understand the type of review,
      • collaborate on the development of the audit or review agenda,
      • provide adequate documentation to reviewers,
      • ensure correction of known deficiencies in the project quality system and other aspects of the project.
