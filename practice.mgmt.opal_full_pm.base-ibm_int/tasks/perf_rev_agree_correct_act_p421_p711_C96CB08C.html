Task: Perform review and agree on corrective actions (IMSP421 and IMSP711)
Disciplines: Supplier Management
Purpose
To perform a subcontractor audit or review, discuss findings, and agree on corrective actions.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Agreement
    		Optional:
    • Project Management System Summary
    • Project Procedures Description
    • Project Quality Plan
      Outputs
      • Action Control Document
      • Action Log
      • Quality Review Documentation
        Main Description
        Review the project situation, discuss review findings with subcontractor personnel, negotiate corrective actions with the subcontractor and then finalize the audit or review report.

        Determine the impact on the overall project and launch corrective actions as agreed with the subcontractor.

        Raise a Change Request if necessary (when actions impact the subcontractor Agreement or elements in the project baseline).

        Update the overall Project Quality Plan and Project Procedures Description, the Operational and project schedules and other work plans according to the impact of the audit or review results.
        Key Considerations

        Reference OPAL Procedures:

        • Project Monitoring and Control Procedure (IMSP421)
        • Supplier Management Procedure (IMSP711)
