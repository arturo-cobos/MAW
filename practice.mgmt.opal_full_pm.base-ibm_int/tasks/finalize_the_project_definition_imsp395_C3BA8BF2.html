Task: Finalize the project definition (IMSP395)
This task will review all associated project documentation ensuring that the Project Definition is consistent.
Purpose
To ensure the Project Definition is complete and consistent.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Definition
  • Work Breakdown Structure (WBS)
    		Optional:
    • High-Level Solution Plans
      Outputs
      • Project Definition
        Main Description
        Review all associated project documentation ensuring that the Project Definition is consistent.

         Refer to the Work Product Inspection Procedure (IMSP530) for information about conducting reviews.

        Store the Project Definition with all related documentation.

        Place the Project Definition under version control.

        Obtain authorization to proceed to Planning from the delivery organization and sponsor or close the proposal process.
