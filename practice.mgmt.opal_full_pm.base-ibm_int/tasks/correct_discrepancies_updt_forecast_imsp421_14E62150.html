Task: Correct discrepancies and update forecast data (IMSP421)
Based on a comparison of project and business system financial data, this task corrects discrepancies and updates forecasts for future tracking periods.
Purpose
To correct financial data discrepancies and assess forecasts for future tracking periods.

      Main Description

      Reference IGSDF - Using GBS CRM on Engagements for links to your local GBS CRM Playbook for specific guidance on CRM Step 8. Execute Project for tools for business unit financial and reporting schedule, requirements and tools.

      The key outputs of this task are the Financial Plan (which may be part of the Project Management System Summary); the Issue Document and/or Issue Log, as needed; and the Action Control Document and/or Action Log, as needed.
      This task includes the following:

      • Determine corrections to be made in response to errors or variances, and document them
      • Determine the impact of resulting variances on the forecasts in the financial plan
      • Update the forecast data within the appropriate project documentation with the re-forecasted actuals for the remaining periods and the project at completion
