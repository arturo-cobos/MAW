Task: Reconcile project management plans (IMSP421)
This task compares the estimating basis used to initially estimate the workload with the trends of the actual workload spent on work completed or in progress. Adjustments are then made to assignments and estimate to complete the work and updates project management plans accordingly.
Purpose
To revise the estimates to complete the work and to take appropriate corrective actions, as needed.

      Main Description

      The key outputs of this task are the Project Management Schedule; Operational Schedule; Project Quality Plan (which may be part of the Project Management System Summary); Project Procedures Description (which may be part of the Project Management System Summary); Change Request and Change Log.
