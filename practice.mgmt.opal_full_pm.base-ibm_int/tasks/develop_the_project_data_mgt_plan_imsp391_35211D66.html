Task: Develop the project data management plan (IMSP391)
This task will develop the project data management plan.
Purpose
To develop the project data management plan.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Management System Summary
    		Optional:
    • None
    Outputs
    • Project Data Management Plan
    • Project Management Schedule
    • Project Management System Summary
      Main Description
      Develop the project data management plan:
      • Identify the data required to support the project in all planned areas 

      Data are the various forms of documentation required (reports, manuals, specifications, files, charts) and data medium (such as printed or electronic) to support the needs of the project team in administration, engineering, configuration management, financial, quality and procurement.

      • Identify the reason for obtaining identified data and the security level associated with the data.
      • Record all appropriate information on identified data in the project data management plan.
      • Plan for data management.  Ensure that the project data management plan is clearly defined, like collection, recording frequency.
      • Summarize this plan in the project management system summary 
      • Add the corresponding data management monitoring activities to the project management schedule.
