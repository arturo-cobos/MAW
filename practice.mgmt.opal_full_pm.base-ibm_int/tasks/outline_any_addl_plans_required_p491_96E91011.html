Task: Outline any additional plans required (IMSP491)
This task will create any additional plans require for the project that have not been created for the project.
Purpose

The purpose of this task is to create additional plans necessary for the project. 

Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Definition
    		Optional:
    • None
    Outputs
    • Client Data Security and Privacy Plan
    • Disaster Recovery Plan
    • Project Management System Summary
    • Release Plan
      Main Description

      Outline any additional plans required, such as:

      • Facilities plan
      • Relocation plan 
      • Conversion plan
      • Migration plan 
      • Installation and delivery plan
      • Measurement plan
      • release plan
      • client data security and privacy plan 
      • application recovery plan
      • backup register
      • disaster recovery plan
      • storage and media control register

      Note:  Where specific plans do not exist (such as Facilities plan) document the information in the Project Management System Summary.

      The client data security and privacy plan is required if one or both of the following statements are true:

      • The client has security and privacy requirements
      • The project is dealing with Personal Information or Sensitive Personal Information

      Note: Personal Information is any information that identifies or can reasonably be used to identify, contact, or locate the individual to whom such information pertains.
      Sensitive Personal Information is any information that could be misused to significantly harm an individual in a financial, employment or social way.

      For further information on Personal Information and Sensitive Personal Information as they apply in IBM, refer to the "Guidance on 'Personal Information' and 'Sensitive Personal Information'" available on the "Data Security & Privacy" intranet site.

       Refer to the Release Management Guideline for additional information on release planning.
