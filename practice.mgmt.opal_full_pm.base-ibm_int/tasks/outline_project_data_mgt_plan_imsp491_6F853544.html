Task: Outline the project data management plan (IMSP491)
This task will outline the project data management plan and record all appropriate information on identified data in the project data management plan.
Purpose
To outline the project data management plan and record all appropriate information.

RolesPrimary Performer:

      Main Description

      The identification of data elements required for the project is documented in the Project Data Management Plan. It contains the content and form. The data elements are to identify the purpose for this data and the distribution plans of these data elements to fulfill the needs of the project.
       
      Here are a few examples of data required by a project:

      • Customer-provided data regarding the applications involved
      • Project Plan
      • Reports
      • Specifications
      • Files
      • Correspondence
      • Internal Design review documents
      • Meeting minutes
      • Action Items
      • Lessons Learned
