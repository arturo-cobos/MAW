Task: Assess project or program performance
Disciplines: Sponsor Agreement Management
Purpose
To assess project performance, identify gaps between objectives and accomplishments, and examine alternatives to reduce any gaps that are identified.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Status Report
    		Optional:
    • Project Definition
      Outputs
      • Project Definition
        Main Description
        Analyze project performance:
        • Identify gaps between accomplishments and objectives, based on the Project Status Reports and all available measurements relative to achieving the goals or mission and customer satisfaction.
        • Depending on the nature of those gaps, examine alternatives to reduce or reinforce the objectives towards achieving the goals or mission.
