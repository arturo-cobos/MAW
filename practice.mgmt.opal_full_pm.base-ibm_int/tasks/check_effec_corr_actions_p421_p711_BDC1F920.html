Task: Check effectiveness of corrective actions (IMSP421 and IMSP711)
Disciplines: Supplier Management
Purpose
To ensure the effectiveness of subcontractor corrective actions.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Action Control Document
  • Action Log
  • Change Log
  • Change Order
  • Change Request
  • Corrective actions
    		Optional:
    • None
    Outputs
    • Action Control Document
    • Action Log
      Main Description
      After the subcontractor has implemented the corrective actions, the project management verifies that the expected results have been achieved.
