Task: Perform initial assessment, reject or accept for analysis (IMSP610)
Disciplines: Change Management
Purpose
To review a new Change Request, make an initial assessment of its impact, and determine how to handle the request.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Change Log
  • Change Request
    		Optional:
    • None
    Outputs
    • Change Log
      Main Description
      Review the Change Request and, if necessary, discuss it with its originator to clarify what is proposed and the rationale behind the proposal. Perform a quick assessment of the potential impacts and determine whether it should be rejected, deferred, accepted as a small change, or accepted for further analysis. When accepted for further analysis, agree upon the timetable and resources for performing the detailed impact analysis and, if required, decide how the costs thereof should be distributed. Document the Change Request accordingly.
