Task: Participate in review (IMSP421)
Disciplines: Quality Management
Purpose
To participate in a project review and evaluate the findings of the review team.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
  • Project Quality Analyst
  • Project Team member
InputsMandatory:
  • Project Procedures Description
    		Optional:
    • Quality Review Documentation
      Outputs
      • Corrective actions
        Main Description
        Attend the review workshop and ask subcontractors to participate, as required. During the review, brief the review team as needed. Evaluate any findings and concerns, assessing possible actions. Review preliminary findings with the review team before the review team finalizes its review report.

        Reference OPAL Procedures:

        • Project Monitoring and Control Procedure (IMSP421)
        • Process Assurance Review Procedure (IMSP232)
