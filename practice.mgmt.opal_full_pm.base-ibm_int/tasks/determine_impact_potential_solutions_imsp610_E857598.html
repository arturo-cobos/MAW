Task: Determine impact and potential solutions (IMSP610)
This task will determine the scope of the requested change by selecting the first lifecycle phase to the impacted.
Purpose

To determine the scope, impact, and solutions for the change request.

Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Change Log
  • Change Request
    		Optional:
    • Operational Schedule
    • Project Management Schedule
      Outputs
      • Change Log
      • Change Request
        Main Description

        Determine the scope of the requested change by selecting the first lifecycle phase to the impacted.  Review any active issues, risks and actions, identifying any relationships with the requested change.   Update the appropriate change documentation and identify relationships to current issues, risks, or actions. 

        Identify participates in the assessment of the impact to project and gather the assessment results for all solution options.  Finally, update the change documentation with each recommended approach to implement and any associated estimates.
        Steps
        Determine scope of change request
        Determine scope of change request by selecting the first lifecycle phase to be affected.

        Examples of lifecycle project phases, include:

        • Solution Definition
        • External Design
        • Internal Design
        • Solution Generation
        • Solution Validation
        • Solution Deployment
        • Training
        Review active issues, risks and actions
        Review active issues, risks and actions, identifying any relationships with the requested change.
        Update the change management documentation
        Update the change request, with the first lifecycle phase affected and any identified relationships with current issues, risks, or actions.
        Identify the appropriate individuals to participate in the assessment
        Identify the appropriate individuals to participate in the assessment of the impact to the project from the request. Also update the status of the request to 'In Process'.
        Distribute the change management documentation
        Distribute the change request and instructions for conducting the assessment of the impact to the project from the request to the identified participants.

        The instructions for conducting the assessment of the impact to the project from the request include, but are not limited to, the following:

        • Identifying information (Requested Change, Affected Project)
        • Timeframe for completing assessment
        • Procedure for submitting assessment results

        The Project Manager and the Client need to agree on the disposition of the change.

        Gather and consolidate the assessment results for all solution options
        Gather and consolidate the assessment results for all solution options.
        • If multiple approaches to implementation are considered, perform the following for each viable option:
          • Estimate the effort to implement the change.
          • Estimate the cost or other financial impact of implementing the change.

        Refer to the Estimating Procedure (IMSP380) for information about estimating the size, effort and costs of requests.

        Update the change management documentation
        Update the change request with each recommended approach to implement and any associated estimates.
