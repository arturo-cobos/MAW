Task: Update the human resource plan (IMSP491)
This task will update the human resource plan.
Purpose

The purpose of this task is to build the Human resource plan that will provide the project manager a tool to help in staff planning, estimating staff costs, coordinating staff acquisition and controlling staff levels, staff related needs and team-building activities over the life of the project.

      Main Description

      This task will document the number of people required for the project by human resource category. It will indicate the required experience and skill levels, timeframe, office and materiel requirements, and an education plan for each category for the project.

      The project manager uses this information to build up or restructuring the resource pool for the project, especially during the startup period of a phase.

      Update the human resource plan to identify resource and training needs for the project.

      Ensure that the training needs for project-defined processes, methodology, work products and tools are identified for each role on the project.
