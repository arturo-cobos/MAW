Task: Review effectiveness of actions (IMSP605)
Disciplines: Event Management
Purpose
To review the results of the critical actions and ensure they have been effective.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Action Control Document
  • Action Log
  • Operational Schedule
    		Optional:
    • None
    Outputs
    • Action Control Document
    • Action Log
      Main Description
      For those critical actions identified as needing review, review the effectiveness of those actions. If the results are consistent with the objectives, notify the originator that the action has been completed and its effectiveness reviewed, and then close the action. If the results of the action are not as expected, either re-plan an effectiveness review or modify the action to improve its effectiveness.
