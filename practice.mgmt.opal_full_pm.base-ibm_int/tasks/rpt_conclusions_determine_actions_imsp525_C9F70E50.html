Task: Report conclusions and determine actions (IMSP525)
Agree on the corrective actions with the originator and correct the situation.
Purpose
To discuss the situation with the originator and agree and report on the corrective actions.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Meeting Documentation
  • Situation Analysis Documentation
    		Optional:
    • None
    Outputs
    • Situation Analysis Documentation
      Main Description

      Discuss the situation analysis with the originator and agree on the corrective actions to address the incident.  Assemble all findings (if not completed) and update the appropriate documentation.   Notify all individuals who may have an assignment for the corrective actions and distribute the situation analysis documentation to all interested parties.

      Steps
      Review the root cause analysis

      Discusses the situation (root cause) analysis conclusions with the Originator.

      Obtain agreement

      Obtains agreement on the approved corrective actions to address the incident.

      Assemble findings

      Assembles findings by reviewing meeting documentation and completing the situation analysis documentation.

      Notify individuals

      Notifies individuals not present at the situation (root cause) analysis meeting of any assignments and their urgency.

      Distribute the approved root cause analysis documentation
      Distributes the approved situation analysis documentation to all appropriate individuals, such as the Originator of the situation (root cause) analysis, senior management, and the organization's Measurements & Reporting Analyst.
