Task: Determine the work classification (IMSP491 and IMSP454)
This task will determine the work classification (such as large project or small work effort) and the level of project management to be applied.
Purpose
The purpose of this task is to determine the work classification.

      Main Description

      This task will classify an approved service request or project. 

      The classification is used to:

      • Understand the project's classification for later use when tailoring the project
      • Determine the type of request and the type of work
      • Classify the service request as a work effort or project, based on estimated effort and risk
      • Determine the level of project management to apply

      Determine the work classification (such as large project or small work effort) and the level of project management to be applied in accordance with the Work Classification Procedure (IMSP454).
