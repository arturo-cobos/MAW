Task: Evaluate options and discuss project objectives (IMSP395)
Disciplines: Project Definition
Purpose
To evaluate the sponsor’s expectations and ensure the project scope and objectives are suitable for the delivery organization.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
  • Opportunity Owner
  • Relationship Owner
  • Technical Solution Architect
InputsMandatory:
  • Project Charter
    		Optional:
    • Service Request
      Outputs
      • Project background analyzed
      • Project objectives analyzed
        Main Description
        Evaluate potential mismatches and severe risks in the sponsor’s expectations. Confirm the sponsor’s charter if the expected project seems safe, or imagine possible project scope options and project shaping directions that suit the delivery organization capabilities. Document accordingly the set of objectives that the project being considered should achieve.
