Task: Reconcile work plans and confirm overall project plans (next phase)
Disciplines: Work Plan Management
Purpose
To integrate individual POU schedules into an overall schedule, review dependencies, ensure proper slack, identify milestones, and verify the overall critical path.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Milestone List
  • Project Definition
  • Project Management Schedule
    		Optional:
    • None
    Outputs
    • Milestone List
    • Project Definition
    • Project Management Schedule
      Main Description
      Reconcile the high level overall schedule that was outlined in the Project Definition, with the individual plans that are produced by each future project organizational unit (POU)-or performing organization-.

      Pay special attention to shared resources and ensure that appropriate slack exists regarding dependencies between POUs.

      Ensure the Project Management Schedule covers all Organizational Breakdown Structure (OBS) levels.

      Extract the key milestones from the integrated project schedule into the Milestone List, and verify the project critical paths again.
