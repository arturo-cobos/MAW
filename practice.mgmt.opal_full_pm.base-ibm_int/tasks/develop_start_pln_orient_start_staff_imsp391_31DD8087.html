Task: Develop the startup plan (IMSP391)
This task will establish a plan to start the project.
Purpose
To ensure that the project startup work is planned appropriately.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Deliverable Definition
  • Financial Plan
  • Human Resource Plan
  • Outline of the Communications Management Plan
  • Outline of the Project Management System Summary
  • Outline of the Project Quality Plan
  • Outline of the Technical Environment Plan
  • Project Definition
  • Project Management Schedule
  • Risk Management Plan
    		Optional:
    • None
    Outputs
    • Operational Schedule
      Main Description

      This task is based on the WWPMM work pattern S1: Initiate phase.

      This task is part of activity that is the first to be launched when starting the execution of a project (or of a new major technical phase or sub-project), and needs a first set of project team members (at least the project manager) who will plan the activities necessary to get the project up and running.

      The key output of this task is the Operational Schedule for the Starting phase.

      This task includes the following:

      • Develop the startup plan, identifying all plans and procedures to be created during startup
      • Record the startup plan in the operational schedule
      • Bring the startup staff on-board
      • Communicate the startup plan
