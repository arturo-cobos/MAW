Task: Formally document and plan dispositions (IMSP540)
Disciplines: Deliverables Management
Purpose
To document the agreements that has been established with the sponsor to resolve findings. If necessary, prepare revisions to the sponsor and/or supplier Agreements.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Action Control Document
  • Change Request
    Outputs
    • Action Control Document
      Main Description
      Precisely document the dispositions (financial responsibility, milestones, schedules...) that have been agreed to with the sponsor to resolve the contentious findings and have this documentation formally approved. If necessary, prepare revisions to sponsor and supplier Agreements and obtain approval.

      Prioritize the steps to implement the agreed to solution, update the project schedule accordingly, and ensure that all project plans and procedures remain consistent.
