Task: Perform initial analysis, accept or escalate (IMSP620)
This task outlines the analysis to be performed on the identified issue.
Disciplines: Event Management
Purpose
To identify and record new issues, analyze them, determine how best to resolve them.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Issue Management Plan
    		Optional:
    • Issue Document
    • Issue Log
      Outputs
      • Issue Document
      • Issue Log
      • Issue Management Plan
      • Risk Management Plan
        Main Description

        Identify issues (from mail, discussions, project management activities, Event Log, other reviews) that require attention due to their possible impact on the project.

        Analyze, prioritize and rank the issue based on the criteria established in the Issue Management plan. 
        Note: Update the ranking criteria in Issue management plan, as appropriate based on project learning

        Perform immediate resolution if possible or pass to appropriate level of project management or determine an optimal time frame for resolution.

        Record the issue in Issue Log.

        Launch a situation analysis if necessary.
