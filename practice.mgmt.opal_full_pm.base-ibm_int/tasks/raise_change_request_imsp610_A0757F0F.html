Task: Raise change request (IMSP610)
Recognizes the need for a change, gathering the reason and nature of the change and notify the Project Manager.
Purpose
To recognize the need and raise the change request.
Relationships
RolesPrimary Performer:
  • Stakeholder
Additional Performers:
  • Project Manager
InputsMandatory:
  • Need for change to the project
    		Optional:
    • None
    Outputs
    • Change Log
    • Change Request
      Main Description
      Recognize the need for a change.  Document the reasons the change is needed.  Notify the Project Manager of the change while providing the reasons and nature of the change.  The change request, log and order are started.
      Steps
      Recognize the need for a change to a project
      Recognize the need for a change to a project.
      Notify the appropriate Project Manager of the need for a change to their project
      Notify the appropriate Project Manager of the need for a change to their project.
      Provide the Project Manager with change details
      Provide the Project Manager with all available details regarding the need for a change to the project, including the reason for and the nature of the change request.
