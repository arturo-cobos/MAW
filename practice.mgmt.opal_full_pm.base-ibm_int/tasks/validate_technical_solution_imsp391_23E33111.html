Task: Validate technical solution (IMSP391)
This task will perform a technical readiness assessment to review and validate the technical solution.
Purpose

To determine the readiness to complete the technical work products and deliverables for the project.

Relationships
RolesPrimary Performer:
  • Technical Solution Architect
Additional Performers:
  • Project Manager
Outputs
  • Task Completed
    Main Description

    The key output of this task is the Technical Readiness Assessment.

    This task includes the following:

    • Complete the technical readiness assessment which is used as a self assessment to review and validate the technical solution, communicating readiness for completing the technical work products and deliverables for the project
    • Assess technical feasibility and requirements, such as obtaining approvals for using open source software
    Key Considerations

    Reference OPAL Procedures:

    • Project Start (Project Launch) Procedure (IMSP391)

    OSSC Approval Process: If you are working on a proposal where IBM employees or subcontractors will be using 3rd party commercial software, Linux OS or open source software, you should follow the Open Source Steering Committee (OSSC) Approval Process prior to the contract being signed.

    Additional info can be found at the Open Source Software core team wiki
