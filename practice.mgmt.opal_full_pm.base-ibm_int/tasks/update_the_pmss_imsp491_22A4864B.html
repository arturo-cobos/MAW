Task: Update the project management system summary (IMSP491)
This task will update Service level agreement management plan section of the project management system summary.
Purpose

The purpose of this task is to update Service level agreement management plan section of the project management system summary, when applicable.

Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Definition
    		Optional:
    • None
    Outputs
    • Deliverable Definition
    • Project Management System Summary
      Main Description

      Where applicable, update Service level agreement management plan section of the project management system summary.

      If the agreement with the sponsor does not explicitly define the SLAs, measurements, reporting and so on, then have the sponsor review and approve the Service level agreement management plan section in the project management system summary.

      The Service level agreement management plan includes:

      • A definition of the SLAs and targets that are part of the agreement
      • A definition of the measurements that will be recorded and used to determine actual performance of the project in relation the SLAs
      • A description of how the measurements will be recorded, stored, analyzed and reported (including reporting frequency - SLAs must be reported at least quarterly).

      If the agreement with the sponsor does not explicitly define the SLAs, measurements, reporting and so on, then have the sponsor review and approve the service level management plan section of the project management system summary.
