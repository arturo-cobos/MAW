Task: Monitor progress of issue resolution (IMSP620)
Disciplines: Event Management
Purpose
To ensure that issues are being resolved in a timely manner.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Issue Document
  • Issue Log
    		Optional:
    • None
    Outputs
    • Issue Document
    • Issue Log
      Main Description
      Assess the status of the issues that are in the process of being resolved and resume activities towards resolution if necessary. Identify any pending or new concern with the issues being resolved, and update priority and target dates as necessary.

      Close the issues for which resolution is complete.
