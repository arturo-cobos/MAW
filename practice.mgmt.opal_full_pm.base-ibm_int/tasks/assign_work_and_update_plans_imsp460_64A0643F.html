Task: Assign work and update plans (IMSP460)
Disciplines: Work Plan Management
Purpose
To identify and assign resources to perform newly requested in scope work.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • In-Scope Request
  • Operational Schedule
  • Work Breakdown Structure (WBS)
    Outputs
    • Operational Schedule
    • Work Breakdown Structure (WBS)
      Main Description
      Identify the resources needed to fulfill the request.

      Request new resources as necessary within the scope of the sponsor Agreement.

      Expand as appropriate the Work Breakdown Structure (WBS) and the Operational Schedule.

      Update the Deliverable Definitions to reflect the new request.

      Assign work packages.
