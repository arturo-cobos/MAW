Task: Conduct meeting and distribute meeting notes (IMSP421)
Disciplines: Communications Management
Purpose
To conduct internal project meeting and document the outcome.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Meeting Documentation
  • Project Status Report
    		Optional:
    • None
    Outputs
    • Action Control Document
    • Action Log
    • Estimation Checkpoint Review Report
    • Meeting Documentation
    • Project Status Report
      Main Description
      Conduct the meeting according to the agenda.

      Disseminate information about the project and its environment.

      Provide the opportunity for team members (or the project management team of the subcontractor) to contribute their own perspectives and information about the project.

      Collect new issues and decide actions to resolve issues.

      Document any issues or actions that arise from the meeting using Issues Documents and Action Control Documents as appropriate.

      Produce and distribute meeting notes.
