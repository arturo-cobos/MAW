Task: Ensure supplier assimilates into project (IMSP391 and IMSP711)
This activity will ensure the supplier assimilates into the project successfully, by reviewing their work and status.
Purpose
To verify that the work on the supplier's sub-project has gotten off to a good start.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Subcontractor Status Report
    		Optional:
    • None
    Outputs
    • Corrective actions
    • Quality Review Documentation
      Main Description

      The key output of this task is the Quality Review Documentation.

      This task includes the following:

      • Ensure supplier assimilates into project successfully, including
      • Review the progress and quality of the supplier work
      • Examine early subcontract status reports to ensure they are getting up to speed
      • Perform a formal review of the way the supplier tracks progress, risks and quality (if the agreement provides for it)
