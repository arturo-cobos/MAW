Task: Document the additional project work environment details (IMSP491)
This task will document additional project work environment details. This may include the buildings, facilities, tools, and equipment that people need to perform their jobs in the project.
Purpose
The purpose of this task is to document additional project work environment details.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Definition
    		Optional:
    • None
    Outputs
    • Technical Environment Plan
      Main Description
      Document the additional project work environment details, if any, using the technical environment plan. This can include the facilities, tools, and equipment that people need to perform their jobs in the project, and has not already been documented as part of any other project plans such as the human resource plan.

       Refer to the Organizational Work Environment section of the organizational reference document for the work environment standards established in the organization and for client mandated standards, if any.
