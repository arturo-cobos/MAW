Task: Create high-level product breakdown structure (IMSP395)
Disciplines: Project Definition
Purpose
To create a high-level Product Breakdown Structure (PBS) of the target solution.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project background analyzed
  • Project objectives analyzed
    		Optional:
    • High-Level Solution Plans
      Outputs
      • High-level PBS analyzed
      • Project Definition
        Main Description
        Decompose the target solution into high level components or sub systems, and identify any offerings or other licensable assets to be used as part of the solution. Document the solution components in the High-level Product breakdown structure (PBS) section of the Project definition.
