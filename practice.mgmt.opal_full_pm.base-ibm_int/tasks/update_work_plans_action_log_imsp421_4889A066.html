Task: Update work plans and Action Log (IMSP421)
Disciplines: Quality Management
Purpose
To update project plans and the Action Log as appropriate to ensure the project avoids a recurrence of any deficiencies.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Action Control Document
  • Action Log
    		Optional:
    • Quality Review Documentation
      Outputs
      • Action Control Document
      • Action Log
      • Project Management System Summary
      • Project Procedures Description
      • Project Quality Plan
        Main Description
        Revise the Project Quality Plan and Project Procedures Description to avoid a recurrence of process deficiencies.

        If corrective actions impact the Operational or project schedules, or other work plans throughout the project structure, update these work products and the Action log.

        Reference OPAL Procedures:

        • Project Monitoring and Control Procedure (IMSP421)
        • Process Assurance Review Procedure (IMSP232)
