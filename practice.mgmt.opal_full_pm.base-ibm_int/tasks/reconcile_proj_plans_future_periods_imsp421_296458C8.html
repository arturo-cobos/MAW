Task: Reconcile project plans for future periods (IMSP421)
Using the updated project status and the resulting identified variances and actions, this task will update and reconcile project plans, and initiate actions, as necessary.
Purpose
To evaluate the project's ability to complete the work on time with the resources currently available or planned, and keep the progress variances within limits that will not put the project at risk.

        Main Description

        The key outputs of this task are the Milestone List, Project Management Schedule, Operational Schedule, Staff Schedule, Project Management System Summary, and the Action Log and Action Control Document, as necessary.

        This task includes the following:

        • Update project plans and notify appropriate stakeholders
        • Initiate necessary action(s)
        • Reconcile project plans
