Task: Investigate issue and plan for situation analysis (IMSP525)
Disciplines: Event Management
Purpose
To investigate issues related to the situation to be analyzed and plan for the analysis.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Defect Analysis Document
  • Defect Log

  • Issue Log
    		Optional:

    Outputs
    • Issue Document
    • Issue Log

    • Operational Schedule
    • Project Management Schedule
    • Project Status Report
    • Root Cause Analysis Record
    • Situation Analysis Documentation
    • Staff Schedule
      Main Description
      Understand the concerns about the project situation; for example recurrent conflicts, sponsor uneasiness, estimate to complete which keep increasing, etc. Collect data from the project plans and records to learn more about the project situation, and build an agenda for specific project reviews and interviews for analyzing the causes of the situation. Acquire any additional resources required to perform the situation analysis.
