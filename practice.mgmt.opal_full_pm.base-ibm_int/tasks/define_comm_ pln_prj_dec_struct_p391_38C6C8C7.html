Task: Define communication plan and project decision structure (P391)
This task will expand upon the detail related to stakeholders, communications, and the format and schedule for project status reviews and report which support the review process(es), including performance critical thread reviews.
Purpose

This task will define the format and schedule for project status reviews and reports which support the review process(es) along with the decision making process.  Identifies all stakeholders receiving reports, including the client, delivery managers and other portfolio managers, and their associated report format.

        Main Description

        This task is based on the WWPMM work pattern S3: Expand work plans.

        The key outputs of this task are the Communications Plan and the Project Decision Structure.

        An initial plan Communications management plan is usually created as part of the planning phase.  The plans are typically only "outlined", being developed to a level which was sufficient to prepare the Agreement, but that is not sufficiently detailed to control project execution. It is now time to "expand"  these plans to a level of detail relevant for project execution control, by documenting any additional project stakeholders and their roles, along with specific details required for communications planning, including:

        • Planning communication both external and internal to the delivery organization
        • Identifying meetings, reporting, and communications that will occur with the sponsor, subcontractors, the project team, and with the performing and delivery organizations' management.
        • Define project reporting in the communications management plan, defining the format and schedule for project status reviews and reports, and establishing the format, tools, schedule and frequency for status reporting.
        • Discuss with the team and client how decisions will be made and define the decision making processes to support orderly and efficient decision making throughout the project.
