Task: Document the preliminary milestone and develop staffing plan (P491)
This task will document the preliminary milestone list and the known dependencies, and also develop the staffing plan.
Purpose

To ensure the preliminary milestones, known dependencies and staffing plan are documented so the project can be planned and started with a clear understanding of these items.

        Main Description

        This task includes the following:

        • Document the preliminary milestone list, including known dependencies
        • Develop the staffing plan
          • Documenting the high-level staffing plan which shows the resources required by time period, for example week, month or quarter, by role through the life of the project
          • Defining technical skill requirements by role, where there are unique requirements
          • Identifying skill gaps and the plan to address them
          • Recording the organization and people information in the human resource plan

        The key outputs of this task are the Project Management Schedule and Milestone List.  Additional output may include the Human Resource Plan, which may be part of the Project Management System Summary.
