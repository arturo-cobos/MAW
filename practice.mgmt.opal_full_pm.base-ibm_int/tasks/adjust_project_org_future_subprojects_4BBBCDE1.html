Task: Adjust project organization and future sub-projects
Disciplines: Project Definition
Purpose
To identify the appropriate project organization and decision structure for future sub-projects.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Organizational Breakdown Structure (OBS)
  • Project Definition
    		Optional:
    • None
    Outputs
    • Organizational Breakdown Structure (OBS)
    • Project Definition
      Main Description
      Based on the planning framework that was defined in the previous task, identify the delivery organizations that have the skills required to produce the work units for the coming phase(s).

      Adjust or revise the project organization and decision structure in the Project Definition and in the Organizational Breakdown Structure (OBS).
