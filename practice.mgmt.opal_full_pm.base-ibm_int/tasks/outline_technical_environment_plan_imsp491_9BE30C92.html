Task: Outline the technical environment plan (IMSP491)
This task will outline the technical environment, including development and test or simulation environments. It will describe the technical environment of the office environment for the project.
Purpose
To create an outline of the technical environment and document it the technical environment plan.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Project Definition
    		Optional:
    • None
    Outputs
    • Process, Methods and Tools Profile
    • Technical Environment Plan
      Main Description
      Outline the technical environment plan, including development and test or simulation environments. Describe in the technical environment plan the project office environment for the project, which includes the following:
      • Hardware
      • Software & Software licensing
      • Netware
      • Equipment
      • Test environments and facilities
      • Other tools infrastructure requirements

      Document the process, methods and tools utilized on the project in the Process, Methods and Tools (PM&T) Profile

        Refer to the Technical Environment Management Guideline for further guidance.
