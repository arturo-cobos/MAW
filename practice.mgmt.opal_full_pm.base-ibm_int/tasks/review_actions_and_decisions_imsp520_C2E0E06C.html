Task: Review actions and decisions (IMSP520)
Review the defect and schedule the corrective action.
Purpose
To review the defect and schedule the corrective action.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Corrective actions
  • Defect Analysis Document

    Outputs
    • Defect Analysis Document
    • Defect Log

      Main Description
      Review the defect documentation and ensure the information is sufficient.  Then determine whether action described is appropriate.  Approve, prioritize and schedule the action to correct the defect.
      Steps
      Review updated defect log
      Review updated defect log to ensure the information is appropriate and sufficient.
      Determine whether or not the recommended action is appropriate for the project
      Determine whether or not the recommended action is appropriate for the project.
      Return the defect for further analysis
      If the recommended action is NOT appropriate, return the defect for further analysis to the Resolution Owner.
      Approve, prioritize, schedule and assign the recommended action

      If the recommended action is appropriate, approve, prioritize, schedule and assign the recommended action.

      It is essential that approved actions get properly prioritized, so that the project is completed in a timely manner.  The most severe defects need to get fixed first.
