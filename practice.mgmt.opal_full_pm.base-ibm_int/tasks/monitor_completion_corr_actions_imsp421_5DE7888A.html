Task: Monitor completion of corrective actions (IMSP421)
This task will monitor the action documentation to ensure closure of corrective actions.
Purpose
To monitor the completion of corrective actions.
Relationships
RolesPrimary Performer:
  • Project Manager
Additional Performers:
InputsMandatory:
  • Action Control Document
  • Action Log
  • Quality Review Documentation
    		Optional:
    • Organizational PQA Non-Compliance Log
    • Process Assurance Review Report - Organization
      Outputs
      • Action Control Document
      • Action Log
      • Organizational PQA Non-Compliance Log
      • Quality Review Documentation
        Main Description
        During the audit or review of a project several corrective actions may be documented.  This is usually documented in the action documentation.  This activity is required to monitor the completion of these required actions.   This will ensure the successful completion of the corrective actions.
