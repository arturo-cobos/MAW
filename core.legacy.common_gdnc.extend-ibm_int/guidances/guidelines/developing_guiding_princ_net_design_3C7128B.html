Guideline: Developing Guiding Principles for a Network Design
This technique paper discusses how to develop guiding principles for a network conceptual design.
Main Description
Development of network guiding principles, while being specific to networking, uses the same underlying techniques for developing all guiding principles.
 
Guiding principles define the general underlying rules which an organization will use to plan and deploy network resources and assets across an enterprise, e.g., "Networks will be designed to be capable of rapid expansion in line with planned and unpredicted business growth." Principles are also developed to help explain why certain products, standards, techniques or methodologies are preferred. "The XYZ Corporate Network will be designed using standards based components to support network management, change and maintainability."
