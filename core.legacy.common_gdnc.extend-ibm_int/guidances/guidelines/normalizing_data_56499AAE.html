Guideline: Normalizing Data
Data normalization is an optimization technique for tying data elements together in a database based on the relationships between data.
Main Description
The use of the normalization technique makes it possible to increase flexibility in the updating, adding, or deleting of data in databases, and to increase flexibility in the changing of data elements.

To be more specific, in the process of data normalization, groups of data are broken down and refined into simpler sets of data without losing the meaning of the original data. Dependencies between data elements in the group (a data relation, or a set of data elements that have a strong relationship, such as DFD data store, file, screen, or form) of data elements under study are analyzed to eliminate redundancies (situation where the same data element is maintained in several places).
