Guideline: Test Strategy and Planning Workshop
Main Description
This paper covers planning, preparation, conducting and reporting for a test strategy and planning workshop. It focuses on the Test Strategy and Master Test Plan work products but the same techniques may be applied to develop detailed test plans and acceptance test plan. This approach may also be used to establish a standard testing framework for a development or maintenance organization to be used as a common base for their applications or products testing (e.g. test focus areas, types of test, levels could all be standardized).

It assists with the selection of appropriate participants, identification of education required for those participants, preparation of supporting materials, sending communications and arranging facilities. It provides agendas and data gathering documentation templates. It includes workshop facilitation guidance and support to build quality workshop deliverables and participant consensus.
