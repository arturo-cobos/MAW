Guideline: Tailoring the Test Method
Main Description

The Testing Methodology consists of a set of core testing and management processes, together with a number of engagement models which structure the core processes into sequenced activities and tasks for different testing projects. The methodology is work product based, which means all activities and tasks are defined for the purpose of generating the work products.
