Tool Mentor: IT Options Analysis
This tool mentor provides an introduction to a new analytical tool called IT options analysis (ITOA). ITOA applies real options analysis to IT investment valuation and decision-making.
Main Description
"I have become convinced that it is time to revisit the usefulness of NPV [net present value] and to reconsider just how much stock we want to place in it. For most investments, the usefulness of the NPV rule is severely limited. If modern finance is to have a practical and salutary impact on investment-decision making, it is now obligated to treat all major investment decisions as option pricing problems."
 
Steven Ross, Sterling Professor of Economics and Finance at Yale School of Management; Keynote address at the 1994 Financial Management Association Annual Meeting
 
Real options analysis extends financial option theory to real, or nonfinancial, assets. This paper reviews NPV (net present value), as it is the foundation for real option analysis. It then defines what real options are and how real options analysis differs from NPV techniques. The case is made as to why traditional NPV analysis is ineffective for evaluating investment decisions. The paper then explains the mechanics of valuing simple options using basic examples of IT investment.
