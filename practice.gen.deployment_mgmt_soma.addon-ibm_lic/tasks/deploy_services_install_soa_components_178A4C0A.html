      Main Description
      Deploy services from the test environment to the production environment and reconfigure as necessary for release. Installing and configure middleware products and other software products that enable Service Oriented Architecture such as platforms and products for Enterprise Service Bus for routing, mediation, and transformation; Service Registry and Repository for service lifecycle management; BPEL engine for process choreography and products that enable information on demand if applicable. Publish the services in service registry for service lifecycle management.
      Steps
      Deploy services and related implementation artifacts to production topology
      Deploy the process and related implementation artifacts to production topology
      Perform the necessary configuration of the production topology
      Configure the endpoints as needed
      Configure services repository and registry with the needed artifacts
      Configure any resources as necessary
      Configure any resources (databases, supporting applications and services) as necessary
