Main Description
This document is an external input from the sponsor or customer. It is a request for work raised by the sponsor as provided for in the Agreement, and describes the piece of work which is to be planned and carried out.
 
For example, an application maintenance agreement may stipulate that the delivery organization will perform periodic minor modifications on demand, within a budget that is already part of the price. Then, from time to time, the project would receive requests which refer to the Agreement and describe the minor modifications that must be performed.
