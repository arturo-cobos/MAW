Main Description
In most cases, a supplier evaluation report document needs to be prepared for the procurement organization.
 
This is a brief report highlighting the strengths and weaknesses of the supplier.
 
The report typically covers areas such as:
  • Quality of deliverables, including documentation
  • Timeliness
  • Value for money
  • Skills and experience of personnel
  • Flexibility and responsiveness
  • Communications
  • Project management
  • Administration
