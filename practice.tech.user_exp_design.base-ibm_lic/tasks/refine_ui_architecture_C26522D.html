Task: Refine User Interface Architecture
Validate that the user-interface conceptual model is implementable and consistent in look, feel, and metaphors
Disciplines: Technical Design, User Experience
Purpose
The UI architecture effort continues into the micro design phase to ensure that the detailed UI design specified during this phase will be supported and implementable through the chosen operating system and application environment.

The second purpose of the user-interface architecture effort is to ensure that the user-interface is consistent in look, feel, metaphors, and conceptual model. This is done through the design of frameworks and widgets that "encapsulate" the UI style as specified in the style guide and detailed user-interface specification.
