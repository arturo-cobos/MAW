Task: Define User Interface Architecture
Ensure that the user-interface conceptual model is implementable and consistent in look, feel, and metaphors
Disciplines: Technical Design, User Experience
Purpose
The first purpose of the user-interface architecture is to ensure that the user-interface conceptual model, as developed during macro design is implementable, given what is known about the expected operating system and application environment for the solution.

The second purpose of the user-interface architecture effort is to ensure that the user-interface is consistent in look, feel, metaphors, and conceptual model. This is done through the design of frameworks and widgets that "encapsulate" the UI style as specified in the style guide and user-interface specification.
