Task: Conduct Usability Test
Conduct the usability test per the plan. A usability test is a controlled evaluation of the usability of a software application or prototype by a set of representative users.
Disciplines: Technical Testing, User Experience
Purpose
Conduct Usability Test provides feedback as to whether the system is meeting its established usability goals. It provides an early indication of how well the system will be received once it is deployed.

      Main Description
      Conduct the Usability Test per the Usability Design and Evaluation Plan. Select and schedule specified users to perform selected user scenarios while their interactions are observed and logged. Collect both quantitative and qualitative data to analyze and report. Note that if a Usability Test Plan exists this may also provide detailed operational planning guidance for schedule and resource allocation information for this effort.
