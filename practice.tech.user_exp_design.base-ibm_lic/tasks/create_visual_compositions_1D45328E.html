Task: Create Visual Compositions
Develop design approaches that explore different themes or concepts, including possible audience segmentation.
Disciplines: Technical Design, User Experience
Purpose
The goal of this task is to visualize and establish initial design directions aligned with graphical and navigational constructs to create the desired user experience.

        Main Description
        Each approach may incorporate palette, layout, fonts, composition, and navigational elements that reinforce the desired user experience. 

        Interactive concepts are much like wireframes except they take specific application functionality and navigation into account. They are meant to allow the designer room to explore communication motifs and validate the functional requirements in the context of application views. Because they combine visual and navigational metaphors, they often result in the initial iconography of the application. At this phase, since these deliver both form and function, they represent the first drafts of the user experience.
