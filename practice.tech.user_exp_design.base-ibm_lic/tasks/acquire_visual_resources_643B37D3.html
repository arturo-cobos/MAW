Task: Acquire Visual Resources
Catalog all the media content and other visual resources
Disciplines: Technical Implementation, User Experience
Purpose
The goal of this task is to create (or acquire) and catalog all the media content and other visual resources required for the user interface (or UI prototype) developed during this phase of the project.
Relationships
RolesPrimary Performer:
  • User Experience Professional
Additional Performers:
  • Application Developer
InputsMandatory:
  • None
Optional:
  • User Interface Conceptual Model
  • User Interface Prototype
    Outputs
    • Media Content
      Main Description
      Media content is all of the images (including photos, illustrations, icons, etc.), automation scripts, applets, user interface widgets, video, typographic elements, etc., that are needed for the user interface (or UI prototype). On larger projects with extensive user interfaces, the work required to build and assemble these resources is significant.
