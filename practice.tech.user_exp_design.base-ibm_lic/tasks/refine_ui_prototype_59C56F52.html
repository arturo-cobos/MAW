Task: Refine User Interface Prototype
Build a high-fidelity user-interface prototype
Disciplines: Technical Design, User Experience
Purpose
The purpose of this task is to build a high-fidelity user-interface prototype that reflects a detailed design direction that has been verified and refined with end user feedback and input. The user-interface prototype will be used as a primary input to the detailed user interface design specification.

    Main Description
    Review and analyze the use case model to identify all use cases that require a user interface. Review the user interface conceptual model and early usability evaluation results to provide direction on further refinement of the low fidelity prototype. Create a high-fidelity prototype, in the eventual development environment, that reflects the conceptual model and also provides user interface views necessary to support use cases. Perform a usability test on this high-fidelity prototype that captures task completion times and error rates. Based on the results of this test, refine the high-fidelity prototype. Continue this prototyping-testing-refinement loop until there is significant confidence that the design as embodied in the prototype, will support the usability requirements for the solution.
