Task: Refine Usability Design and Evaluation Plan
Ensure that adequate resources are allocated and scheduled to achieve usability objectives of the solution
Disciplines: Technical Design, User Experience
Purpose
It ensures clear and shared understanding of how usability and user-interface design will be addressed and that the implemented system meets usability requirements. In addition, it is used to estimate how much usability will impact the project cost, project duration and required skills and resources.

    Main Description
    Specify how usability will be implemented throughout the course of the design and development of a solution, including schedules and task assignments. Identify activities throughout the design process, some of which include: prototyping the user interface, refining and elaborating of the User Interface Guidelines, refining and elaborating high-level and low-level UI design, and further specifying usability testing.
