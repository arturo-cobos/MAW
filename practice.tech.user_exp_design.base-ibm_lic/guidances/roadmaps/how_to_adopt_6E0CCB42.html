Roadmap: How to Adopt the User Experience Driven Design Practice
Main Description

Getting Started

The User Experience Driven Design practice is intended to be a supplemental practice that provides cross-cutting support across requirements, design, architecture, implementation and testing in the area of usability.  It assumes that the development of the core work products in these areas will be produced in other practices.
