Task Descriptor: Evaluate Current User Experience
Identifies strengths and weaknesses of the current solution in meeting user needs and usability requirements.

Main Description
Collect benchmark data on the existing system solution and help desk report details of the system (if they exist) , and determine the strengths and weakness of the current solution's key features. Rank these strengths and weaknesses according to user importance and satisfaction levels. This usability assessment may be based on an expert heuristic review, a user-based subjective walk-through or a user based performance benchmark.

In a user-based subjective walk-through, a user performs the current task scenarios while providing observations and subjective feedback to the observer. For a performance benchmark, the focus is on establishing time on task and error criteria in order to identify problems for new solution design priorities and to establish productivity baselines. A heuristic evaluation involves a human-interface specialist conducting a self-assessment of a solution. A set of guidelines is used to gauge a solution's usability, pinpoint areas for remediation, and provide recommendations for design change.
