Main Description

Identify user-interface components that have been customized to meet the specific needs of the solution. Document required and recommended usage of these components by updating the style guideline. Put in place a practice for recording and tracking deviations from the style guide. Track these deviations over the course of the project.

The steps required to complete this task are:

  • Identify user-interface components that have been customized to meet the specific needs of the solution.
  • Document required and recommended usage of these components in the style guideline.
  • Establish a mechanism for recording and tracking deviations from the style guide.
  • Track these deviations over the course of the project.
